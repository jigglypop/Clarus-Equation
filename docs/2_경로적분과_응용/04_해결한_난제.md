## 4.4 SM 자유 매개변수에 대한 기여

SM의 19개 자유 매개변수 중 CE가 설명하거나 제약하는 수:

| SM 매개변수 | CE 상태 |
|---|---|
| alpha_s | 입력 (유일한 물리 상수) |
| sin^2(theta_W) | 유도됨 (= 4 alpha_s^{4/3}) |
| alpha_EM | 유도됨 (sin^2(theta_W) alpha_s^{2/3}에서) |
| theta_QCD | 유도됨 (= 0, 6절) |
| N_gen = 3 | 유도됨 (= d, 1절) |
| 렙톤 질량비 | 구조적 제약 (Koide 2/3, 9절) |
| CKM CP 위상 | 존재 보장됨 (d=3, 8절) |
| 나머지 (Yukawa 9개, CKM 3각, 힉스 2개) | 미해결 (9절 열린 문제) |

CE에 의해 설명/제약되는 매개변수: 5-7개 (전체의 26-37%).
