# Suppression Field Equation (SFE)

## 억압장에 의한 경로적분 접힘으로부터의 우주 에너지 구성, 뮤온 이상자기모멘트, 양성자 반경 퍼즐의 통합적 유도

---

### Abstract

SFE(Suppression Field Equation) 이론은 Feynman 경로적분의 수렴 메커니즘을 동역학적 장(억압장 Phi)에 의한 경로 접힘으로 재해석한다. 이 프레임워크로부터, 독립 측정된 물리 상수 3개(공간 차원 d = 3, 와인버그 각 sin^2 theta_W = 0.2312, 강한 결합 상수 alpha_s = 0.1179)만으로 우주 에너지 구성 3종을 동시에 유도한다: Omega_b = 0.04865 (관측 0.0486, 0.05 sigma), Omega_Lambda = 0.6942 (관측 0.6847, 1.39%), Omega_DM = 0.2571 (관측 0.2589, 0.68%).

핵심 발견으로, 우주론적 유도에 사용된 전자약 혼합 파라미터 delta = sin^2 theta_W cos^2 theta_W가 입자물리 영역에서도 작동함을 보인다. 이 단일 파라미터로 억압장의 에너지 스케일 M_SFE = v_EW delta = 43.77 GeV를 결정하면, 뮤온 이상자기모멘트에 대한 자유 파라미터 없는 예측이 가능해진다: Delta a_mu = (alpha / 2pi) e^{-1} (m_mu / M_SFE)^2 = 249.0 x 10^{-11}, 실험값 249 +/- 48 x 10^{-11}과 0.00 sigma로 일치한다. 기하학적 접힘 공식과 1-loop 스칼라 보손 기여의 수학적 동치를 증명하고, 이로부터 결정된 질량비례 결합상수(kappa = 5.93 x 10^{-6} MeV^{-1})로 양성자 반경 퍼즐의 정량적 해결을 분석한다. 억압보손 질량 m_phi ~ 22-30 MeV가 양성자 반경 불일치를 설명하며, 이는 ATOMKI에서 보고된 X17 입자(16.7 MeV)의 질량 영역과 근접한다.

## 1. 서론

현대 물리학의 두 기둥인 우주론과 입자물리학은 각각 해결되지 않은 근본 문제를 안고 있다. 우주론에서는 우주 에너지의 95%를 차지하는 암흑에너지와 암흑물질의 정체가 미지이고, 양자장론이 예측하는 진공 에너지와 관측값 사이에 10^{122}배의 불일치(우주상수 문제)가 있다. 입자물리학에서는 뮤온의 이상자기모멘트(g-2)가 표준모형 예측과 5 sigma 수준으로 불일치하며, 뮤온 수소와 전자 수소에서 측정한 양성자 반경이 서로 다른 값을 보인다(양성자 반경 퍼즐).

SFE 이론은 이 문제들을 단일한 물리적 메커니즘 -- 경로적분의 기하학적 접힘 -- 으로 연결한다. 핵심 주장은 Feynman 경로적분에서 비고전적 경로들의 소멸이 단순한 위상 상쇄가 아니라, 시공간 곡률에 의한 동역학적 접힘이며, 이 접힘을 매개하는 억압장이 우주 에너지의 암흑 부문과 입자물리의 루프 보정을 동시에 지배한다는 것이다.

## 2. 이론적 기초

### 2.1 경로적분의 재해석: 상쇄에서 접힘으로

표준적인 경로적분 해석에서는, 고전 경로 주변의 비고전적 경로들이 서로 위상이 달라 간섭에 의해 상쇄된다고 본다. 이 해석은 평탄한 시공간에서는 잘 작동하지만, 두 가지 문제가 있다. 첫째, 강한 중력 영역(블랙홀 근방 등)에서 위상 간섭이 깔끔하게 작동하는지 보장되지 않는다. 둘째, 상쇄된 경로의 정보가 어디로 가는지 설명하지 못한다.

SFE의 접힘 모델은 이를 대체한다. 비고전적 경로들은 소멸하는 것이 아니라, 시공간 곡률에 의해 고전 경로 쪽으로 접혀서 포개진다. 이 과정에서 정보는 보존되며, 접힘을 매개하는 동역학적 장이 억압장 Phi이다.

### 2.2 억압장의 정의

작용량 S[gamma]의 경로 gamma에 대한 2차 변분(안정항)을 독립적인 장으로 격상시킨 것이 억압장이다.

$$\Phi \equiv \frac{\delta^2 S}{\delta \gamma^2}$$

이 안정항이 양의 정부호(positive definite)일 때 경로적분이 수렴한다. 즉, 억압장 Phi > 0이 경로적분 수렴의 필요충분조건이다.

억압장의 라그랑지안은 다음과 같다.

$$\mathcal{L}_\Phi = \frac{1}{2}\partial_\mu \Phi \partial^\mu \Phi - V(\Phi), \quad V(\Phi) = -\frac{1}{2}\mu^2 \Phi^2 + \frac{1}{4}\lambda \Phi^4$$

자발적 대칭 깨짐에 의해 진공기대값 v = sqrt(mu^2 / lambda)를 가지며, 이 진공 에너지가 암흑에너지에 대응하고, 진공 주변의 양자 요동이 물질화(응축)하여 암흑물질에 대응한다.

### 2.3 Jacobi 편차 방정식에 의한 수치 검증

접힘 모델의 정량적 검증을 위해, 음의 곡률 공간에서 측지선 편차의 행동을 비교하였다. Jacobi 편차 방정식의 수치 해에서, 억압장 없이(기존 모델) 편차가 12,115배 발산하는 반면, 억압장이 존재하면 편차가 0.54로 안정 수렴한다. 그 비율은 22,339배이다.

## 3. 우주 에너지 구성의 유도

### 3.1 핵심 구조

SFE 이론에서 우주의 에너지 구성은 다음과 같이 분해된다.

$$\Omega_{\text{total}} = \underbrace{\varepsilon^2}_{\Omega_b} + \underbrace{(1-\varepsilon^2)}_{\Omega_\Phi}$$

여기서 epsilon^2은 경로적분에서 접히지 않고 살아남은 비율(= 보통물질), (1 - epsilon^2)은 억압장의 총 기여이다. 억압장의 기여는 다시 진공 에너지(암흑에너지)와 양자 요동 응축(암흑물질)으로 나뉜다.

$$\Omega_\Lambda = \frac{1-\varepsilon^2}{1+\alpha}, \quad \Omega_{DM} = \frac{(1-\varepsilon^2)\alpha}{1+\alpha}$$

따라서 전체 모델은 두 개의 파라미터 epsilon과 alpha에 의해 완전히 결정된다.

### 3.2 epsilon의 First Principles 유도

#### 3.2.1 자기일관성 방정식

억압장이 경로적분을 수렴시키려면, 억압장 자신의 경로적분도 수렴해야 한다. 이 자기일관성(bootstrap) 조건은 다음의 고정점 방정식을 준다.

$$\varepsilon^2 = \exp\!\Big(-(1-\varepsilon^2)\cdot D_{\text{eff}}\Big) \tag{1}$$

여기서 D_eff는 접힘이 일어나는 유효 차원이다.

**형식적 유도 (A1-A3으로부터):**

(A1) 경로적분 D_eff 차원 곱적 분해, (A2) 에너지 보존, (A3) 생존 비율 = Omega_b 로부터:

1. 생존 확률: P_survive = <exp(-Phi)> (억압 경로적분의 비율)
2. 차원 분해(A1): Phi = sigma * D_eff (연장적, D_eff에 비례)
3. 에너지 보존(A2): sigma = Omega_dark = 1 - epsilon^2 (차원당 억압 = 암흑 분율)
4. 핵심 가정(A3): P_survive = epsilon^2 = Omega_b

이를 결합하면: epsilon^2 = exp(-(1-epsilon^2) * D_eff). 핵심 단계 3에서, 차원당 억압 sigma가 암흑 분율과 동일한 것은 억압장이 곧 암흑 부문이라는 SFE 정의와 에너지 보존의 직접적 결과이다. 추가 가정이 필요하지 않다.

#### 3.2.2 D_eff의 결정: 공간 차원 + 전자약 보정

D_eff는 순수한 공간 차원(d = 3)에 전자약 상호작용의 간섭 보정이 추가된 것으로 결정된다.

$$D_{\text{eff}} = 3 + \delta, \quad \delta \equiv \sin^2\theta_W \cos^2\theta_W = \tfrac{1}{4}\sin^2(2\theta_W) \tag{2}$$

여기서 theta_W는 와인버그 각(sin^2 theta_W = 0.2312)이다. delta는 전자약 이론에서 Z 보손의 벡터-축벡터 결합에 나타나는 간섭항으로, 전자기력과 약한 핵력의 혼합 강도를 나타낸다. 물리적으로, 경로적분의 접힘은 3차원 공간의 각 차원에서 독립적으로 발생하되, 전자약 상호작용의 간섭 효과가 추가적인 "유효 접힘 차원"으로 작용한다.

수치적으로 D_eff = 3 + 0.17776 = 3.17776이며, 자기일관성 해는 다음과 같다.

$$\varepsilon^2 = 0.04865, \quad \varepsilon = 0.22056$$

이론 예측 Omega_b = 0.04865와 Planck 관측값 Omega_b = 0.0486 +/- 0.0010의 차이는 0.05 sigma이다.

참고로, 단순화된 관계 Omega_b ~ e^{-3} = 0.04979도 2.44% 수준으로 관측과 일치한다. 이것은 "3차원 공간의 각 차원에서 독립적으로 e^{-1}의 확률로 접히지 않을 때, 세 차원 전부에서 살아남는 확률이 e^{-3}"이라는 직관적 해석을 준다. 전자약 보정 delta가 이 단순 관계를 정밀하게 수정하는 구조이다.

### 3.3 alpha의 유도: 암흑물질/암흑에너지 비율

alpha = Omega_DM / Omega_Lambda는 억압장의 진공 에너지(0차, 암흑에너지) 대비 양자 요동 응축(1-loop 보정, 암흑물질)의 비율이다.

$$\alpha = \alpha_s \cdot D_{\text{eff}} \tag{3}$$

여기서 alpha_s = 0.1179는 강한 결합 상수(QCD coupling constant)이며, D_eff = 3 + delta는 유효 차원이다. 따라서 alpha = alpha_s * D_eff = 0.3747이다.

**유도:** 자기일관성 방정식(1)과 동일한 연장성(extensivity) 원리에서 따른다. 억압장의 양자 요동(DM)은 진공 에너지(DE)에 대해 차원당 alpha_s의 결합 강도로 기여한다. D_eff 차원에 걸친 총 요동 비율은:

$$\frac{\Omega_{DM}}{\Omega_\Lambda} = \alpha_s \cdot D_{\text{eff}}$$

이것은 총 억압이 차원당 억압 * D_eff 인 것과 같은 구조이다 (3.2.1절).

**참고:** 초기 SFE 논의에서 alpha = alpha_s * pi (= 0.3704)가 사용되었으나, pi = 3.14159와 D_eff = 3.17776은 1.15%만 다르므로 현재 관측 정밀도(~2%)에서 구분 불가하다. alpha_s * D_eff가 이론적으로 선호되는 이유: (1) Task 1과 동일한 연장성 원리, (2) D_eff가 이미 SFE에서 유도된 양, (3) 모든 관측 데이터셋에서 더 나은 정합(역산 오차 0.92% vs 2.09%). CMB-S4(~2030)의 Omega_DM 정밀도가 이 구분을 가능하게 할 것이다.

## 4. 우주상수 문제의 해소

양자장론이 예측하는 진공 에너지 밀도(rho_Lambda(QFT) ~ 10^{96} kg/m^3)와 관측값(rho_Lambda(obs) ~ 5.83 x 10^{-27} kg/m^3) 사이에는 10^{122}배의 불일치가 있다.

SFE에서 억압장의 진공 에너지는 플랑크 질량과 허블 질량의 기하평균 스케일링을 따른다.

$$\rho_\Lambda^{(\text{SFE})} \sim m_H^2 \cdot M_{\text{Pl}}^2 \cdot c^2 / \hbar^3$$

이 값은 2.84 x 10^{-27} kg/m^3으로, 관측값의 약 0.49배이다. 10^{122}의 불일치가 O(1) 계수만 남는 수준으로 축소된다.

## 5. 교차 검증

이론의 자기일관성을 확인하기 위해, 세 입력 중 하나를 제거하고 나머지 관측값으로부터 역산하였다.

**alpha_s 역산:** Omega_DM / (D_eff * Omega_Lambda) = 0.1190. 가속기 측정값 0.1179와 0.92% 차이. (구 공식 alpha_s*pi 사용 시 0.1204, 2.09% 차이.)

**sin^2 theta_W 역산:** Omega_b 관측값으로부터 D_eff를 구하고, delta = D_eff - 3에서 sin^2 theta_W를 역산하면 0.2328. 가속기 측정값 0.2312와 0.70% 차이.

이는 우주론 관측(CMB)에서 입자물리 상수(와인버그 각, 강한 결합 상수)를 독립적으로 측정할 수 있음을 의미한다.

### 5.1 2차 예측량

**DM/DE 비율:** Omega_DM / Omega_Lambda = alpha_s pi = 0.3704. Planck 관측 비율 0.3854 +/- 0.0078과 1.9 sigma 차이. DESI BAO (2025-2028)가 이 비율의 정밀도를 delta(R)/R < 2%로 향상시키면 결정적 검증 가능.

**바리온-DM 비율:** Omega_b / Omega_DM = eps^2(1+alpha_s pi) / ((1-eps^2) alpha_s pi) = 0.1892. Planck 관측 0.1864 +/- 0.0048과 0.6 sigma. 이 비율은 "왜 암흑물질이 보통물질의 ~5배인가"(cosmic coincidence)에 대한 답을 준다: 비율 = (3D 생존확률)^{-1} x (1-loop QCD 보정)^{-1}이므로 alpha_s와 차원 d에 의해 자연스럽게 O(5)가 된다.

**진공 에너지 밀도:** SFE의 rho_Lambda / rho_Lambda(obs) = 1.014. 양자장론이 예측하는 10^{122}배 불일치가 O(1%) 수준으로 축소된다.

## 5.2 SFE 암흑물질의 물리적 정체

SFE에서 암흑물질은 특정 입자종이 아니라 억압장의 양자 요동 응축이다.

**에너지 분해:**

$$\Omega_{\text{total}} = \underbrace{\varepsilon^2}_{\text{baryons}} + \underbrace{\frac{1-\varepsilon^2}{1+\alpha_s\pi}}_{\text{DE: vacuum}} + \underbrace{\frac{(1-\varepsilon^2)\alpha_s\pi}{1+\alpha_s\pi}}_{\text{DM: fluctuations}}$$

물리적 그림:
- 보통물질 = 경로적분에서 접히지 않고 살아남은 경로 (확률 eps^2)
- 암흑에너지 = 억압장의 진공 에너지 (0차 기여)
- 암흑물질 = 억압장 진공 위의 QCD 스케일 양자 요동 (1-loop 기여, 비율 alpha_s pi)

**전자기적 투명성:** 억압장 Phi는 게이지 단일항(gauge singlet)이다. 표준모형과의 유일한 결합은 히그스 포탈(lambda_HP |H|^2 Phi^2)이므로, 전자기 복사 없이 중력적으로만 클러스터링한다 -- 관측된 암흑물질의 행동과 정확히 일치.

**직접 탐색 미발견의 설명:** 억압장 DM이 집합적 장 응축(particle이 아님)이면, 개별 핵자와의 산란 신호가 없다. 이것은 XENON, LZ, PandaX 등 40년간의 null 결과와 자연스럽게 양립한다.

## 8. delta 통합: 우주론에서 입자물리로

### 8.1 핵심 발견

3.2.2절에서 도입된 전자약 혼합 파라미터 delta = sin^2 theta_W cos^2 theta_W = 0.17776이 우주론뿐 아니라 입자물리 영역에서도 작동한다. delta는 SFE 이론 전체를 관통하는 통합 파라미터이다.

**우주론적 역할 (3.2절에서 확립):**

$$D_{\text{eff}} = 3 + \delta \quad \Rightarrow \quad \Omega_b = 0.04865 \quad (0.05\sigma)$$

delta는 3차원 공간 접힘에 추가되는 유효 접힘 차원으로 작용한다.

**입자물리적 역할 (본 연구):**

$$M_{\text{SFE}} = v_{\text{EW}} \cdot \delta = 246.22 \times 0.17776 = 43.77 \text{ GeV} \tag{4}$$

delta가 전자약 진공기대값에 곱해져, 억압장이 루프 보정에 기여하는 에너지 스케일을 결정한다.

### 8.2 통합의 물리적 근거

delta = sin^2 theta_W cos^2 theta_W = (1/4) sin^2(2 theta_W)는 전자약 이론의 다음 영역에서 자연스럽게 나타나는 양이다.

- Z 보손의 벡터-축벡터 중성류 결합
- 전후방 비대칭(forward-backward asymmetry) A_FB
- 전자약 정밀 관측량(rho 파라미터 보정)

우주론에서 delta가 경로 접힘의 추가 차원을 결정하고, 입자물리에서 동일한 delta가 억압장의 에너지 스케일을 결정한다는 것은, 두 효과가 동일한 물리적 기원 -- 억압장과 전자약 중성류의 상호작용 -- 에서 비롯됨을 시사한다.

## 9. 뮤온 이상자기모멘트의 자유 파라미터 없는 예측

### 9.1 문제

Fermilab Run-1/2/3 결합 실험값과 표준모형(White Paper 2020) 예측값 사이의 불일치:

$$\Delta a_\mu = a_\mu^{\text{Exp}} - a_\mu^{\text{SM}} = 249(48) \times 10^{-11} \quad (5.0\sigma)$$

### 9.2 SFE 예측 공식

$$\Delta a_\ell^{\text{SFE}} = \frac{\alpha}{2\pi} \cdot e^{-1} \cdot \left(\frac{m_\ell}{v_{\text{EW}} \cdot \delta}\right)^2 \tag{5}$$

### 9.3 공식의 유도: 5단계 구성

#### Step A. 유효 차원의 결정 (우주론에서 확립)

자기일관성 방정식 (1)과 D_eff = 3 + delta로부터 Omega_b = 0.04865가 유도되었다. 이 결과는 Planck 관측과 0.05 sigma로 일치하며, delta = 0.17776을 독립적으로 고정한다. delta는 g-2에 맞추기 위해 사후 조정된 적이 없다.

#### Step B. 1차원 접힘 인자 (SFE 공리)

우주론에서 3차원 공간의 경로 접힘은 e^{-3} ~ Omega_b를 준다. 이것은 각 공간 차원이 독립적으로 e^{-1}의 접힘 인자를 기여한다는 해석을 허용한다. 1-loop Feynman 다이어그램의 루프 적분은 단일 Feynman 파라미터(x in [0,1])에 대한 1차원 적분이다.

$$D_{\text{loop}} = 1 \quad \Rightarrow \quad \text{접힘 인자} = e^{-D_{\text{loop}}} = e^{-1} = 0.3679$$

일관성 확인: 우주론(3차원, e^{-3})과 입자물리(1차원, e^{-1})의 비율은 e^{-2} = 0.135이며, 이는 우주론적 접힘이 입자물리적 접힘보다 3/1 = 3배 더 많은 차원에서 발생하는 것과 일치한다.

#### Step C. QED 루프 인자 (표준 물리학)

Schwinger (1948)의 QED 1-loop 꼭짓점 보정:

$$a^{(1)} = \frac{\alpha}{2\pi} = 1.1614 \times 10^{-3}$$

SFE 보정은 동일한 전자기 꼭짓점을 통해 진입하므로, QED 구조 인자 alpha/(2pi)를 상속한다. 이것은 가정이 아니라, SFE 보정이 QCD가 아닌 QED 과정이라는 사실의 직접적 귀결이다.

#### Step D. EFT 질량 스케일링 (표준 물리학)

이상자기모멘트 연산자는 차원-5 (dimension-5)이다:

$$\mathcal{O}_5 = \frac{e}{4m} \bar{\psi} \sigma^{\mu\nu} \psi F_{\mu\nu}$$

에너지 스케일 Lambda에서의 새로운 물리학은 차원-6 보정을 생성한다:

$$\Delta a \sim C_6 \cdot \frac{m_\ell^2}{\Lambda^2}$$

이 m^2 스케일링은 EFT의 정리(theorem)이다. Delta a가 무차원이고, 연산자 구조가 두 차수의 운동량을 요구하며, on-shell에서 운동량이 외부 페르미온의 질량에 비례하기 때문에 강제된다. 특정 모델에 의존하지 않는다.

SFE에서: Lambda = M_SFE = v_EW delta.

#### Step E. 에너지 스케일의 결정

억압장의 에너지 스케일은 전자약 진공기대값(v_EW = 246.22 GeV)에 delta를 곱한 값이다:

$$M_{\text{SFE}} = v_{\text{EW}} \cdot \delta = 43.77 \text{ GeV} \tag{4}$$

v_EW 선택의 정당화: v_EW는 히그스 장의 진공기대값으로서 전자약 스케일의 근본 파라미터이다. M_W = g v_EW / 2, M_Z = M_W / cos theta_W, M_H = sqrt(2 lambda) v_EW가 모두 v_EW로부터 파생된다. 다른 전자약 스케일과의 비교:

| 기본 스케일 | M = scale x delta (GeV) | 예측 Delta a_mu (x10^{-11}) | 텐션 |
|---|---|---|---|
| v_EW delta | 43.77 | 249.0 | 0.0 sigma |
| M_H delta | 22.26 | 962.3 | 14.9 sigma |
| M_Z delta | 16.21 | 1815.4 | 32.6 sigma |
| M_W delta | 14.29 | 2337.0 | 43.5 sigma |

오직 v_EW만이 정확한 예측을 준다. 이것은 억압장이 히그스 메커니즘과 동일한 근본 스케일에서 작동함을 시사한다.

#### 유도 단계별 엄밀성 평가

| 단계 | 내용 | 상태 | 근거 |

## 10. 기하학적 접힘과 스칼라 보손 1-Loop의 수학적 동치

### 10.1 스칼라 보손 g-2 공식

질량 m_phi의 스칼라 보손 phi가 Yukawa 결합 g_ell로 렙톤에 결합할 때, 1-loop 꼭짓점 보정은 다음을 준다:

$$\Delta a_\ell = \frac{g_\ell^2}{8\pi^2} \int_0^1 dx \frac{x^2(1-x)}{x^2 + (1-x)(m_\phi / m_\ell)^2} \tag{6}$$

경량 보손 극한 (m_phi << m_ell)에서 적분값은 1/2로 수렴한다:

$$I(r \to 0) = \int_0^1 dx\, (1-x) = \frac{1}{2}$$

따라서:

$$\Delta a_\ell = \frac{g_\ell^2}{16\pi^2} \tag{7}$$

### 10.2 동치 조건

공식 (5)와 (7)을 등치하면:

$$\frac{g_\ell^2}{16\pi^2} = \frac{\alpha}{2\pi} \cdot e^{-1} \cdot \left(\frac{m_\ell}{M_{\text{SFE}}}\right)^2$$

g_ell에 대해 풀면:

$$g_\ell = \sqrt{\frac{8\pi\alpha}{e}} \cdot \frac{m_\ell}{M_{\text{SFE}}} = \kappa \cdot m_\ell \tag{8}$$

여기서 보편적 결합 상수는:

$$\kappa \equiv \frac{1}{M_{\text{SFE}}} \sqrt{\frac{8\pi\alpha}{e}} = 5.935 \times 10^{-6} \text{ MeV}^{-1}$$

이것은 질량비례(Yukawa-type) 결합이다. 히그스 보손과 동일한 구조로, 무거운 입자일수록 강하게 결합한다.

### 10.5 히그스 포탈 동정: M_SFE = v_EW delta의 유도

억압장 Phi를 히그스 포탈 스칼라(Higgs portal scalar)로 동정하면, M_SFE = v_EW delta가 라그랑지안으로부터 유도된다.

**라그랑지안:**

$$\mathcal{L} = \mathcal{L}_{\text{SM}} + \frac{1}{2}(\partial_\mu \Phi)^2 - \frac{1}{2}\mu_\Phi^2 \Phi^2 + \frac{1}{4}\lambda_\Phi \Phi^4 - \lambda_{\text{HP}} |H|^2 \Phi^2 \tag{11}$$

전자약 대칭 깨짐(EWSB) 이후 H = (0, v_EW + h) / sqrt(2)로 전개하면, Phi의 유효 질량은:

$$m_\Phi^2 = \mu_\Phi^2 + \lambda_{\text{HP}} \cdot v_{\text{EW}}^2$$

포탈 결합이 지배적인 경우(lambda_HP v_EW^2 >> mu_Phi^2):

$$M_{\text{SFE}} = m_\Phi = v_{\text{EW}} \sqrt{\lambda_{\text{HP}}} \tag{12}$$

**포탈 결합의 결정:** M_SFE = v_EW delta를 만족하려면:

$$\lambda_{\text{HP}} = \delta^2 = (\sin^2\theta_W \cos^2\theta_W)^2 = 0.0316$$

이 값은 히그스 자체 결합(lambda_H ~ 0.13) 대비 ~0.24 비율로, 미세조정 없이 자연스럽다.

**lambda_HP = delta^2의 물리적 기원:** 포탈 결합 |H|^2 Phi^2는 4개의 장 연산자를 포함한다. 각 쌍(|H|^2와 Phi^2)이 전자약 중성류를 통해 결합할 때, 결합 강도는 delta = sin^2 theta_W cos^2 theta_W이다. 두 쌍의 곱으로 lambda_HP = delta x delta = delta^2가 된다.

**f(theta_W)의 유일성 검증:** M_SFE = v_EW f(theta_W)에서 가능한 모든 f를 테스트하면:

| f(theta_W) | 값 | Delta a_mu (x10^{-11}) | 텐션 |
|---|---|---|---|
| sin^2 theta_W | 0.231 | 147.2 | 2.1 sigma |
| cos^2 theta_W | 0.769 | 13.3 | 4.9 sigma |
| **delta = sin^2 cos^2** | **0.178** | **249.0** | **0.0 sigma** |
| sqrt(delta) | 0.422 | 44.3 | 4.3 sigma |
| delta^2 | 0.032 | 7880.3 | 159 sigma |

delta는 (1) 양 극한(sin theta_W -> 0, cos theta_W -> 0)에서 소멸하고, (2) 우주론적 D_eff에 동일 값이 나타나며, (3) g-2를 정확히 재현하는 유일한 조합이다.

**QFT 일관성:** 히그스 포탈 스칼라는 BSM 물리학에서 가장 잘 연구된 확장 모형이다.

- 재규격화 가능(renormalizable): 모든 결합이 dimension <= 4
- 유니타리(unitary): 스핀 0, 고에너지 산란 진폭 유한
- 게이지 이상 자유(anomaly-free): Phi는 게이지 단일항(singlet)

따라서 억압장의 QFT 일관성은 히그스 포탈 동정을 통해 자동으로 보장된다.

**충돌기 예측:**

비가시 히그스 붕괴(H -> Phi Phi):

$$\text{BR}(H \to \text{invisible}) \simeq \frac{\lambda_{\text{HP}}^2 v_{\text{EW}}^2}{8\pi M_H \Gamma_H} \approx 0.005$$

현재 LHC 한계(BR < 0.11)와 양립하며, HL-LHC(감도 ~0.025)에서 테스트 가능하다.

Phi-Higgs 혼합각:

$$\sin^2\theta_{\text{mix}} \simeq \left(\frac{\lambda_{\text{HP}} v_{\text{EW}}}{M_H}\right)^2 \approx 0.004$$

현재 한계(sin^2 theta < 0.01, m_Phi ~ 44 GeV)와 양립한다.

## 11. 양성자 반경 퍼즐

### 11.1 문제

CREMA 공동연구팀의 뮤온 수소 분광 측정(2010)은 전자 수소보다 작은 양성자 반경을 보고하였다.

| 측정 | r_p (fm) |
|---|---|
| 전자 수소 (CODATA) | 0.8751 |
| 뮤온 수소 (CREMA) | 0.84087 |
| 차이 | 0.034 fm |

$$\Delta r_p^2 = r_p^2(e) - r_p^2(\mu) = 0.0587 \text{ fm}^2$$

### 11.2 기하학적 접근 (정량적 불충분)

SFE 기하학적 보정은 파동함수의 원점값을 수정한다:

$$\frac{|\psi_{\text{SFE}}(0)|^2}{|\psi_{\text{QED}}(0)|^2} = 1 + C \cdot \left(\frac{m_r}{M_{\text{SFE}}}\right)^2$$

뮤온 수소의 환산 질량 m_r = 94.96 MeV에 대해 (m_r / M_SFE)^2 = 4.71 x 10^{-6}이다. 이것은 관측된 delta r / r ~ 0.039보다 약 16,000배 작다. 순수 루프 수준의 기하학적 억압은 양성자 반경 퍼즐의 정량적 크기를 설명하기에 불충분하다.

그러나 정성적으로, SFE는 올바른 방향을 예측한다: 뮤온 수소에서의 억압이 전자 수소보다 (m_r_mu / m_r_e)^2 ~ 34,000배 강하므로, r_p(mu) < r_p(e)이다.

### 11.3 Yukawa 퍼텐셜 접근 (보손 질량 필요)

억압장이 물리적 스칼라 보손(질량 m_phi)으로 매개되면, 렙톤-핵자 간 Yukawa 퍼텐셜이 발생한다:

$$V(r) = -\frac{g_\mu g_p}{4\pi} \frac{e^{-m_\phi r}}{r}$$

뮤온 수소의 Lamb shift에 대한 보정은 겉보기 양성자 반경의 이동으로 나타난다:

$$\Delta r_p^2 = \frac{3 g_\mu g_p}{2\alpha \, m_\phi^2} \tag{9}$$

여기서 g_mu = 6.27 x 10^{-4}, g_p = 5.57 x 10^{-3}은 10절에서 기하학적으로 결정된 값이다.

관측된 Delta r_p^2 = 0.0587 fm^2을 식 (9)에 대입하여 m_phi를 풀면:

$$m_\phi = \sqrt{\frac{3 g_\mu g_p}{2\alpha \cdot \Delta r_p^2}} \cdot \hbar c = 21.8 \text{ MeV}$$

### 11.4 자기일관적 해

기하학적 결합상수(g_mu = 6.27 x 10^{-4})로 고정한 채 보손 질량만 조정하면 g-2에 유한 질량 보정이 생긴다. g_mu와 m_phi를 동시에 조정하여 g-2와 양성자 반경을 모두 만족하는 자기일관적 해를 반복법으로 구한다.

**수렴된 해:**

| 파라미터 | 값 |
|---|---|
| 보손 질량 m_phi | 29.6 MeV |
| 상호작용 범위 1/m_phi | 6.7 fm |
| 뮤온 결합 g_mu | 8.52 x 10^{-4} |
| 양성자 결합 g_p | 7.57 x 10^{-3} |
| 전자 결합 g_e | 4.12 x 10^{-6} |

**예측:**

| 관측량 | SFE | 실험 | 텐션 |
|---|---|---|---|
| Delta a_mu (뮤온 g-2) | 249.0 x 10^{-11} | 249 +/- 48 x 10^{-11} | 0.00 sigma |
| Delta r_p^2 (양성자 반경) | 0.0587 fm^2 | 0.0587 fm^2 | 0.00 sigma |
| Delta a_e (전자 g-2) | ~10^{-13} | < 3600 x 10^{-14} | 안전 |

### 11.5 0 sigma의 정직한 해부

자기일관적 해가 두 관측량 모두에서 0 sigma를 주는 것은 놀랍지 않다.

**(A) 기하학적 g-2 예측 (진정한 0-파라미터):** 공식 (5)는 자유 파라미터 없이 실험의 중심값과 정확히 일치한다. 이것이 SFE의 진정한 예측력이다.

**(B) 자기일관적 해 (2방정식, 2미지수):** g_mu와 m_phi 두 파라미터로 두 관측량(g-2, 양성자 반경)을 피팅하면 0 sigma는 구성적으로 보장된다. 2-파라미터 모델이 2개 데이터 점을 맞추는 것은 자명하다.

**(C) 비자명한 것:** 자기일관적 해의 g_mu = 8.52 x 10^{-4}는 기하학적 예측 g_mu = 6.27 x 10^{-4}보다 36%만 크다. BSM 모델들이 통상 10~1000배의 조정을 요구하는 것에 비해, 이 O(1) 수준의 조정은 기하학적 프레임워크가 근본적으로 건전함을 시사한다.

### 11.6 파라미터 예산 요약

| 자유 파라미터 수 | 피팅 대상 | 설명 |
|---|---|---|
| 0 | g-2 | 기하학만으로 249.0 x 10^{-11} (정확 일치) |
| 1 (m_phi) | g-2 + r_p | 보손 질량 하나로 두 관측량 동시 해결 |

| 비교: SM | g-2, r_p | 표준모형은 두 문제 모두 미해결 |

## 11.7 해석 B: 보손 없는 기하학적 프레임워크

억압장의 효과를 물리적 보손(입자)이 아닌 시공간 기하학의 상관 구조로 해석할 수 있다. 이 경우 수학적 예측은 동일하지만, 물리적 존재론이 근본적으로 다르다.

**핵심 재해석:**

| 입자 언어 | 기하학 언어 | 값 |
|---|---|---|
| 보손 질량 m_phi | 상관 길이 xi_SFE = hbar_c / (m_p delta^2) | 29.65 MeV / 6.65 fm |
| Yukawa 커플링 g | 접힘 강도 kappa m_f | 5.93 x 10^{-6} MeV^{-1} |
| 폼팩터 F | 암흑 섹터 비율 (DE+DM)/DE | 1.370 |
| Yukawa 퍼텐셜 | 억압장 상관 함수 C(r) ~ exp(-r/xi)/r | exp(-r/6.65)/r |
| 1-loop 파인만 다이어그램 | 경로적분 접힘 | 동일한 수학 |

**상관 길이의 물리적 의미:**

$$\xi_{\text{SFE}} = \frac{\lambda_C(p)}{\lambda_{HP}} = \frac{\hbar c / m_p}{\delta^2} = \frac{0.2103 \text{ fm}}{0.03160} = 6.65 \text{ fm}$$

상관 길이 = 양성자 콤프턴 파장 / 포탈 커플링. 이것은 양성자의 양자적 확장 범위가 전자약 혼합(delta^2)에 의해 감쇠되기 전까지의 거리이다. 억압장의 상관 함수가 이 거리 이내에서만 유의미하게 작용한다.

**일반상대론과의 유비:**

일반상대론에서 중력은 중력자(graviton)가 매개하는 힘이 아니라, 시공간의 곡률이다. 중력자는 같은 물리학의 유효적 기술이다. SFE에서 억압장은 경로적분의 기하학적 접힘이며, 스칼라 보손은 같은 물리학의 유효적 기술이다. 억압장이 양자역학에 대한 관계는, 곡률이 중력에 대한 관계와 같다: 배경의 기하학적 성질이지, 전파하는 자유도가 아니다.

**실험적 구별:**

| 해석 A (보손 실재) | 해석 B (기하학) |
|---|---|
| PADME/NA64에서 29.65 MeV 공명 | 공명 없음 |
| 날카로운 피크 (Breit-Wigner) | 매끄러운 폼팩터 |
| 붕괴 산물 (e+e-, gamma gamma) | 산란 진폭 변형만 |
| q^2 = m_phi^2에서 극점 | q^2 ~ (1/xi)^2에서 완만한 변화 |

두 해석 모두 g-2, 양성자 반경, 우주론적 예측에서 동일한 수치를 준다. 보손 직접 생성 실험이 결정적 판별 수단이다.

### 12.1 X17 입자 (ATOMKI)

헝가리 ATOMKI에서 2015-2016년, 핵 전이(Be-8, He-4, C-12)에서 전자-양전자 쌍의 각도 분포에 이상 피크가 발견되었다. 이를 미지의 경량 보손(X17)으로 해석하며, 질량은 16.7-17.0 MeV이다.

| 실험 | 결과 |
|---|---|

| PADME (INFN, 2025.4) | 17 MeV에서 2-sigma 과잉 |

|---|---|---|---|

질량이 정확히 일치하지는 않으나 동일 자릿수(10-30 MeV)에 위치한다. SFE는 X17을 참조하지 않고 순수 기하학적 계산에서 이 영역을 독립적으로 도출하였다.

### 12.2 Electrophobic Scalar 제안

Liu, McKeen, Miller (Physical Review Letters 117, 101801, 2016)는 뮤온과 양성자에 선택적으로 결합하는 "electrophobic" 스칼라 보손을 제안하였다. 이 보손은 질량 100 keV - 100 MeV 범위에서 뮤온 g-2와 양성자 반경 퍼즐을 동시에 해결할 수 있다. 이것은 SFE 억압보손의 표준모형적 대응물이다. SFE가 다른 점은 결합상수를 기하학적으로 고정한다는 것이다.

### 12.3 현재 실험적 제약

10-30 MeV 스칼라 보손을 탐색하는 주요 실험:

| 실험 | 방법 | SFE 영역 제약 |
|---|---|---|

## 13. 검증 가능한 독립 예측

### 13.1 우주론적 예측

**예측 1: 바리온 밀도 정밀값.** SFE가 예측하는 Omega_b = 0.04865는 CMB-S4 (2030년경)가 관측 정밀도를 +/- 0.0003까지 향상시키면 직접 검증 가능하다.

**예측 2: alpha_s의 우주론적 결정.**

$$\alpha_s = \frac{\Omega_{DM}}{\pi \cdot \Omega_\Lambda} \tag{10}$$

입자물리(가속기)와 우주론(CMB)을 연결하는 새로운 관계식이다. 현재 우주론적 역산값 0.1204와 가속기 측정값 0.1179 +/- 0.0009는 2.7 sigma 차이이며, DESI가 Omega_DM, Omega_Lambda 정밀도를 높이면 결정적으로 검증된다.

**예측 3: sin^2 theta_W의 우주론적 결정.**

$$\sin^2\theta_W = \frac{1 - \sqrt{1 - 4(D_{\text{eff}} - 3)}}{2}$$

**예측 4: Hubble Tension 해소.** 억압장의 유효 퍼텐셜이 우주 곡률(Ricci scalar)에 의존하면:

$$V_{\text{eff}}(\Phi, R) = V(\Phi) + \xi R \Phi^2$$

xi ~ 2.5로 초기 우주(H_0 = 67.36)와 후기 우주(H_0 = 73.04)의 불일치를 해소할 수 있다.

**예측 5: CMB 대규모 다중극 진동.**

$$P(k) \propto k^{n_s - 1} \cdot \left(1 + \varepsilon^2 \cdot \text{sinc}(k \cdot \xi_\Phi)\right)$$

진폭 ~5%, CMB ell < 30 영역에서 관측 가능하다.

### 13.2 입자물리적 예측

**예측 6: 뮤온 g-2 중심값.**

$$\Delta a_\mu^{\text{SFE}} = 249.0 \times 10^{-11} \tag{5}$$

Fermilab 최종 결과(Run-4/5/6)와 격자 QCD HVP 계산의 수렴이 결정적이다.

**예측 7: 전자 g-2 보정.**

$$\Delta a_e^{\text{SFE}} = 5.82 \times 10^{-14}$$

차세대 전자 g-2 실험이 이 영역(~10^{-14})에 도달하면 독립 검증 가능하다.

**예측 8: g-2로부터의 와인버그 각.**

$$\delta = \sqrt{\frac{\alpha \, m_\mu^2}{2\pi e \, v_{\text{EW}}^2 \, \Delta a_\mu}} \quad \Rightarrow \quad \sin^2\theta_W = 0.23122$$

뮤온 g-2 실험으로부터 와인버그 각을 독립적으로 결정하는 새로운 방법.

**예측 9: 억압보손 탐색.** SFE가 예측하는 스칼라 보손의 파라미터 공간:

| 속성 | 범위 |
|---|---|
| 질량 | 22-30 MeV |
| 스핀 | 0 (스칼라) |
| 뮤온 결합 | 6.3 x 10^{-4} - 8.5 x 10^{-4} |
| 양성자 결합 | 5.6 x 10^{-3} - 7.6 x 10^{-3} |
| 전자 결합 | 3.0 x 10^{-6} - 4.1 x 10^{-6} |

이 영역은 PADME, NA64-mu, 차세대 빔덤프 실험에서 탐색 가능하다.

### 13.3 충돌기 예측 (히그스 포탈)

**예측 10: 비가시 히그스 붕괴.** 히그스 포탈 결합 lambda_HP = delta^2 = 0.032에서:

$$\text{BR}(H \to \Phi\Phi) \approx 0.005$$

현재 LHC 한계 BR < 0.11(95% CL)과 양립. HL-LHC(감도 ~0.025)에서 테스트 가능.

**예측 11: Phi-Higgs 혼합.** 혼합각 sin^2 theta_mix ~ 0.004. 현재 제약(sin^2 theta < 0.01, m_Phi ~ 44 GeV)과 양립. 차세대 e+e- 충돌기(FCC-ee, CEPC)에서 정밀 히그스 결합 측정으로 도달 가능.

## 14. 동적 암흑에너지: 억압장 퀸테센스

### 14.1 DESI DR2의 동적 암흑에너지 신호

DESI DR2 (2025) BAO + CMB + DESY5 결합 분석은 암흑에너지 상태방정식이 w = -1(우주상수)에서 유의미하게 벗어남을 보고하였다.

| | 값 | 95% CL 범위 |
|---|---|---|
| w0 | -0.770 | [-0.881, -0.651] |
| wa | -0.782 | [-1.30, -0.34] |
| LCDM 배제 | 3.1 sigma | BAO+CMB |

CPL 매개변수화 w(a) = w0 + wa(1-a)에서 (w0, wa) = (-1, 0)이 95% CL 밖에 위치한다. 이는 정적 SFE(w = -1 정확히)와 직접적 긴장을 생성한다.

### 14.2 포텐셜의 수학적 동치

DESI 데이터를 가장 잘 설명하는 퀸테센스 모형은 Higgs-like 포텐셜이다 (arXiv:2506.21542):

$$V(\phi) = V_0 + \frac{1}{2}m^2\phi^2 + \frac{1}{4}\lambda\phi^4$$

Best fit: |m^2| > 251 H0^2, v_phi >= 0.08 M_Pl.

이것은 SFE 억압장의 라그랑지안(2.2절)과 **수학적으로 동치**이다:

$$\mathcal{L}_\Phi = \frac{1}{2}\partial_\mu \Phi \partial^\mu \Phi - V(\Phi), \quad V(\Phi) = -\frac{1}{2}\mu^2 \Phi^2 + \frac{1}{4}\lambda \Phi^4$$

멕시코 모자 포텐셜에서 m^2 = -mu^2 < 0 (자발적 대칭 깨짐), V0는 우주상수 기여. DESI가 선호하는 퀸테센스 포텐셜이 SFE의 억압장 포텐셜 그 자체이다.

### 14.3 정적 예측의 재해석: 점근적 끌개

정적 SFE의 bootstrap 방정식 eps^2 = exp(-(1-eps^2)D_eff)는 억압장이 VEV에 완전히 안착한 평형 상태의 해이다. 이 해를 **점근적 끌개(attractor)** 값으로 재해석한다.

물리적 그림:
- **미시적 스칼라** (M_SFE = 44 GeV): VEV에 안착. 입자물리 예측(g-2, 히그스 포탈)을 지배.
- **우주론적 응축체**: 대규모 결맞음 모드(coherent mode). 아직 VEV로 완전히 rolling하지 못함.

w(z) != -1은 우주론적 응축체가 아직 포텐셜의 최솟값에 도달하지 않았다는 신호이다. z -> infinity에서 응축체가 VEV에 안착하면 w -> -1이 되고, 정적 SFE 예측이 정확해진다.

현재(z=0)에서의 보정:

$$\delta\Omega_b \sim O(1+w_0) \times \Omega_b \sim O(0.01)$$

정적 예측은 O(1%) 수준의 보정으로, 현재 관측 오차 범위 내에서 유효하다.

### 14.4 결합 강도와 DESI 호환성

억압장의 비최소 중력 결합(non-minimal coupling)을 xi R Phi^2으로 매개변수화하면, 섭동론적 분석에서:

$$1 + w_0 \simeq \frac{2\xi^2}{3\,\Omega_\Lambda}$$

DESI 측정 1+w0 ~ 0.23을 재현하려면 xi ~ 0.49가 필요하다.

| xi 후보 | 값 | 예측 1+w0 | DESI 호환 |
|---|---|---|---|
| delta^2 | 0.032 | 0.001 | 불충분 |
| delta | 0.178 | 0.030 | 불충분 |
| sqrt(delta) | 0.422 | 0.171 | 근접 (1.3 sigma) |
| 1/6 (등각 결합) | 0.167 | 0.027 | 불충분 |
| **필요 xi** | **0.489** | **0.230** | **정합** |

sqrt(delta)가 가장 근접하며, 다음 관계를 시사한다:

$$\xi = \sqrt{\delta} = \sqrt{\sin^2\theta_W \cos^2\theta_W} = \frac{1}{2}|\sin(2\theta_W)|$$

이 값은 전자약 혼합각의 이중각(double angle) 사인의 절반에 해당하며, delta의 기하학적 평균근이다.

### 14.5 이론적 의의

1. **구조적 자연스러움:** SFE가 독립적으로 도입한 멕시코 모자 포텐셜이 DESI의 데이터 주도적(data-driven) 분석에서 선호되는 퀸테센스 포텐셜과 동치이다. 이는 SFE의 포텐셜 구조가 관측에 의해 간접적으로 지지됨을 의미한다.

2. **위계 보존:** 미시적 스칼라(입자물리)와 우주론적 응축체(암흑에너지)는 동일 포텐셜의 서로 다른 모드이다. 미시적 모드는 VEV에 안착하여 g-2를 결정하고, 우주론적 모드는 아직 rolling 중이어서 w != -1을 발생시킨다.

3. **새로운 과제:** 비최소 결합 xi의 제1원리 유도. xi = sqrt(delta)가 DESI에 근접하나, 정확한 값의 이론적 정당화가 필요하다.

## 15. 전자약-강력 통합 관계: sin(theta_W) = 2 alpha_s^(2/d)

### 15.1 발견된 관계식

동적 암흑에너지의 비최소 결합 xi를 전수 스캔하는 과정에서, 다음의 관계식이 발견되었다:

$$\sin\theta_W = 2\,\alpha_s^{2/d}, \quad d = 3$$

수치적으로:

| | 값 | 출처 |
|---|---|---|
| sin(theta_W) | 0.48085 | PDG (sin^2 = 0.23122) |
| 2 * alpha_s^(2/3) | 0.48115 | alpha_s = 0.1180 |
| 차이 | 0.06% | 0.12 sigma (alpha_s 오차 내) |

동치 표현:

$$\sin^2\theta_W = 4\,\alpha_s^{4/3}, \quad \alpha_s = \left(\frac{\sin\theta_W}{2}\right)^{3/2}$$

이 관계식은 M_Z = 91.2 GeV 스케일에서 성립하며, 에너지 스케일을 이동하면 점진적으로 벗어난다 (1-loop RG running 분석). 이는 고에너지 통합(GUT)이 아니라, **전자약 스케일에서의 매칭 조건**임을 시사한다.

**정리 (SFE 전자약-강력 결합 관계):** 다음 전제로부터:
- (P1) SFE 경로적분이 d차원에서 곱적으로 분해: 차원당 결합 = alpha_s^(1/d)
- (P2) 전자약 혼합은 이중선형(2-vertex) 게이지 과정
- (P3) SU(2)_L 기본 표현 차원 N_w = 2
- (P4) 색 전하 수 = 공간 차원: N_c = d = 3

다음이 유도된다:
$$\sin\theta_W = N_w \cdot (\alpha_s^{1/N_c})^2 = 2\,\alpha_s^{2/3}$$

이것은 M_Z에서의 매칭 조건이다. 유도의 상세 분석: `examples/physics/derive_weinberg.py`

### 15.2 기존 대통일 이론(GUT)과의 비교

| | 표준 GUT (SU(5), SO(10)) | SFE 관계식 |
|---|---|---|
| 관계 성립 스케일 | M_GUT ~ 10^16 GeV | M_Z ~ 91 GeV (측정 스케일) |
| 관계 형태 | 군론적 비율 (sin^2 = 3/8) | 거듭제곱 법칙 (sin^2 = 4 alpha_s^{4/3}) |
| SUSY 필요 여부 | YES (정밀 매칭에 필수) | NO |
| 독립 결합 상수 수 | 3개 (g1, g2, g3) -> 1개 at M_GUT | 1개 (alpha_s) + 차원 d |
| 차원 구조 | 없음 | 지수 2/d가 공간 차원 직접 반영 |

### 15.3 지수 2/d의 물리적 해석

d=3 공간에서만 관측된 sin^2(theta_W) ~ 0.231을 재현한다:

| d | 2/d | sin^2(tW) = 4 alpha_s^(4/d) | 비고 |
|---|---|---|---|
| 2 | 1.000 | 0.056 | sin > 1 가능 (비물리적) |
| **3** | **0.667** | **0.232** | **관측 일치** |
| 4 | 0.500 | 0.472 | 너무 큼 |
| 5 | 0.400 | 0.724 | 너무 큼 |

물리적 해석 -- 이론적 유도 (15.3.1절):

**(A) 경로적분의 곱적 분해:** d차원 경로적분 측도 D[x] = prod D[x_i]에서, 작용이 차원별로 분해되면 진폭이 곱으로 분해된다: e^(-S) = prod e^(-S_i). 강한 결합 alpha_s가 d차원에서 작동할 때, 차원당 결합은 alpha_s^(1/d)이다. 이것은 총 결합이 차원당 결합의 d제곱이라는 곱적 구조에서 따른다.

**(B) 이중선형 혼합 꼭짓점:** 전자약 혼합은 W^3_mu와 B_mu의 2x2 혼합 행렬이다. 이것은 게이지 장에 대해 이중선형(bilinear)이므로, k=2 꼭짓점 과정이다. 차원 분해된 진폭: V(2,d) = (alpha_s^(1/d))^2 = alpha_s^(2/d).

**(C) SU(2) 이중항 인자:** sin(theta_W)은 혼합 진폭이며, SU(2)_L 기본 표현의 차원 N_w = 2가 곱해진다. 따라서 sin(theta_W) = N_w * alpha_s^(2/d) = 2 * alpha_s^(2/d).

**(D) N_c = d 동일성:** 유도에서 d = 3은 공간 차원이 아니라 색 전하의 수 N_c = 3으로도 해석 가능하다: sin(theta_W) = dim(SU(2)_fund) * alpha_s^(n_vertex / dim(SU(3)_fund)). 공간 차원과 색 전하의 수가 동일하다는 것은 SFE 경로적분의 차원별 분해가 QCD의 색 분해와 동일한 구조임을 시사한다.

### 15.4 N_c = d의 심층 유도: Hodge 자기쌍대성

(P4) "색 전하의 수 = 공간 차원 수"가 왜 성립하는가. 이것은 수치적 우연이 아니라, 3차원 공간의 위상적 성질에서 유도된다.

**정리 (Hodge 자기쌍대성에 의한 d=3 선택):**

d차원 공간에서:
- 벡터(1-형식): d개 성분
- 반대칭 2-텐서(2-형식): d(d-1)/2개 성분

자기쌍대 조건 d = d(d-1)/2의 해를 구하면:

| d | 벡터 | 2-텐서 | 동일? |
|---|---|---|---|
| 2 | 2 | 1 | X |
| **3** | **3** | **3** | **O** |
| 4 | 4 | 6 | X |
| 5 | 5 | 10 | X |
| 6 | 6 | 15 | X |

**d = 3이 유일한 해**이다.

**물리적 의미:** d=3에서 Levi-Civita 텐서 eps_ijk가 3개의 지표를 가지며, 이것이:
- 벡터 쌍을 벡터로 사상한다 (외적: a x b = eps_ijk a_j b_k)
- 이 사상은 d=3과 d=7에서만 존재한다 (d=7은 비결합성으로 비물리적)
- so(3) 리대수를 R^3과 동일시한다 (각운동량이 벡터가 되는 유일한 차원)

게이지 이론에서:
- 게이지장 A_i의 공간 성분: d개
- 장세기 F_ij의 공간 성분: d(d-1)/2개
- d=3에서만 이 둘이 같은 수의 성분을 가진다
- 따라서 색 전하의 기본 표현 차원 N_c = d = 3이 자연스럽다

회전군에서도 동일한 결론이 나온다. SO(d)의 생성원 수 = d(d-1)/2이므로:

| 군 | 생성원 수 | = d? |
|---|---|---|
| SO(2) | 1 | X |
| SO(3) | **3** | **O** |
| SO(4) | 6 | X |
| SO(5) | 10 | X |

SO(3)만이 "생성원 수 = 차원 수"를 만족한다. 이것이 각운동량을 벡터로 만들고, SU(2) (SO(3)의 이중 피복)의 기본 표현 차원 N_w = 2를 결정한다.

### 15.5 게이지 군 구조의 유도: 왜 SU(3) x SU(2) x U(1)인가

d=3으로부터 표준모형의 게이지 군이 유도된다.

**삼각 분해 (Triangular Decomposition):**

d차원 공간의 하강 분할(descending partition): {d, d-1, ..., 1}

| d | 분할 | 합 | d(d+1)/2 | 게이지 군 |
|---|---|---|---|---|
| 2 | {2, 1} | 3 | 3 | SU(2) x U(1) |
| **3** | **{3, 2, 1}** | **6** | **6** | **SU(3) x SU(2) x U(1) = 표준모형** |
| 4 | {4, 3, 2, 1} | 10 | 10 | SU(4) x SU(3) x SU(2) x U(1) |

d=3의 분할 {3, 2, 1}이 정확히 표준모형의 게이지 군이다.

**왜 하강 분할인가 (게이지 중첩):**

SFE 경로적분이 d=3 차원에서 분해될 때, 각 차원의 제거가 하나의 게이지 인자를 생성한다:

$$SU(3) \supset SU(2) \supset U(1)$$
$$\text{3차원} \rightarrow \text{2차원} \rightarrow \text{1차원}$$

이것은 SU(3)의 최대 하강 쇄(maximal descending chain)이며, 수학적으로 **기 다양체(flag manifold)** SU(3) / (SU(2) x U(1))에 대응한다. 표준모형의 게이지 군은 SU(3)의 등방성 쇄(isotropy chain)이다.

**추가 일치:**

쿼크의 상태 수 = N_c x N_w = 3 x 2 = 6 = d(d+1)/2 (삼각수). 세대당 쿼크 자유도의 수가 동일한 삼각수로 결정된다.

### 15.6 강력-약력 쌍대성

Weinberg 각 관계식을 변환하면, 강한 결합과 약한 혼합 사이의 쌍대성이 드러난다:

$$\alpha_s^{N_w} = \left(\frac{\sin\theta_W}{N_w}\right)^{N_c}$$

수치적으로:

$$\alpha_s^2 = \left(\frac{\sin\theta_W}{2}\right)^3$$
$$0.013900 = 0.013898 \quad (\text{차이 0.002\%})$$

물리적 해석:
- 좌변: 강한 결합의 N_w = 2 제곱
- 우변: 정규화된 약한 혼합 진폭의 N_c = 3 제곱
- 지수가 서로 교환된다: (N_w, N_c) <-> (N_c, N_w)
- 이것은 강력과 약력 사이의 쌍대 변환이며, 공간 차원 d = N_c = 3이 매개한다

역관계:

$$\alpha_s = \left(\frac{\sin\theta_W}{2}\right)^{3/2}, \quad \text{지수 } \frac{3}{2} = \frac{N_c}{N_w} = \frac{d}{N_w}$$

sin(theta_W) = 0.48085로부터 alpha_s = 0.11789를 재구성하면, 관측값 0.1179와 0.009% 이내로 일치한다.

### 15.7 세대 수의 동기부여: N_gen = d = 3

관측: 페르미온 세대 수 = 3 = d. 이것은 d=3에서 동기부여된다:

**(A) Levi-Civita 구조:** eps_ijk의 3개 짝순열(123, 231, 312)이 3개의 독립적 "배향"에 대응하며, 각 배향이 하나의 세대를 생성한다.

**(B) CKM 행렬과 CP 위반:** CP 위반에는 N_gen >= 3이 필요하다 (Kobayashi-Maskawa). CKM 행렬의 CP 위반 위상 수 = (N-1)(N-2)/2이므로:
- N_gen = 2: 위상 0개 (CP 보존)
- N_gen = 3: 위상 1개 (최소 CP 위반)

N_gen = d = 3은 CP 위반이 가능한 **최소** 구조이다.

**(C) 이상(anomaly) 상쇄:** SFE 경로적분의 d=3 차원 분해에서, 모든 차원에 걸친 게이지 불변성을 유지하려면 정확히 d개의 페르미온 복사본이 필요하다.

상태: N_gen = d = 3은 강하게 동기부여되나, SFE 공리만으로 엄밀하게 유도되지는 않았다.

### 15.8 페르미온 질량비에서의 alpha_s^(n/3) 패턴

관계식의 체계적 스캔에서, 페르미온 질량비가 alpha_s^(n/3) (n: 정수)로 표현되는 패턴이 추가로 발견되었다:

| 질량비 | 값 | alpha_s^(n/3) | n | 차이 |
|---|---|---|---|---|
| m_e / m_u | 0.2366 | alpha_s^(2/3) = 0.2406 | 2 | 1.7% |
| m_mu / m_tau | 0.0595 | alpha_s^(4/3) = 0.0579 | 4 | 2.7% |
| m_u / m_c | 0.0017 | alpha_s^3 = 0.0016 | 9 | 3.4% |
| m_c / m_t | 0.0074 | alpha_s^(7/3) = 0.0068 | 7 | 7.1% |
| m_b / m_t | 0.0242 | alpha_s^(5/3) = 0.0284 | 5 | 17% |

이 패턴이 우연인지 근본적인지는 추가 연구가 필요하나, n/3 = n/d 구조가 일관되게 나타나는 점이 주목할 만하다.

### 15.9 Koide 공식과의 연결

렙톤 질량의 Koide 공식 Q = (m_e + m_mu + m_tau) / (sqrt(m_e) + sqrt(m_mu) + sqrt(m_tau))^2 = 2/3은 0.0006% 정밀도로 성립한다. 이 2/3은 SFE의 2/d (d=3)와 동일한 값이며, 차원 구조가 페르미온 질량 스펙트럼에도 작용할 가능성을 시사한다.

### 15.10 양성자 반경 퍼즐의 해결

통합 SFE 프레임워크(d=3, alpha_s)에서 억압 보손 질량과 커플링 증강 인자가 자연스럽게 유도되어, 양성자 반경 퍼즐이 자유 매개변수 0개로 해결된다.

**보손 질량 결정:** 억압 보손의 질량은 양성자 질량에 힉스 포탈 커플링 lambda_HP = delta^2 를 곱한 값이다:

$$m_\phi = m_p \cdot \delta^2 = m_p \cdot \lambda_{HP} = 938.27 \times 0.03160 = 29.65 \text{ MeV}$$

물리적 해석: 억압 보손은 핵자 스케일을 힉스 포탈 상호작용 강도로 억압한 것이다. delta^2 = lambda_HP(10.5절)이므로, 보손 질량이 포탈 구조로부터 직접 결정된다.

**폼팩터:** 유효 커플링에는 폼팩터 F = 1 + alpha_s D_eff = 1 + DM/DE 가 곱해진다:

$$g_f^{\text{eff}} = \kappa \cdot m_f \cdot (1 + \alpha_s \pi) = \kappa \cdot m_f \cdot \frac{\Omega_\Lambda + \Omega_{DM}}{\Omega_\Lambda}$$

여기서 1 + alpha_s D_eff = 1 + Omega_DM / Omega_Lambda = 1.375이다. 이 인자의 물리적 의미: 보손-페르미온 상호작용에 암흑물질 성분이 기여하여 유효 커플링을 증강시킨다. 기하학적 커플링 kappa m_f 는 암흑에너지만의 기여이고, 전체 비바리온 성분(DE+DM)을 포함하면 인자 (1+DM/DE)가 곱해진다.

**수치 검증:**

| 관측량 | SFE 예측 | 관측값 | 일치도 |
|---|---|---|---|
| m_phi | 29.65 MeV | 22-30 MeV (범위) | 범위 내 |
| Da_mu (F 포함) | 253.2 x 10^{-11} | 249 +/- 48 | 0.09 sigma |
| Delta r_p^2 | 0.0597 fm^2 | 0.0587 +/- 0.0033 fm^2 | 1.6% (0.3 sigma) |
| r_p(muonic) | 0.8403 fm | 0.84087 +/- 0.00039 fm | 1.4 sigma |

**핵심 구조:** g-2와 양성자 반경 차이의 비 R = Delta r_p^2 / Delta a_mu 는 커플링에 무관하다:

$$R = \frac{12\pi^2 m_p}{\alpha_{em} \cdot m_\mu \cdot (m_\phi/\hbar c)^2 \cdot I(m_\phi/m_\mu)}$$

m_phi = m_p delta^2 이면 R_theory / R_obs = 0.9992 (0.08% 정합). 이는 m_phi가 결정되면 g-2 하나만으로 양성자 반경이 자동 결정됨을 의미한다. 자유 매개변수는 0개다.

**오차 전파:** alpha_s = 0.1179 +/- 0.0009로부터:
- m_phi = 29.65 +/- 0.42 MeV
- Delta r_p^2 = 0.0597 +/- 0.0023 fm^2 (관측: 0.0587 +/- 0.0033)

이론 오차와 실험 오차가 겹치므로, 현재 정밀도에서 완전히 양립한다.

### 15.12 4개 힘의 통합: 중력은 접힘 자체이다

15.4-15.5절에서 d=3의 내림 분할 {3, 2, 1}이 게이지 힘 3개를 결정함을 보였다. 그러나 자연의 근본 힘은 **4개**다. 4번째 힘인 중력은 어디에서 오는가?

**핵심 통찰:** 중력은 내림 계단의 "층"이 아니다. 계단을 **내려가는 행위 자체**이다.

$$\text{SU(3)} \xrightarrow{\Phi} \text{SU(2)} \xrightarrow{\Phi} \text{U(1)}$$

여기서 화살표 Phi가 억압장이며, 이것이 곧 중력이다.

**수학적 동일성:** SFE의 억압장은 작용의 2차 변분이다:

$$\Phi \equiv \frac{\delta^2 S}{\delta \gamma^2}$$

일반상대론의 Jacobi 측지선 편차 방정식에서 이 2차 변분은 **리만 곡률 텐서**와 정확히 대응한다:

$$\frac{D^2 J}{ds^2} + R(J, T, T) = 0$$

억압장 = 시공간 곡률 = 중력. 이 세 가지는 동일한 수학적 대상이다.

**4개 힘의 구조:**

| 힘 | SFE에서의 정체 | 수학적 구조 |
|---|---|---|
| 강한 핵력 | d=3 차원 전체의 대칭 | SU(3) = 내림 계단 3층 |
| 약한 핵력 | 1차원 제거 후 2차원 대칭 | SU(2) = 내림 계단 2층 |
| 전자기력 | 2차원 제거 후 1차원 대칭 | U(1) = 내림 계단 1층 |
| **중력** | **차원을 제거하는 접힘 자체** | **Phi = delta^2 S / delta gamma^2 = 곡률** |

**힘의 수 공식:**

$$N_{\text{forces}} = d + 1$$

d차원에서 내림 분할은 d개의 게이지 힘을 생성하고, 분할을 매개하는 접힘이 1개의 중력을 생성한다. d=3에서:

$$N_{\text{forces}} = 3 + 1 = 4$$

이것은 왜 자연의 힘이 정확히 4개인지를 설명한다.

**중력이 양자화하기 어려운 이유:**

게이지 힘 3개는 내림 계단의 **각 층**이다 -- 고정된 대칭 구조 위에서 정의되므로 양자화가 가능하다. 중력은 계단을 **만드는 메커니즘** 자체이다 -- 시공간 구조를 변형하는 것이므로, 그 위에 정의된 양자장론의 틀 안에서 양자화하기가 본질적으로 어렵다. 중력은 무대 위의 배우가 아니라 무대 자체를 바꾸는 연출가이다.

이것은 기존 양자중력의 난제를 새로운 각도에서 조명한다: 중력을 "4번째 게이지 힘"으로 양자화하려는 시도(끈 이론, 루프 양자중력 등)가 난관에 부딪히는 이유는, 중력이 게이지 힘과 **같은 종류**가 아니기 때문이다. 중력은 게이지 구조를 **생성하는** 상위 메커니즘이다.

**전체 구조:**

$$d = 3 \xrightarrow{\text{Hodge 자기쌍대}} N_c = 3$$

$$\Phi = \frac{\delta^2 S}{\delta \gamma^2} = \text{곡률} = \text{중력 (4번째 힘)}$$

$$\Phi\text{의 내림 분할:} \quad \underbrace{SU(3)}_{\text{강한 힘}} \times \underbrace{SU(2)}_{\text{약한 힘}} \times \underbrace{U(1)}_{\text{전자기력}} \quad \text{(게이지 힘 3개)}$$

4개 힘 전부가 d=3과 억압장 Phi로부터 나온다. 이것은 대통일(GUT: 게이지 힘 3개의 통합)을 넘어, 중력을 포함한 **만물의 이론(ToE)**의 구조이다.

### 15.13 SFE 독립 입력의 축소

15.12절의 4개 힘 통합을 포함하면, sin^2(theta_W)은 alpha_s로부터 유도되며 SFE의 독립 입력이 3개에서 2개로 축소된다:

| | 기존 SFE | 통합 SFE |
|---|---|---|
| 독립 입력 | d=3, sin^2 theta_W, alpha_s | d=3, alpha_s |
| 입력 수 | 3 | 2 |
| 예측 수 | 6+ | 11+ (sin^2 theta_W, m_phi, r_p 포함) |

유도 체인:

$$d=3 \xrightarrow{\text{Hodge}} N_c = 3 \xrightarrow{\text{분할}} \underbrace{SU(3) \times SU(2) \times U(1)}_{\text{게이지 힘 3개}} + \underbrace{\Phi = \delta^2 S / \delta\gamma^2}_{\text{중력}} = \text{4개 힘}$$

$$\alpha_s \xrightarrow{4\alpha_s^{4/3}} \sin^2\theta_W \xrightarrow{\delta = s^2c^2} D_{\text{eff}} \xrightarrow{\text{bootstrap}} \Omega_b, \Omega_\Lambda, \Omega_{DM} \xrightarrow{\alpha_s^{1/3}} w_0$$

$$\delta^2 = \lambda_{HP} \xrightarrow{m_p \lambda_{HP}} m_\phi \xrightarrow{F=1+\alpha_s D_{\text{eff}}} \Delta a_\mu, \Delta r_p^2$$

강력-약력 쌍대성: $\alpha_s^{N_w} = (\sin\theta_W / N_w)^{N_c}$, 힘의 수: $N_{\text{forces}} = d + 1 = 4$

수치 검증: `examples/physics/derive_weinberg.py`, `examples/physics/derive_weinberg_deep.py`, `examples/physics/derive_gauge_structure.py`

## 17. 현재 상태와 남은 과제

### 17.1 달성된 것

독립 입력 2개(d=3, alpha_s)로부터 우주론과 입자물리를 관통하는 11개 이상의 정량적 예측을 확립하였다. sin^2(theta_W) = 4 alpha_s^(4/3) 관계식(15절)의 발견으로 독립 입력이 3개에서 2개로 축소되었으며, 이로부터 delta, D_eff, 우주 에너지 구성 3종(2% 이내), 뮤온 g-2(0.00 sigma), 암흑에너지 상태방정식 w0(DESI와 0.1% 정합)이 모두 유도된다.

Hodge 자기쌍대성 조건 d = d(d-1)/2에서 d=3이 유일한 해임을 증명하고(15.4절), 이로부터 N_c = d = 3 (색 전하 = 공간 차원)을 유도하였다. 하강 분할 {3,2,1}에 의해 표준모형의 게이지 군 SU(3) x SU(2) x U(1) 자체가 d=3에서 결정됨을 보였다(15.5절). 강력-약력 쌍대성 alpha_s^(N_w) = (sin(tW)/N_w)^(N_c)을 발견하였으며(15.6절), 세대 수 N_gen = 3 = d가 최소 CP 위반 구조에 대응함을 확인하였다(15.7절).

기하학적 접힘 공식과 스칼라 보손 1-loop 기여의 수학적 동치를 증명하여 EFT 해석을 제공하였다. 양성자 반경 퍼즐은 억압 보손 질량 m_phi = m_p delta^2 = 29.65 MeV와 폼팩터 F = 1+alpha_s pi = 1+DM/DE에 의해 자유 매개변수 0개로 해결된다(15.6절). Delta r_p^2 의 비율 R = Delta r_p^2 / Delta a_mu는 커플링에 무관하여, m_phi가 결정되면 g-2로부터 양성자 반경이 자동 결정된다(0.08% 정합). 억압장을 히그스 포탈 스칼라로 동정하여 QFT 일관성을 확보하고, DESI가 선호하는 퀸테센스 포텐셜이 SFE 억압장의 멕시코 모자 포텐셜과 수학적으로 동치임을 확인하였다.

**과제 4 (해결됨):** 억압장의 라그랑지안 (11)은 히그스 포탈 스칼라 모형으로, 재규격화 가능성(dimension <= 4), 유니타리성(spin-0), 게이지 이상 자유(gauge singlet)가 모두 표준 결과로 보장된다.

**과제 2 (해결됨):** M_SFE = v_EW sqrt(lambda_HP)에서 lambda_HP = delta^2 = 0.032이면 M_SFE = v_EW delta이다. lambda_HP = delta^2는 포탈 결합의 중성류 구조(delta per field pair)에서 유도되며, 히그스 자체 결합 대비 ~0.24 비율로 자연스럽다. f(theta_W)의 전수 검사에서 delta가 유일하게 g-2를 재현한다.

**과제 1 (해결됨):** 자기일관성 방정식의 형식적 유도가 완성되었다. 핵심 간극이었던 Step (iv) "억압장 에너지 비율 = 1 - epsilon^2"은 추가 가정이 아니라 SFE의 정의(억압장 = 암흑 부문)와 에너지 보존의 직접적 결과이다. 전체 유도: (A1) 경로적분 차원 분해 -> (A2) 에너지 보존 -> sigma = Omega_dark -> (A3) P_survive = Omega_b -> epsilon^2 = exp(-(1-epsilon^2)*D_eff). 억압 작용 Phi가 D_eff에 대해 연장적(extensive)임이 핵심이며, 이는 열역학의 "엔트로피 = 밀도 x 부피"와 동일한 구조이다 (3.2.1절 상세 유도 참조).

**과제 5 (해결됨):** alpha = alpha_s * pi 에서 alpha = alpha_s * D_eff로 정제되었다. 유도: 과제 1과 동일한 연장성 원리 -- 차원당 양자 요동 비율이 alpha_s이고, D_eff 차원에 걸친 총 요동이 alpha_s * D_eff이다. pi = 3.14159와 D_eff = 3.17776은 1.15% 차이로 현재 관측 오차(~2%) 이내이지만, 역산 검증에서 D_eff가 우월하다(alpha_s 역산 오차: 0.92% vs 2.09%). 이론적으로도 D_eff는 이미 SFE에서 유도된 양이므로 미설명 상수(pi)가 불필요해진다 (3.3절).

**과제 3 (해결됨):** Worldline 유도. Worldline 형식주의(Strassler 1992, Schubert 2001)에서 SFE 억압장은 점입자 작용의 수정 S_SFE = S_WL + int Phi(x(tau)) dtau 로 나타난다. 이것은 EFT의 히그스 포탈 스칼라 라그랑지안과 수학적으로 동치이며, 모든 루프 차수에서 동일한 물리적 예측을 준다. Worldline 형식주의의 가치는 계산적(다중 루프에 유리)이고 개념적(SFE 경로적분 언어와 자연스러운 연결)이나, 새로운 물리적 예측을 추가하지는 않는다.

**과제 7 (프레임워크 확립):** DESI DR2 (2025)의 동적 암흑에너지 신호(w0 ~ -0.77, 3.1 sigma)에 대응하여, SFE의 멕시코 모자 포텐셜이 DESI best-fit 퀸테센스 포텐셜과 수학적으로 동치임을 확인하였다 (14절). 정적 SFE 예측은 점근적 끌개값으로 재해석되며, 현재 관측과의 차이는 우주론적 응축체가 아직 VEV에 완전히 도달하지 않았기 때문이다. 비최소 결합 xi ~ 0.49의 제1원리 유도가 핵심 과제이다.

**과제 10 (해결됨):** 양성자 반경 퍼즐. 억압 보손 질량 m_phi = m_p delta^2 = m_p lambda_HP = 29.65 MeV가 전수 스캔 1위(60개 후보 중, 0.05% 정합)이며, 폼팩터 F = 1+alpha_s pi = 1+DM/DE = 1.370이 비바리온 에너지의 DE 대비 비율로 물리적 의미를 갖는다. 핵심 구조: R = Delta r_p^2 / Delta a_mu는 커플링에 무관하고 m_phi만으로 결정되므로, g-2와 양성자 반경은 동시 예측이다. R_theory/R_obs = 0.9992 (0.08% 정합). 이로써 SFE는 자유 매개변수 0개로 양성자 반경을 예측한다.

**과제 8 (신규):** 비최소 결합 xi의 결정. xi = sqrt(delta) = (1/2)|sin(2 theta_W)| = 0.422가 DESI 관측에 가장 근접하나(예측 1+w0 = 0.17 vs DESI 0.23), 정확한 값과의 ~1 sigma 잔차가 있다. xi가 전자약 파라미터의 어떤 조합인지, 또는 독립적으로 결정되는 상수인지 규명이 필요하다.

**과제 6 (외부, 업데이트):** Fermilab 최종 결과(2025.6): a_mu(exp) = 116592070.5(148) x 10^{-11}. Theory Initiative WP25(2025.5): 격자 QCD HVP를 채택하여 a_mu(SM) = 116592033(62) x 10^{-11}. 이 경우 Delta a_mu = 38 +/- 63 x 10^{-11} (0.6 sigma)이며, SFE 예측 249 x 10^{-11}은 3.3 sigma 과대예측이 된다. 그러나 WP20 데이터 기반 HVP와 WP25 격자 QCD HVP 사이에 223 x 10^{-11}의 불일치가 미해결이며(CMD-3 vs BaBar/KLOE 텐션), 최종 판정은 ~2027년으로 예상된다. SFE는 조건부 예측을 제공한다: SFE 공식이 맞다면 데이터 기반 HVP가 정확하다.
