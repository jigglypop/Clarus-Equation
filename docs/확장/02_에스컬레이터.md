# 에스컬레이터 -- 3+1의 구조

## 비유

CE의 전체 구조는 **3층짜리 에스컬레이터와 작동 버튼 1개**로 압축된다.

**건물은 3층이다.** Hodge 자기쌍대성에 의해 3층만 구조적으로 안정하다.

각 층에는 고유한 성격이 있다:

- **3층** = SU(3), 강한 핵력. 3가지 색으로 작동한다.
- **2층** = SU(2), 약한 핵력. 2가지 상태(위/아래)로 작동한다.
- **1층** = U(1), 전자기력. 1가지 전하(+/-)로 작동한다.

**에스컬레이터** = 중력. 층과 층 사이를 **접는** 장치. 층이 아니다. 층 사이를 이동시키는 메커니즘이다. 중력이 다른 3개 힘과 본질적으로 다른 이유가 여기에 있다. 같은 방식으로 양자화할 수 없는 것이 당연하다.

**작동 버튼** = alpha_s = 0.11789 (1/(2pi) 자기일관 풀이에서 도출). 이 버튼이 에스컬레이터의 속도를 결정하고, 각 층의 모든 물리량을 결정한다.

## N = d + 1 = 4

3개의 게이지 힘(층) + 1개의 접힘 메커니즘(에스컬레이터) = 4개 힘.

왜 자연의 힘이 정확히 4개인가? 3층짜리 건물에는 에스컬레이터가 1개 필요하기 때문이다.

## 3+1의 보편성

어디서 잘라도 3+1이 나온다:

| 구조 | 3 | +1 |
|---|---|---|
| 게이지 힘 | SU(3), SU(2), U(1) | 중력 |
| 공간 | 3차원 | 시간 1차원 |
| 세대 | 3세대 | 1개 커플링 |
| 색 전하 | 3색 | 1개 색중성 (바리온) |
| 쿼크 | 3종 (u,d,s 기본) | 1개 렙톤 대응 |

## 대응표

| 에스컬레이터 | CE |
|---|---|
| 건물이 3층인 이유 | d = 3 (Hodge 자기쌍대성) |
| 3층 / 2층 / 1층 | SU(3) / SU(2) / U(1) |
| 에스컬레이터 기계 | 중력 (시공간의 접힘) |
| 작동 버튼 | alpha_s (강한 핵력의 세기) |
| 에스컬레이터의 속도 | 광명장 (Phi) |
| 1층에 도착한 사람 | 보통 물질 (5%) |
| 에스컬레이터 위의 사람 | 암흑물질 (26%) |
| 에스컬레이터의 전기 | 암흑에너지 (69%) |

설계도를 읽으려면 수학만 있으면 된다: **건물이 몇 층인가**(3, Hodge에서 강제), **버튼의 세기가 얼마인가**(1/(2pi)에서 도출). 외부에서 넣는 숫자는 없다.
