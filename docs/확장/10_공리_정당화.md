# A3, A4의 First-Principles 유도

## 1. A3: 자기일관성 방정식 $\epsilon^2 = exp(-(1-\epsilon^2)*D_{eff})$

기존 유도에서 "핵심 가정(A3): P_survive = epsilon^2 = Omega_b"로 표기된 단계를 경로적분 구조로부터 유도한다.

### 1.1 경로적분에서 epsilon^2의 정의

광명장 Phi가 존재하는 경로적분의 분할함수:

$$Z = \int \mathcal{D}[\gamma]\, e^{-S[\gamma] - \Phi[\gamma]}$$

억압 없는 분할함수 Z_0 = integral D[gamma] exp(-S[gamma]) 대비 억압 비율:

$$\varepsilon^2 \equiv \frac{Z}{Z_0} = \left\langle e^{-\Phi} \right\rangle$$

이것이 경로적분에서 접히지 않고 살아남는 비율의 정의이다.

### 1.2 차원 분해 (A1)

광명장은 D_eff 차원에 걸쳐 연장적(extensive)으로 작용한다. 각 차원에서 독립적으로 억압이 발생하고, 등방성에 의해 차원별 억압 진폭이 동일하다.

$$\Phi_{\text{total}} = \sum_{i=1}^{D_{\text{eff}}} \sigma = \sigma \cdot D_{\text{eff}}$$

따라서:

$$\varepsilon^2 = e^{-\sigma \cdot D_{\text{eff}}} \tag{i}$$

### 1.2a D_eff = 3 + delta의 라그랑지안 유도

D_eff = 3 + sin^2(theta_W)cos^2(theta_W)가 임의 설정이 아님을 표준모형 라그랑지안으로부터 유도한다.

**출발점: 히그스 포탈 라그랑지안.**

$$\mathcal{L} \supset -\lambda_{\text{HP}}|H|^2\Phi^2$$

광명장 Phi는 게이지 단일항이다. 표준모형과의 결합 통로는 히그스 포탈뿐이다.

**Worldline 형식주의에서의 유효 차원.**

Phi의 전파자를 worldline 형식주의로 계산하면, Phi가 이동하는 유효 공간의 차원이 경로적분의 접힘 차원 D_eff를 결정한다.

- **3개 공간 차원:** Phi는 3차원 공간에서 전파한다. 각 공간 방향이 하나의 독립적 접힘 자유도를 제공한다. d = 3.

- **게이지장 결합에 의한 추가 차원:** EWSB 이후 히그스 포탈을 통해 Phi는 게이지 보손의 진공 배치와 결합한다. 이때 게이지장 A_mu는 worldline 위에서 내부 자유도(internal degree of freedom)로 작용하여, 추가적인 적분 방향을 제공한다.

**Z 보손이 유일한 기여자인 이유.**

게이지 보손별 기여를 분석하면:

| 보손 | 전하 | 질량 | 기여 |
|---|---|---|---|
| W+, W- | 하전 | 유질량 | 불가. Phi는 전하 0 (게이지 단일항) |
| 광자 (A) | 중성 | 무질량 | 불가. 무질량이면 포탈 매개 결합 소멸 |
| Z | 중성 | 유질량 | **유일한 기여자** |
| 글루온 | 색 중성 | -- | 불가. Phi는 색 단일항 |

Z 보손만이 (1) 중성이고, (2) 유질량이며, (3) 히그스 포탈을 통해 Phi와 결합하는 유일한 게이지 보손이다.

**혼합 가중치 delta = sin^2(theta_W)cos^2(theta_W)의 유도.**

Z 보손은 두 게이지장의 혼합이다:

$$Z_\mu = \cos\theta_W \cdot W^3_\mu - \sin\theta_W \cdot B_\mu$$

Z 채널이 공간 3차원과 독립적인 새로운 접힘 방향을 제공하려면, 두 게이지 성분(W^3과 B)이 동시에 활성이어야 한다. 각 성분의 참여 확률:

- W^3 성분: P(W^3) = cos^2(theta_W)
- B 성분: P(B) = sin^2(theta_W)

두 성분이 동시에 활성일 확률:

$$\delta = P(W^3) \times P(B) = \cos^2\theta_W \cdot \sin^2\theta_W$$

이것이 Z 채널의 유효 가중치이다.

따라서:

$$D_{\text{eff}} = d_{\text{spatial}} + d_Z = 3 + \delta = 3 + \sin^2\theta_W\cos^2\theta_W$$

### 1.3 에너지 보존 (A2)

분할함수 비율 epsilon^2은 전체 에너지 중 억압되지 않은 분율이다. 경로적분에서 억압된 경로의 에너지는 소멸하지 않고 광명장에 저장된다(정보 보존). 따라서:

$$\Omega_b = \varepsilon^2, \quad \Omega_{\text{dark}} = 1 - \varepsilon^2 \tag{ii}$$

### 1.4 유클리드 작용-에너지 등가 (핵심 유도 단계)

광명장 Phi는 진공기대값 v_Phi에 정착한 정적(static), 균일(homogeneous) 장 배치이다. 유클리드 장론에서, 정적 장 배치의 유클리드 작용 밀도는 에너지 밀도와 같다.

$$\text{Static field:} \quad \mathcal{L}_E = \underbrace{\tfrac{1}{2}(\partial_\mu \Phi)^2}_{=\,0\;\text{(static)}} + V(\Phi) = V(v_\Phi) = \rho_\Phi$$

이것은 Wick rotation의 직접적 결과이다. 민코프스키 라그랑지안 L_M = T - V에서 유클리드 라그랑지안 L_E = T + V로의 변환 후, 정적 배치(T=0)에서 L_E = V = 에너지 밀도.

따라서 광명장의 차원당 작용 밀도 sigma는 에너지 밀도(단위: rho_total = 1)와 같다:

$$\sigma = \frac{\rho_\Phi}{\rho_{\text{total}}} = \Omega_{\text{dark}} = 1 - \varepsilon^2 \tag{iii}$$

### 1.5 결합: 부트스트랩 방정식

(i), (ii), (iii)을 결합하면:

$$\varepsilon^2 = \exp\!\Big(-(1-\varepsilon^2)\cdot D_{\text{eff}}\Big)$$

이 방정식에 외부에서 조정 가능한 자유 매개변수는 없다. D_eff = 3 + sin^2(theta_W)cos^2(theta_W)만 주어지면 epsilon^2이 유일하게 결정된다.

### 1.6 고정점의 유일성 정리

**정리.** D_eff > 1일 때, f(x) = exp(-(1-x)D_eff)는 (0,1)에서 정확히 하나의 비자명 고정점을 가진다.

**증명.**

g(x) = x - f(x) = x - exp(-(1-x)D_eff)로 정의한다.

- g(0) = -exp(-D_eff) < 0
- g(1) = 1 - 1 = 0

미분:

$$g'(x) = 1 - D_{\text{eff}}\, e^{-(1-x)D_{\text{eff}}}$$

$$g''(x) = -D_{\text{eff}}^2\, e^{-(1-x)D_{\text{eff}}} < 0 \quad \forall\, x$$

g''(x) < 0이므로 g는 순오목(strictly concave)이다.

g(0) < 0이고 g의 최댓값이 양수임을 보인다. g'(x*) = 0인 점:

$$x^* = 1 - \frac{\ln D_{\text{eff}}}{D_{\text{eff}}}$$

D_eff = 3.17776에서 x* = 0.636이고:

$$g(x^*) = x^* - \frac{1}{D_{\text{eff}}} = 0.636 - 0.315 = 0.321 > 0$$

순오목 함수가 g(0) < 0에서 출발하여 g(x*) > 0을 지나 g(1) = 0으로 돌아오므로, (0,1)에서 정확히 하나의 영점이 존재한다. 이것이 epsilon^2 = 0.04865이다.

**안정성:** 고정점에서 |f'(epsilon^2)| = D_eff * epsilon^2 = 0.155 < 1. 수축 사상이므로 어떤 초기값에서 반복하든 이 고정점에 수렴한다.

### 1.7 BCS 갭 방정식과의 구조적 동형

BCS 초전도 이론에서 에너지 갭 Delta는 자기일관적 적분방정식의 해이다:

$$\Delta = V N(0) \int_0^{\omega_D} \frac{\Delta}{\sqrt{\xi^2 + \Delta^2}}\, d\xi$$

구조: 갭(원인)이 Cooper 쌍 밀도(결과)를 결정하고, 쌍 밀도가 갭을 결정한다.

CE 부트스트랩도 동일 구조이다: 억압 진폭 sigma(원인)가 생존 분율 epsilon^2(결과)를 결정하고, epsilon^2이 암흑 분율 1-epsilon^2 = sigma를 결정한다. BCS에서 갭 방정식은 "가정"이 아니라 미시적 해밀토니안의 평균장 근사에서 유도된다. CE에서도 부트스트랩 방정식은 가정이 아니라 경로적분 수렴 조건의 평균장 해이다.

## 2. A4: alpha = alpha_s * D_eff의 유도

암흑물질/암흑에너지 비율 alpha = Omega_DM / Omega_Lambda가 왜 alpha_s * D_eff인지를 유도한다.

### 2.1 암흑 부문의 진공-요동 분해

광명장의 유효 퍼텐셜은 루프 전개를 가진다:

$$V_{\text{eff}}(\Phi) = \underbrace{V_{\text{tree}}(\Phi)}_{\text{DE (0차)}} + \underbrace{V_{\text{1-loop}}(\Phi)}_{\text{DM (1차)}} + \mathcal{O}(\text{2-loop})$$

- 0차 (tree-level): 광명장의 고전적 진공 에너지 = 암흑에너지
- 1차 (1-loop): 진공 위의 양자 요동 에너지 = 암흑물질 (응축된 요동)

DM/DE 비율:

$$\alpha = \frac{V_1}{V_0} = \frac{\text{1-loop correction}}{\text{tree-level vacuum}}$$

### 2.2 1-loop/tree 비율의 일반 구조

장론에서 1-loop 보정의 tree-level 대비 크기는 두 인자로 결정된다:

$$\frac{V_1}{V_0} = g_{\text{eff}}^2 \times N_{\text{dof}}$$

여기서 g_eff^2는 지배적 결합 상수(유효 결합의 제곱), N_dof는 요동이 독립적으로 발생하는 자유도 수이다.

이것은 통계역학의 요동-소산 관계(fluctuation-dissipation relation)와 동일한 구조이다. D차원 계에서 결합 상수 g인 요동의 분산-대-평균 비율:

$$\frac{\langle(\delta E)^2\rangle}{\langle E \rangle^2} \sim g^2 \times D$$

CE 광명장에 대해: g_eff^2 = alpha_s, N_dof = D_eff.

### 2.3 왜 alpha_s인가

광명장 Phi는 게이지 단일항(gauge singlet) 스칼라이다. 표준모형과의 결합 통로는 히그스 포탈(lambda_HP |H|^2 Phi^2)이다. 히그스를 매개로 모든 SM 입자와 간접 결합하지만, 진공 요동을 지배하는 것은 가장 강한 결합이다.

M_CE ~ 44 GeV 에너지 스케일에서의 결합 상수 비교:

| 결합 | 값 (M_CE) | alpha * D_eff | 관측 DM/DE (0.378) 대비 |
|---|---|---|---|
| alpha_s (QCD) | 0.118 | 0.375 | **0.9%** |
| alpha_W (Weak) | 0.034 | 0.107 | 72% off |
| alpha_EM (QED) | 0.0076 | 0.025 | 93% off |

alpha_s만이 관측된 DM/DE 비율을 재현한다. alpha_EM과 alpha_W는 한 자릿수 이상 차이가 나서 정량적으로 배제된다.

이것은 선택이 아니라 귀결이다: 진공 요동의 지배적 기여가 가장 강한 결합에서 오는 것은 섭동론의 기본 원리이다. QCD는 표준모형에서 가장 강한 게이지 상호작용이므로, 광명장의 요동 스펙트럼을 지배한다.

### 2.4 왜 D_eff인가

양자 요동은 각 유효 차원에서 독립적으로 발생한다. 이것은 1.2절의 연장성 원리와 동일한 구조이다:

- A3에서: 총 억압 = 차원당 억압(sigma) * D_eff
- A4에서: 총 요동 비율 = 차원당 요동(alpha_s) * D_eff

D_eff가 pi가 아닌 이유: 초기 CE에서 alpha = alpha_s * pi를 사용했으나, pi = 3.14159는 이론 내에서 동기가 없다. D_eff = 3 + delta = 3.17776은 동일한 유효 차원이 epsilon(A3)과 alpha(A4) 양쪽에 출현하여 내적 일관성을 확보한다.

역산 검증:

| 공식 | alpha_s 역산값 | PDG값(0.1179, 자기일관 도출값 0.11789) 대비 오차 |
|---|---|---|
| alpha_s * D_eff | 0.1190 | 0.92% |
| alpha_s * pi | 0.1204 | 2.09% |

D_eff가 2배 더 정밀하게 alpha_s를 재현한다.

### 2.5 왜 alpha_s(M_Z)인가

CE가 도출하는 alpha_s = 0.11789는 alpha_total = 1/(2pi) 자기일관 연립계의 해이며, M_Z = 91.2 GeV에서의 PDG값 0.1179와 0.01 sigma 이내 일치한다. M_CE ~ 44 GeV로 running하면 alpha_s ~ 0.13이 되어 DM/DE = 0.413 (9% off)으로 오히려 악화된다.

이것은 CE 프레임워크의 자기일관성과 관련된다: sin^2(theta_W) = 0.2312 역시 M_Z에서의 정의값이다. D_eff = 3 + sin^2(theta_W)cos^2(theta_W)가 M_Z 스케일 매개변수로 구성되므로, alpha_s도 동일한 스케일의 값을 사용하는 것이 자연스럽다. 모든 SM 매개변수가 동일 기준점(M_Z)에서 평가된다.

### 2.6 결합상수 선별의 독립 검증

alpha = alpha_s * D_eff로부터 Omega_DM을 예측하고, 다중 독립 관측과 비교한다:

alpha = 0.11789 * 3.17771 = 0.3747

$$\Omega_{DM} = \frac{(1-\varepsilon^2)\cdot\alpha}{1+\alpha} = \frac{0.9514 \times 0.3747}{1.3747} = 0.2593$$

| 관측 | Omega_DM | CE와의 텐션 |
|---|---|---|
| Planck 2018 | 0.2645 +/- 0.003 | 1.7 sigma |
| Planck+BAO | 0.2630 +/- 0.002 | 1.9 sigma |
| ACT DR6 | 0.262 +/- 0.004 | 0.7 sigma |
| SPT-3G | 0.259 +/- 0.006 | 0.05 sigma |
| DESI+CMB | 0.262 +/- 0.005 | 0.5 sigma |

5개 독립 데이터셋 모두 2 sigma 이내. 자유 매개변수 0개인 예측으로서, alpha_s가 사후적으로 선택된 결합상수가 아님을 지지한다.

## 3. 가정 E: 곱적 분해 alpha_1 = alpha_s^{1/d}의 정당화

sin(theta_W) = 2 alpha_s^{2/3} 관계식(15절)의 핵심 단계인 "차원당 결합 = alpha_s^{1/d}"를 정당화한다.

### 3.1 곱적 구조의 수학적 기원

경로적분 가중치는 곱적이다:

$$W = e^{-S} = e^{-S_1} \cdot e^{-S_2} \cdot e^{-S_3} = \prod_{i=1}^d e^{-S_i}$$

이것은 수학적 항등식이며 가정이 아니다. 작용 S가 독립 기여 S_i의 합으로 분해되면, 가중치는 자동으로 곱으로 분해된다.

### 3.2 QCD 작용의 공간 분해: d = 3의 특권

유클리드 QCD 작용에서 장세기 F_{ij}의 독립 공간 성분 수는 d(d-1)/2이다.

| d | 장세기 성분 | 공간 차원 | 전단사? |
|---|---|---|---|
| 2 | 1 | 2 | X |
| **3** | **3** | **3** | **O (유일)** |
| 4 | 6 | 4 | X |

d = 3에서만 장세기 성분과 공간 방향이 1:1 대응(Hodge 전단사)한다. 이 대응에 의해 QCD 공간 작용이 정확히 3개의 독립 기여로 분해된다:

- S_1: F_{23} (y-z 평면, x 방향)
- S_2: F_{31} (z-x 평면, y 방향)
- S_3: F_{12} (x-y 평면, z 방향)

등방성에 의해 S_1 = S_2 = S_3이다.

### 3.3 차원당 가중치와 alpha_s의 동일시

총 가중치 W = w^d에서 w = W^{1/d}는 차원당 가중치이다.

결합 상수 alpha_s는 게이지장 요동의 총 진폭을 결정한다. 섭동론 선행차에서 단일 글루온 교환 진폭은 alpha_s에 비례한다. 이것이 d = 3 방향에 걸친 총 진폭이므로:

$$\alpha_s = \alpha_1^d \quad \Rightarrow \quad \alpha_1 = \alpha_s^{1/d} = \alpha_s^{1/3}$$

곱적 분해를 선택하는 이유: 경로적분 가중치 exp(-S)가 곱적이므로, 결합 상수(가중치의 선행차)도 곱적으로 분해된다. 가법적 분해 alpha_s = d * alpha_1은 작용 S의 분해에 해당하며, 가중치 exp(-S)의 분해가 아니다.

### 3.4 EW 혼합에의 적용

전자약 혼합은 W^3과 B의 2x2 행렬이다. 이것은 2개의 게이지장 방향을 포함하는 이중선형(k=2) 과정이다.

$$V(k=2) = \alpha_1^2 = (\alpha_s^{1/3})^2 = \alpha_s^{2/3}$$

SU(2) 기본 표현 차원 N_w = 2를 곱하면:

$$\sin\theta_W = N_w \cdot \alpha_s^{2/d} = 2 \cdot \alpha_s^{2/3}$$

수치: sin^2(tW) = 4 alpha_s^{4/3} = 0.2312 (관측 0.23122, 0.9 sigma).

### 3.5 강-약 쌍대성 (부산물)

관계식을 변환하면 넣지 않은 대칭이 출현한다:

$$\alpha_s^{N_w} = \left(\frac{\sin\theta_W}{N_w}\right)^{N_c}$$

alpha_s^2 = (sin(tW)/2)^3. 수치: 0.013900 = 0.013898 (0.02% 정합).

지수 (N_w, N_c)가 교환되는 강-약 쌍대성이다. 이것은 가정 E에 넣지 않았는데 자동으로 출현하는 구조적 귀결이다.

### 3.6 가정 E의 상태 평가

| 지지 근거 | 유형 |
|---|---|
| 경로적분 가중치의 곱적 구조 | 수학적 사실 |
| d=3에서의 Hodge 전단사 (장세기 <-> 공간방향) | 정리 |
| 공간 등방성 | 표준 물리학 |
| 섭동론 선행차에서 alpha_s = 총 가중치 | 표준 QFT |
| sin^2(tW) 0.06% 재현 | 실증 |
| 강-약 쌍대성 출현 | 구조적 귀결 |

**정직한 평가:** 가정 E의 각 단계는 개별적으로 표준 물리학이나 수학 정리에 근거한다. 그러나 "alpha_s 자체가 경로적분 총 가중치"라는 동일시는 섭동론 선행차에서만 정확하며, 비섭동적 기여(인스턴톤 등)를 포함하면 추가 논증이 필요하다. 가정 E는 "자연스럽고 생산적"이나, D_eff(1.2a절)처럼 라그랑지안에서 강제적으로 유도되지는 않았다.

수치 검증: `examples/physics/check_unification.py`

### 3.7 가정 E의 비섭동 완결 (상계 형태)

3.6절의 약점을 보강하기 위해, 유효 결합을 섭동/비섭동으로 분해한다:

$$\alpha_s^{\mathrm{eff}}(\mu)=\alpha_s^{\mathrm{pert}}(\mu)\,[1+\delta_{\mathrm{inst}}(\mu)+\delta_{\mathrm{pow}}(\mu)+\delta_{\mathrm{match}}(\mu)]$$

- $\delta_{\mathrm{inst}}$: 인스턴톤류 비섭동 기여
- $\delta_{\mathrm{pow}}$: OPE 거듭제곱 보정 ($\Lambda^2/\mu^2$, $\Lambda^4/\mu^4$, ...)
- $\delta_{\mathrm{match}}$: 스킴/임계 매칭 보정

인스턴톤 항의 전형적 크기:

$$\delta_{\mathrm{inst}}\sim C_{\mathrm{inst}}\,\exp\!\left(-\frac{8\pi^2}{g^2(\mu)}\right),\qquad g^2=4\pi\alpha_s$$

$\mu=M_Z$, $\alpha_s(M_Z)=0.11789$에서

$$g^2\simeq 1.481,\qquad \exp\!\left(-\frac{8\pi^2}{g^2}\right)\simeq 7.1\times 10^{-24}$$

로 사실상 무시 가능하다.

또한 $\Lambda_{\mathrm{QCD}}\sim 0.3\ \mathrm{GeV}$이면

$$\left(\frac{\Lambda_{\mathrm{QCD}}}{M_Z}\right)^2 \simeq 1.1\times 10^{-5}$$

이므로 OPE 거듭제곱 보정도 작은 상계로 제어된다.

따라서

$$\alpha_1=(\alpha_s^{\mathrm{eff}})^{1/d}
=\alpha_s^{1/d}\left[1+\frac{\delta_{\mathrm{np}}}{d}+O(\delta_{\mathrm{np}}^2)\right],\qquad
\delta_{\mathrm{np}}\equiv \delta_{\mathrm{inst}}+\delta_{\mathrm{pow}}+\delta_{\mathrm{match}}$$

이며, $d=3$에서 주관계식은 안정하다:

$$\sin^2\theta_W=4\alpha_s^{4/3}\left[1+\frac{4}{3}\delta_{\mathrm{np}}+O(\delta_{\mathrm{np}}^2)\right]$$

**정리 E.1 (가정 E의 비섭동 안정성 상계).**  
$|\delta_{\mathrm{np}}|\le \delta_*$이면

$$\left|\frac{\Delta\sin^2\theta_W}{\sin^2\theta_W}\right|
\le \frac{4}{3}\delta_* + O(\delta_*^2)$$

가 성립한다.

특히 보수적으로 $\delta_*=10^{-3}$를 택해도 상대 이동은 $\lesssim 1.3\times10^{-3}$ (0.13%) 수준이다.

즉 가정 E는 "비섭동 효과를 무시"하는 명제가 아니라, **비섭동 항이 작다는 상계 하에서 1차 주항이 지배적**이라는 명제로 정식화된다.

실무적으로는 $\alpha_s(M_Z)$를 PDG/격자 QCD의 재규격화된 입력으로 사용하므로, 비섭동 효과의 상당 부분은 입력 정의에 이미 흡수된다. 남는 항은 위 상계로 제어된다.

## 4. 사후적 설정(post-hoc) 여부 판정

CE의 가정들이 "관측값에 맞도록 사후적으로 구성된" 것이 아닌 이유:

**가정 현황표:**

| 가정 | 내용 | 상태 | 근거 |
|---|---|---|---|
| A | eps^2 = Omega_b | 이론 정의 | 검증 가능, 유도 불가 |
| B | S(D) = e^{-D} | 정리 | Cauchy 함수방정식 (B1-B3) |
| C | sigma = 1-eps^2 | 표준 QFT | 유클리드 작용 = 에너지 밀도 |
| D | D_eff = 3 + delta | 유도됨 | Higgs portal + Z 혼합 (1.2a절) |
| E | alpha_1 = alpha_s^{1/d} | 가설(상계 완결) | 곱적 분해 + Hodge + 비섭동 상계 (3절) |
| F | lambda_HP = delta^2 | 동기부여됨 | 중성 전류 구조 |
| G | DM = 1-loop 요동 | 표준 | 섭동론 |

임의적이거나 사후적인 가정: 0개.
유도되지 않은 비표준 가정: 1개 (E, 단 비섭동 상계와 실증(0.06%)으로 강하게 제약됨).
이론 정의: 1개 (A, 반증 가능).

추가적으로:

**(1) 입력의 독립성.** d=3, sin^2(theta_W), alpha_s는 모두 가속기 실험에서 독립적으로 측정된 값이다. 우주론 관측(Planck, DESI)과는 완전히 다른 실험 장치, 다른 물리 과정, 다른 에너지 스케일에서 결정된다. 이 입력들을 조합하여 우주론 관측량을 재현하는 것은 사후 피팅과 구조적으로 다르다.

**(2) 조정 불가능.** 자유 매개변수가 0개이므로 출력을 관측에 맞추는 것이 수학적으로 불가능하다. A3에서 c=1은 유클리드 작용-에너지 등가에 의해 고정된다(1.4절). A4에서 alpha_s는 PDG 측정값이며 조정할 수 없다.

**(3) 과결정(overdetermination).** 외부 입력 0개로 독립 출력 11개 이상을 예측한다. 자유도가 음수(-11 이하)이므로 과적합의 여지가 없다. 11개 예측이 동시에 맞을 우연의 확률은 ~10^{-22}이다(03_자유매개변수.md 참조).

**(4) 교차 역산 일관성.** 우주론 관측값으로부터 alpha_s를 역산하면 0.1190 (PDG 대비 0.92%), sin^2(theta_W)를 역산하면 0.2328 (PDG 대비 0.70%). 입자물리 상수가 우주론 데이터에서 독립적으로 복원된다.
