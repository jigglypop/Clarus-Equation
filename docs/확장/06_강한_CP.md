# 미해결 전선 B: 강한 CP 문제

## 문제

QCD 라그랑지안에는 CP 위반 항 theta * (g^2/32pi^2) * G_{\mu\nu} \tilde{G}^{\mu\nu}이 허용된다. 이 항이 존재하면 중성자의 전기 쌍극자 모멘트가 생긴다. 그러나 실험적으로 중성자 EDM은 극도로 작다: |d_n| < 1.8 x 10^{-26} e cm. 이것은 theta < 10^{-10}을 의미한다. 왜 theta가 0에 가까운지 설명하는 메커니즘이 없다.

기존 해법: Peccei-Quinn 대칭 + 액시온. 아직 발견되지 않음.
