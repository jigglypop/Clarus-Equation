# 미해결 전선 E: 페르미온 질량 계층

## 문제

페르미온 질량은 극단적인 계층 구조를 보인다:

| 세대 | 전하 +2/3 쿼크 | 전하 -1/3 쿼크 | 하전 렙톤 |
|---|---|---|---|
| 1 | u: 2.2 MeV | d: 4.7 MeV | e: 0.511 MeV |
| 2 | c: 1275 MeV | s: 95 MeV | mu: 105.7 MeV |
| 3 | t: 173,000 MeV | b: 4180 MeV | tau: 1777 MeV |

최대/최소 비율: m_t / m_e ~ 340,000. 왜 이런 계층이 존재하는가? 표준모형은 각 질량을 유카와 커플링으로 넣지만, 커플링의 값 자체는 설명하지 못한다.

## CE와의 접점

관측된 패턴: 세대 간 질량비가 alpha_s^{n/3}에 비례하는 경향이 있다. 예를 들어:

- m_c / m_t ~ alpha_s^{2/3} ~ 0.0074 (실제: 0.0074)
- m_u / m_c ~ alpha_s^{2/3} ~ 0.0017 (실제: 0.0017)

이것은 관측된 수치적 패턴이며, 엄밀한 유도는 아직 없다.

Koide 공식과의 연결: Koide가 발견한 렙톤 질량 관계 (m_e + m_mu + m_tau) / (sqrt(m_e) + sqrt(m_mu) + sqrt(m_tau))^2 = 2/3에서, 2/3 = 2/d가 d=3과 연결될 가능성이 있다.
