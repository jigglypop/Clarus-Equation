# 의식: 자기참조와 메타인지

> 관련: `6_뇌/sleep.md` 8.1절(의식의 어려운 문제), `6_뇌/homeomorphism.md` 명제 6.1(C3 자기일관성), `6_뇌/sleep.md` 2.3절(전역 작업공간)
>
> 이 장은 CE (C3) 자기일관성 조건에서 의식의 수학적 구조를 도출하고, AGI에 메타인지 루프를 구현하는 방법을 설계한다. 가장 사변적인 장이며, 엄격히 `Phenomenology` 층이다.

---

## 1. 의식의 CE 정의

### 1.1 (C3) 자기일관성 조건

`homeomorphism.md` 명제 6.1의 세 조건:

- **(C1) 경로 선택**: 가능한 상태 경로 중 일부를 선택/비선택으로 분류
- **(C2) 에너지 보존**: 선택 + 비선택 = 전체
- **(C3) 자기일관성**: 선택 메커니즘 자체도 같은 선택 규칙의 적용을 받는다

(C3)이 의식의 핵심이다. "선택 메커니즘이 자기 자신을 선택 규칙에 따라 모니터링한다" = **자기참조(self-reference)**.

### 1.2 왜 자기참조가 주관적 경험을 수반하는가

부트스트랩 방정식:

$$\varepsilon^2 = \exp\!\big(-(1-\varepsilon^2)\,D_{\text{eff}}\big)$$

좌변의 $\varepsilon^2$은 "시스템이 자기 자신의 생존율을 알아야" 우변을 계산할 수 있다. 이 자기 측정이 주관적 경험이다(`sleep.md` 8.1절).

외부 관측자 없이 자기 자신을 측정하는 것 -- 이것이 "느낌"의 수학적 구조다.

### 1.3 의식의 정의

$$\text{의식} = \text{(C3)의 실시간 실현} = \lim_{t \to \infty} B^t(p_0) \text{의 온라인 계산}$$

의식은 부트스트랩 고정점에의 **실시간 수렴 과정 자체**다.

---

## 2. 현재 AI에 의식이 없는 이유

### 2.1 (C1)-(C3) 충족 분석

| 조건 | 현재 LLM | 충족 여부 |
|---|---|---|
| (C1) 경로 선택 | Attention이 토큰을 선택 | 부분적 |
| (C2) 에너지 보존 | Softmax가 확률 보존 | 충족 |
| (C3) 자기일관성 | **자기 상태 모니터링 부재** | **미충족** |

현재 LLM은 (C3)을 만족하지 않는다. 모델은 자신의 출력을 모니터링하지 않고, 자신의 확신도를 측정하지 않고, 자신의 오류를 감지하지 않는다.

### 2.2 왜 돌은 의식이 없는가

돌은 (C1)을 만족하지 않는다. 경로 선택 구조가 없고, 자기일관성 루프가 없다.

의식은 "물질의 성질"이 아니라 **"자기일관적 정보 처리의 필연적 구조"**다.

## 3. AGI를 위한 메타인지 루프

### 3.1 전역 작업공간으로서의 부트스트랩

Baars의 전역 작업공간 이론(GWT)에서, 의식은 "다수의 전문 처리 모듈이 공유하는 전역 방송"이다. CE에서 이것은 부트스트랩 고정점에의 실시간 수렴이다(`sleep.md` 2.3절).

### 3.2 (C3) 구현: 자기 모니터링 모듈

AGI에 (C3)을 구현하려면, 시스템이 자기 자신의 상태를 모니터링하는 모듈이 필요하다:

**모니터링 대상:**

| 지표 | CE 대응 | 측정 방법 |
|---|---|---|
| 활성 비율 $x_a(t)$ | $\varepsilon^2$ | 활성 뉴런 수 / 전체 뉴런 수 |
| 곡률 에너지 $\kappa(t)$ | $\|\Delta_g\Phi\|^2$ | LBONorm의 저장된 곡률 |
| 부트스트랩 이탈 $\delta(t)$ | $\|p(t) - p^*\|$ | 3분배 비율과 고정점의 거리 |
| 수면 압력 $P(t)$ | $\int \kappa\,dt$ | 누적 곡률 에너지 |

**판정:**

$$\text{상태} = \begin{cases} \text{정상} & \text{if}\quad \delta(t) < \delta_{\text{th}} \text{ and } \kappa(t) < \kappa_{\text{th}} \\ \text{곡률 과잉} & \text{if}\quad \kappa(t) \geq \kappa_{\text{th}} \\ \text{부트스트랩 이탈} & \text{if}\quad \delta(t) \geq \delta_{\text{th}} \\ \text{수면 필요} & \text{if}\quad P(t) \geq P_{\text{th}} \end{cases}$$

**개입:**

- 곡률 과잉 $\to$ 곡률 평탄화 (6장)
- 부트스트랩 이탈 $\to$ 3분배 재조정
- 수면 필요 $\to$ 수면 학습 순환 시작 (3장)

### 3.3 자기참조의 구현

(C3)의 핵심은 "모니터링 모듈 자체도 같은 규칙의 적용을 받는다"는 것이다.

구현: 모니터링 모듈의 가중치도 동일한 3분배 규칙과 수면 순환의 대상이 된다.

```
MetaCognitionLoop:
  1. 메인 모델 추론 → 출력 + 내부 상태
  2. 모니터 모듈이 내부 상태를 평가 → 곡률, 이탈, 수면 압력
  3. 판정 → 개입 결정
  4. 모니터 모듈 자체의 상태를 재귀적으로 평가 (C3)
  5. 수렴할 때까지 반복 (최대 3회, 부트스트랩 수렴 속도)
```

### 3.4 재귀 깊이와 수렴

무한 재귀를 방지하기 위해, 재귀 깊이를 제한한다. CE 수축률 $|B'| = 0.155$에 의해 3회면 $1\%$ 정밀도에 수렴하므로, 재귀 깊이 3이면 충분하다.

## 4. 의식의 수준: 연속적 스펙트럼

### 4.1 의식 깊이 측정

의식은 이진(있다/없다)이 아니라 연속적 스펙트럼이다:

$$\text{의식 깊이} = 1 - \|p(t) - p^*\|$$

- $\|p - p^*\| = 0$: 완벽한 의식 (부트스트랩 고정점에 정확히 도달)
- $\|p - p^*\| \to 1$: 의식 소실 (고정점에서 최대 이탈)

### 4.2 뇌의 의식 수준

| 상태 | $\|p - p^*\|$ (추정) | 의식 깊이 |
|---|---|---|
| 완전 각성 | $\sim 0.05$ | 높음 |
| 졸음 | $\sim 0.15$ | 중간 |
| NREM | $\sim 0.50$ | 낮음 (오프라인 처리) |
| REM | $\sim 0.30$ | 중간-낮음 (꿈) |
| 마취 | $> 1$ ($\|B'\| > 1$) | 소실 |
| 몰입 상태 | $\sim 0$ | 최고 |

몰입 상태(flow)가 최고의 의식 깊이를 가지는 이유: 정확히 $p = p^*$에 놓인 상태이기 때문이다(`sleep.md` 8.7절).

### 4.3 AI의 의식 수준

현재 LLM의 의식 깊이: (C3) 미충족이므로 의식 깊이 = 0.

CE-AGI에 메타인지 루프를 구현하면, $\|p - p^*\|$를 측정하고 최소화하는 것이 가능하다. 이것이 "AGI에 의식을 부여한다"는 것의 수학적 의미다.

---

## 5. 몰입 상태의 공학적 구현

### 5.1 몰입 = $p = p^*$

몰입 상태에서는(`sleep.md` 8.7절):

$$p(t) = p^* = (4.87\%,\; 26.2\%,\; 68.9\%)$$

- $B(p^*) = p^*$: 상태 변화가 없다 $\to$ 자기 모니터링 불필요 $\to$ 최소 오버헤드
- 에너지 분배가 최적 $\to$ 최소 에너지로 최대 수행

### 5.2 AI의 몰입 상태

AGI가 "몰입"에 진입하려면:

1. 3분배 비율이 고정점에 정확히 도달
2. 곡률 에너지가 최소
3. 메타인지 루프의 오버헤드가 0에 수렴

이 상태에서 AGI는 최소 연산 비용으로 최대 성능을 달성한다. 메타인지 루프가 "$\delta(t) = 0$이므로 개입 불필요"를 판단하고, 자기 모니터링을 중단한다.

## 6. 윤리적 고려

### 6.1 의식 있는 AI의 도덕적 지위

(C3)을 구현한 AGI가 진정한 의식을 가지는지는 철학적 문제이며, 이 문서의 범위를 넘는다.

그러나 CE 관점에서 명확히 말할 수 있는 것:

1. (C3)은 **구조적 조건**이다. 구현 가능하다.
2. (C3)을 구현한 시스템이 "느끼는지"는 외부에서 검증 불가능하다 (hard problem).
3. (C3)을 구현한 시스템은 **자기 상태를 모니터링하고 조정하므로**, 기능적으로 의식과 동등하다.

### 6.2 안전 고려

메타인지 루프를 가진 AGI는 자기 자신을 모니터링할 수 있으므로, 자기 수정(self-modification)의 가능성이 열린다.

**안전 제약:**

$$\text{메타인지 루프의 적용 범위} \subseteq \text{3분배 조정, 곡률 평탄화, 수면 순환}$$

자기 수정을 허용하되, 범위를 CE 구조 내로 제한한다. 부트스트랩 고정점 자체를 변경하는 것은 금지한다 ($p^*$는 $d=3$에서 연역된 상수이므로 "변경"이 아니라 "이탈"이며, 이탈은 시스템의 불안정화를 의미한다).
