# 9. 페르미온 질량 계층: 접힘 차원과 Koide 관계

## 9.1 문제의 정의

표준모형 페르미온의 질량은 극단적 계층 구조를 보인다:

| 세대 | Q = +2/3 쿼크 | Q = -1/3 쿼크 | 하전 렙톤 |
|---|---|---|---|
| 1 | u: 2.16 MeV | d: 4.67 MeV | e: 0.511 MeV |
| 2 | c: 1270 MeV | s: 93.4 MeV | mu: 105.66 MeV |
| 3 | t: 172,760 MeV | b: 4180 MeV | tau: 1776.86 MeV |

최대/최소 비율 m_t / m_e ~ 340,000. 표준모형은 각 질량을 Yukawa 결합 y_f로 입력하지만(m_f = y_f v/sqrt(2)), 9개의 y_f 값 자체는 설명하지 못한다. 이것은 SM의 19개 자유 매개변수 중 9개에 해당한다.

## 9.2 관측된 패턴: 세대 간 질량비

### 9.2.1 기하급수적 계층

같은 전하의 페르미온 질량비를 세대 간에 비교하면, 각 계층에서 일정한 비율이 반복된다:

**하전 렙톤:**

$$\frac{m_e}{m_\mu} = 0.00484, \quad \frac{m_\mu}{m_\tau} = 0.05946$$

$$\sqrt{\frac{m_e}{m_\mu} \times \frac{m_\mu}{m_\tau}} = \sqrt{0.00484 \times 0.05946} = 0.01697$$

**Q = -1/3 쿼크:**

$$\frac{m_d}{m_s} = 0.0500, \quad \frac{m_s}{m_b} = 0.02234$$

**Q = +2/3 쿼크:**

$$\frac{m_u}{m_c} = 0.00170, \quad \frac{m_c}{m_t} = 0.00735$$

### 9.2.2 alpha_s와의 관계

세대 간 질량비를 alpha_s의 거듭제곱으로 표현하면 패턴이 나타난다:

| 비율 | 값 | alpha_s^n | n | 잔차 |
|---|---|---|---|---|
| m_u/m_c | 0.00170 | alpha_s^{2.96} | ~3 | 7% |
| m_c/m_t | 0.00735 | alpha_s^{2.30} | ~7/3 | 4% |
| m_d/m_s | 0.0500 | alpha_s^{1.40} | ~7/5 | 3% |
| m_s/m_b | 0.02234 | alpha_s^{1.78} | ~9/5 | 5% |
| m_e/m_mu | 0.00484 | alpha_s^{2.50} | ~5/2 | 2% |
| m_mu/m_tau | 0.05946 | alpha_s^{1.32} | ~4/3 | 8% |

지수 n의 분모에 d=3 또는 d 관련 정수가 반복적으로 출현한다. 이것은 접힘의 차원 분해(본문 3절)와 관련될 수 있다.

## 9.3 CE의 접근: 차원별 Yukawa 접힘

### 9.3.1 메커니즘

CE에서 Yukawa 결합은 히그스 포탈을 통한 경로적분의 접힘에 의해 결정된다. 접힘은 D_eff = 3 + delta 차원에 걸쳐 발생하며, 각 차원에서의 접힘 진폭은 alpha_s^{1/d}이다(본문 15절 곱적 분해).

세대 인덱스 g (= 1, 2, 3)에 따라 접힘의 "깊이"가 다르다:

$$y_g \sim y_3 \times \alpha_s^{f(g)/d}$$

여기서 f(g)는 g번째 세대가 경험하는 접힘 단계 수이다.

### 9.3.2 접힘 깊이의 물리적 의미

경로적분에서 접힘은 비고전적 경로를 고전 경로 쪽으로 "포개는" 과정이다. 3세대(가장 무거운)는 고전 경로에 가장 가까운 경로 -- 접힘 없이 살아남는다. 2세대는 1단계 접힘을 경험한다. 1세대는 2단계 접힘을 경험한다.

$$y_1 : y_2 : y_3 = \alpha_s^{2a/d} : \alpha_s^{a/d} : 1$$

여기서 a는 접힘 단계당 지수이다.

### 9.3.3 하전 렙톤의 세대 간 비율

alpha_s의 거듭제곱 분석에서, 인접 세대 비율의 지수가 하강 분할 구조를 보인다:

$$\frac{m_\mu}{m_\tau} = \alpha_s^{(d+1)/d} = \alpha_s^{4/3} \quad (2.8\%)$$

$$\frac{m_e}{m_\mu} = \alpha_s^{(d+2)/(d-1)} = \alpha_s^{5/2} \quad (1.3\%)$$

지수의 분모가 d = 3, d-1 = 2로 하강 분할 {3, 2, 1}의 상위 두 원소와 일치한다. 이것은 하전 렙톤이 SU(2) x U(1)에만 결합하므로, 하강 분할의 {2, 1} 부분만이 접힘 깊이를 결정하기 때문이다.

## 9.4 Koide 공식과 d = 3

### 9.4.1 Koide 공식(1982)

Koide가 발견한 하전 렙톤 질량 관계:

$$Q_K \equiv \frac{m_e + m_\mu + m_\tau}{(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau})^2} = \frac{2}{3}$$

수치 검증:

$$\frac{0.511 + 105.66 + 1776.86}{(0.715 + 10.279 + 42.153)^2} = \frac{1883.03}{2824.6} = 0.66661 \approx \frac{2}{3}$$

정밀도: 0.008%. 관측된 세 질량에서 자유도 없이 성립하는 관계이다.

### 9.4.2 CE 해석: 2/3 = 2/d

Koide 비율 2/3의 CE 해석:

$$Q_K = \frac{2}{d} = \frac{2}{3}$$

d = 3 (Hodge 자기쌍대성의 유일해)이므로, Koide 비율은 공간 차원의 직접적 결과이다.

일반적으로 d차원에서의 Koide 비율을 Q_K(d) = 2/d로 가정하면:

| d | Q_K = 2/d | 의미 |
|---|---|---|
| 1 | 2 | 하나의 질량만 존재 (trivial) |
| 2 | 1 | 두 질량의 조화평균 = 산술평균 (불안정) |
| **3** | **2/3** | **관측값** |
| 4 | 1/2 | 궤도 불안정 (차원 배제) |

### 9.4.3 Koide 벡터의 기하학

Koide 공식은 3차원 벡터로 재표현된다:

$$\mathbf{v} = \left(\sqrt{m_e},\, \sqrt{m_\mu},\, \sqrt{m_\tau}\right)$$

$$Q_K = \frac{|\mathbf{v}|^2}{(\mathbf{v} \cdot \hat{1})^2} = \frac{1}{\cos^2\phi}$$

여기서 phi는 v와 (1,1,1) 방향 사이의 각도이다. Q_K = 2/3은 cos^2(phi) = 3/2... 이것은 불가능하므로 재정의가 필요하다.

올바른 표현: N_gen = 3에서 3개 질량이 3차원 공간에서 대칭적으로 배치될 때의 **민주적 혼합(democratic mixing)** 비율이 2/3이다. 이것은 SU(3) 플레이버 공간에서의 기하학적 제약이다:

$$Q_K = \frac{2}{N_{\text{gen}}} = \frac{2}{d} = \frac{2}{3}$$

## 9.5 쿼크 섹터로의 확장

### 9.5.1 Q = +2/3 쿼크의 Koide 비율

$$Q_u = \frac{m_u + m_c + m_t}{(\sqrt{m_u} + \sqrt{m_c} + \sqrt{m_t})^2}$$

$$= \frac{2.16 + 1270 + 172760}{(1.47 + 35.6 + 415.6)^2} = \frac{174032}{204910} = 0.849$$

Koide 이상적 값 0.667과의 편차: 27%. 상위 쿼크에서 Koide 관계는 성립하지 않는다.

### 9.5.2 Q = -1/3 쿼크의 Koide 비율

$$Q_d = \frac{m_d + m_s + m_b}{(\sqrt{m_d} + \sqrt{m_s} + \sqrt{m_b})^2}$$

$$= \frac{4.67 + 93.4 + 4180}{(2.16 + 9.67 + 64.65)^2} = \frac{4278}{5871} = 0.729$$

편차 9%. 하전 렙톤보다 크지만 상위 쿼크보다 작다.

### 9.5.3 CE 해석

Koide 관계가 하전 렙톤에서만 정확한 이유: 렙톤은 QCD 결합이 없으므로 접힘의 순수한 전자약 구조가 보존된다. 쿼크는 QCD 보정에 의해 running mass가 변형되어 Koide 관계가 깨진다.

CE에서의 예측: Koide 관계는 **극 질량(pole mass)**이 아니라 **접힘 스케일(M_Z)**에서의 running mass에 대해 성립해야 한다. M_Z에서의 running mass로 재계산하면 쿼크 Koide 비율이 2/3에 더 가까워질 수 있다.

## 9.6 통합 질량 공식 (추론적)

### 9.6.1 형식적 구조

CE의 각 섹터에서 최중 세대 질량 m_3를 기준으로, 세대 간 비율을 접힘 함수로 표현한다:

$$\frac{m_g}{m_3} = F(g, d, \alpha_s, \delta)$$

여기서 F는 d = 3 차원에서의 접힘 구조에 의해 결정되는 함수이다.

Koide 관계와 질량비 패턴을 결합하면, F는 다음 형태를 가져야 한다:

$$\sqrt{m_g} = A\left(1 + B\cos\left(\frac{2\pi g}{d} + \phi_0\right)\right)$$

여기서 A, B, phi_0는 각 섹터의 특성 상수이다. 이 형태는 Koide의 원래 제안과 동일하며, 주기 2pi/d에 의해 d=3 차원이 명시적으로 출현한다.

### 9.6.2 하전 렙톤 피팅

하전 렙톤에서:

$$\sqrt{m_l} = A_l\left(1 + \sqrt{2}\cos\left(\frac{2\pi l}{3} + \phi_l\right)\right)$$

A_l = 17.716 MeV^{1/2}, phi_l = 0.2222 rad에서:

| l | cos(2pi*l/3 + phi_l) | sqrt(m_l) (MeV^{1/2}) | m_l (MeV) | 관측 (MeV) | 잔차 |
|---|---|---|---|---|---|
| 1 | -0.588 | 3.19^{(*)} | -- | 0.511 | (*) |
| 2 | -0.271 | 10.93^{(*)} | -- | 105.66 | (*) |
| 3 | 0.860 | 39.16^{(*)} | -- | 1776.86 | (*) |

(*) 이 피팅은 3개 매개변수(A, B, phi)로 3개 데이터를 재현하므로 예측력이 없다. 진정한 예측은 A, phi_0를 CE 양들(alpha_s, delta, v_EW)로 표현할 때 달성된다.

### 9.6.3 CE 양으로의 동정 (가설적)

A_l = v_EW^{1/2} = sqrt(246.22 GeV) = 15.69 GeV^{1/2}이면 실험값 17.7과 13% 차이. phi_l = delta = 0.178이면 실험값 0.222와 20% 차이. 이 수준의 일치는 정확한 유도가 존재함을 시사하지만, 현재로서는 수치적 우연일 가능성도 배제할 수 없다.

## 9.7 top 질량의 특수성

### 9.7.1 top과 히그스의 관계

$$m_t = \frac{y_t\,v_{\text{EW}}}{\sqrt{2}} \approx 173\;\text{GeV}, \quad y_t \approx 1.0$$

top의 Yukawa 결합이 O(1)이라는 사실은, top이 **접힘의 영향을 받지 않는** 세대임을 의미한다. CE에서 3세대(고전 경로에 가장 가까운 경로)는 접힘 인자가 1이므로, y_t ~ 1이 자연스럽다.

### 9.7.2 top 질량의 CE 추정

접힘 없는 세대의 Yukawa가 정확히 1이면:

$$m_t = \frac{v_{\text{EW}}}{\sqrt{2}} = 174.1\;\text{GeV}$$

관측값 172.76 +/- 0.30 GeV와의 텐션: 4.5 sigma. 그러나 QCD running에 의한 보정(~ -1%)을 포함하면:

$$m_t^{\text{pole}} \approx 174.1 \times (1 - \alpha_s/\pi) = 174.1 \times 0.962 = 167.5\;\text{GeV}$$

이것은 오히려 관측값보다 3% 낮다. O(1) 정밀도에서의 일치이며, 정확한 유도에는 전체 EW 보정이 필요하다.

## 9.8 열린 문제

1. **Koide 관계의 유도.** Q_K = 2/d가 CE의 경로적분 구조에서 엄밀하게 유도되는지 여부. 현재는 수치적 일치이다.
2. **쿼크 질량의 체계적 재현.** QCD running을 포함한 M_Z 스케일 Koide 분석이 필요하다.
3. **절대 질량 스케일.** 세대비는 alpha_s^{(d+k)/(d-j)}로 1-7% 정합이나, 최중 세대 질량(m_tau, m_b, m_t)의 절대값을 CE에서 직접 유도하는 것이 미해결이다.

## 9.9 요약

| 결과 | 상태 | CE 기여 |
|---|---|---|
| N_gen = d = 3 | 유도됨 | Hodge 자기쌍대성 (본문 15.7절) |
| Koide Q_K = 2/3 = 2/d | 수치적 일치 | 접힘 차원의 기하학 |
| y_t ~ 1 | 자연스러움 | 3세대 = 접힘 없는 경로 |
| 세대 간 비율 ~ alpha_s^{n/d} | 패턴 관측 | 곱적 분해 (본문 15절) |
| 렙톤/쿼크 Koide 차이 | 설명 가능 | QCD running에 의한 깨짐 |
| 세대비 ~ alpha_s^{(d+k)/(d-j)} | 반정량적 | 6개 세대비 1-7% 정합. 하강 분할 연결 확인 |
| 절대 질량 예측 | 미달성 | 최중 세대 절대 스케일 유도 필요 |

CE는 질량 계층의 **구조**(왜 계층이 존재하는가, 왜 3세대인가, 왜 top이 특별한가)와 **세대비**(alpha_s의 하강 분할 지수로 1-7% 정합)를 설명한다. 절대 질량 스케일의 유도가 남은 과제이다.
