# 6. 강한 CP 문제: 경로적분 접힘에 의한 theta 억압

## 6.1 문제의 정의

QCD 라그랑지안에는 CP 위반 항이 허용된다:

$$\mathcal{L}_\theta = \frac{\theta\, g_s^2}{32\pi^2}\, G_{\mu\nu}^a \tilde{G}^{a\mu\nu}$$

이 항이 존재하면 중성자 전기 쌍극자 모멘트(nEDM)가 생긴다:

$$|d_n| \sim \frac{e\, m_q}{m_N^2}\,\theta \sim 10^{-16}\,\theta \;\text{e cm}$$

실험 상한(Abel et al. 2020):

$$|d_n| < 1.8 \times 10^{-26}\;\text{e cm} \quad \Rightarrow \quad |\theta| < 10^{-10}$$

왜 $\theta$가 O(1)이 아니라 $10^{-10}$ 이하인가? 표준모형 내에서 $\theta$를 제약하는 대칭이 없다. 이것이 강한 CP 문제이다.

### 기존 해법

**Peccei-Quinn 대칭(1977):** 전역 $U(1)_{PQ}$ 대칭을 도입하고, 자발적 파괴로 액시온(Weinberg 1978, Wilczek 1978)을 생성한다. 액시온이 theta를 동역학적으로 0으로 구동한다. 그러나 액시온은 40년간 미발견이다.

**Nelson-Barr 메커니즘:** $\theta = 0$을 tree level에서 강제하고, 루프 보정으로 작은 값을 생성한다. 특수한 장 구조를 요구한다.

## 6.2 SFE의 접근: 접힘에 의한 인스턴톤 억압

### 6.2.1 $\theta$ 항의 경로적분 해석

$\theta$ 항은 경로적분에서 위상학적 전하 Q에 가중치를 부여한다:

$$Z(\theta) = \sum_{Q=-\infty}^{\infty} e^{i\theta Q}\, Z_Q$$

여기서 Z_Q는 위상학적 전하 Q인 배치(인스턴톤)의 분할함수이다. Q != 0 배치는 위상학적으로 비자명(topologically non-trivial)한 경로이다.

### 6.2.2 접힘의 작용

SFE에서 억압장 $\Phi$는 경로의 복잡도(topological complexity)에 비례하여 억압한다. 인스턴톤은 구조적으로 비자명한 경로이므로, 접힘에 의해 추가 억압을 받는다.

Q != 0 인스턴톤의 유클리드 작용:

$$S_{\text{inst}} = \frac{8\pi^2}{g_s^2}\,|Q| = \frac{8\pi^2}{\alpha_s}\,|Q| \approx 670\,|Q|$$

SFE 접힘 인자:

$$P_{\text{survive}}(Q) = \exp\!\bigl(-|Q|\cdot S_{\text{inst}}/S_0\bigr)$$

여기서 S_0는 기준 작용(Q=0 섹터)이다. 표준 QCD에서도 인스턴톤은 $exp(-8pi^2/g^2)$로 억압되지만, SFE에서는 접힘이 추가 억압을 제공한다.

### 6.2.3 실수 스칼라의 CP 보존

SFE의 핵심 논증은 더 단순하다. 억압장 Phi는 **실수 스칼라 게이지 단일항**이다. 실수 장의 VEV는 실수이다. 히그스 포탈 결합:

$$\mathcal{L}_{\text{HP}} = -\lambda_{\text{HP}}|H|^2\Phi^2$$

이 결합은 CP를 보존한다. lambda_HP는 실수이고, |H|^2과 Phi^2 모두 CP-even이다.

따라서 억압장이 VEV를 획득할 때:
1. 새로운 CP 위반 위상이 도입되지 않는다.
2. QCD 진공의 theta 매개변수에 추가 기여가 없다.
3. 억압장의 자기일관성(부트스트랩)이 에너지를 최소화하는 진공을 선택한다.

## 6.3 에너지 최소화에 의한 theta = 0 선택

### 6.3.1 부트스트랩의 진공 선택

부트스트랩 방정식 epsilon^2 = exp(-(1-epsilon^2)D_eff)는 에너지 최소화 조건이다(10절 1.4항, 유클리드 작용-에너지 등가). theta != 0이면 QCD 진공 에너지에 추가 기여가 발생한다:

$$\rho_{\text{QCD}}(\theta) = -\chi_{\text{top}}\,(1 - \cos\theta) + \mathcal{O}(\theta^4)$$

여기서 chi_top은 위상학적 감수율(topological susceptibility):

$$\chi_{\text{top}} = f_\pi^2 m_\pi^2 \frac{m_u m_d}{(m_u+m_d)^2} \approx (75.5\;\text{MeV})^4$$

theta != 0은 에너지를 증가시킨다. 부트스트랩 방정식에서 sigma = 1-epsilon^2 = Omega_dark이므로, 진공 에너지의 변동은 sigma를 변화시키고 고정점을 이동시킨다.

### 6.3.2 정량적 추정

theta에 의한 진공 에너지 변동:

$$\delta\rho / \rho_{\text{total}} \sim \frac{\chi_{\text{top}}\,\theta^2}{\rho_c} \sim \frac{(75.5\;\text{MeV})^4\,\theta^2}{(2.5\;\text{meV})^4}$$

$$\sim \frac{3.25 \times 10^{7}\;\text{MeV}^4}{3.9 \times 10^{-47}\;\text{MeV}^4}\,\theta^2 \sim 10^{53}\,\theta^2$$

부트스트랩 고정점의 안정성 조건(10절 1.6항)에서 |f'(epsilon^2)| = D_eff epsilon^2 = 0.155이다. 고정점이 안정하려면 sigma의 변동이 epsilon^2의 수렴 반경 이내여야 한다:

$$\delta\sigma < \varepsilon^2 \approx 0.05$$

이것은 delta_rho / rho_total < 0.05를 요구하며:

$$10^{53}\,\theta^2 < 0.05 \quad \Rightarrow \quad |\theta| < 10^{-27}$$

이는 실험 상한 10^{-10}보다 17자릿수 이상 강한 제약이다.

### 6.3.3 Peccei-Quinn과의 비교

| 특성 | PQ + 액시온 | SFE |
|---|---|---|
| 새로운 대칭 | U(1)_PQ (전역) | 불필요 |
| 새로운 입자 | 액시온 a | 억압장 Phi (이미 존재) |
| theta 억압 메커니즘 | V(a) 최소화 | 부트스트랩 고정점 안정성 |
| theta 상한 | ~ 10^{-10} (동역학적) | ~ 10^{-27} (부트스트랩 안정성) |
| 추가 가정 | 1 (PQ 대칭) | 0 (SFE 구조에서 자동) |

SFE는 액시온 없이 강한 CP 문제를 해결한다. 그러나 이것이 "액시온이 없다"는 의미는 아니다. 억압장 Phi 자체가 CP 보존 스칼라로서, 구조적으로 액시온과 유사한 역할을 수행한다. 차이는 Phi가 pseudo-scalar가 아니라 scalar이며, 전역 대칭의 Goldstone 보손이 아니라는 점이다.

## 6.4 구조적 보호: 왜 theta는 루프에서도 작게 유지되는가

Peccei-Quinn 없이 theta = 0을 tree level에서 달성하더라도, 루프 보정이 theta를 재생성하는 것이 Nelson-Barr 모형의 기술적 난제이다. SFE에서는 이 문제가 발생하지 않는다:

1. **1-loop:** 히그스 포탈 lambda_HP|H|^2 Phi^2는 CP-even이므로 theta를 생성하지 않는다.
2. **2-loop:** Phi가 QCD 글루온과 직접 결합하지 않는다(색 단일항). Higgs를 매개로 한 2-loop 기여는 (lambda_HP/(16pi^2))^2 ~ (delta^2/16pi^2)^2 ~ 10^{-8} 수준으로, theta < 10^{-10} 조건을 자동으로 만족한다.
3. **모든 차수:** Phi는 게이지 단일항이므로 QCD theta 항에 대한 복소 위상 기여가 구조적으로 금지된다. 이것은 게이지 대칭에 의한 보호(gauge symmetry protection)이다.

## 6.5 예측과 검증

### 6.5.1 nEDM 예측

SFE의 theta 상한 ~ 10^{-27}로부터:

$$|d_n^{\text{SFE}}| < 10^{-16} \times 10^{-27}\;\text{e cm} = 10^{-43}\;\text{e cm}$$

이것은 실질적으로 d_n = 0 예측이다. 차세대 nEDM 실험(n2EDM, ~10^{-28} e cm)에서도 검출 불가능하다.

### 6.5.2 액시온 탐색에 대한 함의

SFE에서 강한 CP 문제는 억압장에 의해 해결되므로, QCD 액시온은 불필요하다. 그러나 이것이 "액시온이 존재하지 않는다"는 의미는 아니다. SFE는 액시온의 존재 여부에 대해 불가지적(agnostic)이다. 다만, 강한 CP 문제의 해결을 위해 액시온이 **필요하지는** 않다.

만약 ADMX, ABRACADABRA, CASPEr 등의 실험에서 QCD 액시온이 발견된다면, SFE는 이를 수용할 수 있다(추가적인 경로적분 경로로 해석). 발견되지 않아도 SFE는 영향받지 않는다.

| 실험 | 시기 | SFE 예측 |
|---|---|---|
| n2EDM | ~2025-2030 | d_n < 10^{-28} e cm: 미검출 |
| ADMX (QCD 액시온) | 진행 중 | 액시온 불필요하나 존재 가능 |
| CASPEr | ~2030 | 동일 |

## 6.6 요약

SFE의 강한 CP 해결은 세 가지 독립 논증의 합이다:

1. **실수 스칼라 보호:** Phi는 실수 게이지 단일항이므로 CP 위반 위상을 도입하지 않는다.
2. **부트스트랩 안정성:** 에너지 최소화 조건이 theta = 0을 동역학적으로 선택한다. 상한 ~ 10^{-27}.
3. **루프 보호:** 게이지 단일항 구조에 의해 모든 루프 차수에서 theta 재생성이 억압된다.

추가 대칭(PQ), 추가 입자(액시온), 추가 미세조정 없이 SFE의 기존 구조만으로 해결된다.
