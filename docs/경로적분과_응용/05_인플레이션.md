# 5. 인플레이션: 접힘 개시 이전의 우주

## 5.1 문제의 정의

표준 빅뱅 우주론은 세 가지 초기조건 문제를 안고 있다:

1. **평탄성 문제.** 우주의 공간 곡률 |Omega_k| < 0.001은 초기에 |Omega_k| < 10^{-60} 수준의 미세조정을 요구한다.
2. **지평선 문제.** CMB의 온도 균일성(Delta T/T ~ 10^{-5})은, 빛의 속도로도 인과적 접촉이 불가능한 영역들이 왜 같은 온도인지 설명하지 못한다.
3. **자기단극자 문제.** GUT 상전이에서 생성되는 위상학적 결함이 관측되지 않는다.

Guth(1981)의 인플레이션 가설: 초기 우주가 ~10^{-36}초 동안 지수적으로 팽창(~60 e-folds)하면 세 문제가 동시에 해소된다. 그러나 이 팽창을 구동하는 "인플라톤" 장의 정체는 40년간 미정이다.

## 5.2 CE에서의 인플레이션: 접힘 개시 이전 상태

### 5.2.1 핵심 해석

CE의 부트스트랩 방정식(본문 3절):

$$\varepsilon^2 = \exp\!\bigl(-(1-\varepsilon^2)\,D_{\text{eff}}\bigr)$$

이 방정식의 고정점 epsilon^2 = 0.04865는 우주의 **현재** 상태이다. 그러면 고정점 이전의 상태는 무엇인가?

고정점 반복의 시간 역추적:

| 반복 | epsilon^2 | Omega_b | Omega_dark | sigma |
|---|---|---|---|---|
| 현재 (고정점) | 0.04865 | 4.87% | 95.1% | 0.951 |
| -1 | 0.0500 | 5.00% | 95.0% | 0.950 |
| -2 | 0.0534 | 5.34% | 94.7% | 0.947 |
| -5 | 0.109 | 10.9% | 89.1% | 0.891 |
| -10 | 0.529 | 52.9% | 47.1% | 0.471 |
| 시점 0 | 1.000 | 100% | 0% | 0 |

**시점 0**: epsilon = 1, 억압 없음. 모든 경로가 살아남는다.

이것은 접힘이 시작되기 **이전**의 상태이다. 광명장 Phi가 아직 VEV를 획득하지 않은 대칭상(symmetric phase)이다.

### 5.2.2 대칭상의 물리

광명장의 멕시코 모자 퍼텐셜:

$$V(\Phi) = -\frac{1}{2}\mu^2\Phi^2 + \frac{1}{4}\lambda\Phi^4$$

대칭상(Phi = 0):

$$V(0) = 0, \quad V''(0) = -\mu^2 < 0 \quad \text{(불안정 극대)}$$

깨진상(Phi = v_Phi):

$$V(v_\Phi) = -\frac{\mu^4}{4\lambda} < 0 \quad \text{(안정 극소)}$$

대칭상과 깨진상의 에너지 차이:

$$\Delta V = V(0) - V(v_\Phi) = \frac{\mu^4}{4\lambda}$$

이 에너지 차이가 인플레이션을 구동하는 진공 에너지이다.

### 5.2.3 인플라톤 = 광명장

CE에서 인플라톤은 별도의 장이 아니다. 광명장 Phi 자체가 인플라톤이다:

| 인플레이션 표준 | CE 대응 |
|---|---|
| 인플라톤 phi | 광명장 Phi |
| 인플라톤 퍼텐셜 V(phi) | 멕시코 모자 V(Phi) |
| 대칭상 (phi = 0) | 접힘 이전 (epsilon = 1) |
| 깨진상 (phi = v) | 접힘 완료 (epsilon^2 = 0.04865) |
| 느린 구름 (slow-roll) | 부트스트랩 반복의 수렴 |
| 재가열 (reheating) | 접힘으로 방출된 에너지 -> 바리온 물질 |

## 5.3 느린 구름 조건과 비최소 결합

### 5.3.1 에너지 스케일 문제

표준 인플레이션은 V^{1/4} ~ 10^{16} GeV (GUT 스케일)을 요구한다. CE의 광명장 스케일은 M_CE = v_EW delta = 43.8 GeV로 13자릿수 이상 낮다.

이 문제는 Bezrukov & Shaposhnikov(2008)의 히그스 인플레이션과 동일한 구조로 해결된다: 비최소 중력 결합.

### 5.3.2 비최소 결합

광명장과 중력의 비최소 결합:

$$S = \int d^4x \sqrt{-g}\left[\frac{M_P^2 + \xi\Phi^2}{2}R - \frac{1}{2}(\partial\Phi)^2 - V(\Phi)\right]$$

Jordan 프레임에서 Einstein 프레임으로 Weyl 변환:

$$\tilde{g}_{\mu\nu} = \Omega^2 g_{\mu\nu}, \quad \Omega^2 = 1 + \frac{\xi\Phi^2}{M_P^2}$$

Einstein 프레임의 유효 퍼텐셜:

$$U(\chi) = \frac{V(\Phi(\chi))}{\Omega^4}$$

대역(Phi >> M_P / sqrt(xi)):

$$U(\chi) \to \frac{\lambda}{4\xi^2}M_P^4\left(1 - e^{-\sqrt{2/3}\,\chi/M_P}\right)^2$$

이것은 Starobinsky(1980)의 R^2 인플레이션과 동일한 고원(plateau) 형태이다.

### 5.3.3 CE의 비최소 결합 상수

CE에서 비최소 결합 xi는 독립 매개변수가 아니다. 본문 14절의 동적 암흑에너지 분석에서:

$$\xi = \alpha_s^{1/3} = 0.490$$

이 값은 DESI 관측의 w_0 = -0.770과 일치하도록 결정된다(0.1% 정합). 인플레이션과 현재 우주의 동적 DE를 같은 상수가 지배한다.

## 5.4 예측: 스펙트럼 지수와 텐서-스칼라비

### 5.4.1 느린 구름 매개변수

Einstein 프레임에서 고원 퍼텐셜의 느린 구름 매개변수:

$$\epsilon_V = \frac{M_P^2}{2}\left(\frac{U'}{U}\right)^2 = \frac{4}{3}\frac{1}{(e^{\sqrt{2/3}\,\chi/M_P}-1)^2}$$

$$\eta_V = M_P^2\frac{U''}{U} = -\frac{4}{3}\frac{e^{\sqrt{2/3}\,\chi/M_P}-2}{(e^{\sqrt{2/3}\,\chi/M_P}-1)^2}$$

e-folds 수 N과의 관계:

$$N \approx \frac{3}{4}e^{\sqrt{2/3}\,\chi_*/M_P}$$

여기서 chi_*는 CMB 관측 스케일이 지평선을 벗어나는 시점의 장값이다.

### 5.4.2 관측 예측

N = 55-60 e-folds에서:

$$n_s = 1 - 6\epsilon_V + 2\eta_V \approx 1 - \frac{2}{N}$$

$$r = 16\epsilon_V \approx \frac{12}{N^2}$$

| 관측량 | CE 예측 (N=60) | Planck 2018 관측 | 상태 |
|---|---|---|---|
| n_s | 0.967 | 0.9649 +/- 0.0042 | 0.5 sigma |
| r | 0.0033 | < 0.036 (95% CL) | 양립 |
| dns/dlnk | -5.6 x 10^{-4} | -0.0045 +/- 0.0067 | 양립 |

이 예측은 Starobinsky / 히그스 인플레이션과 동일한 보편 고원 예측(universal plateau prediction)이다. CE의 고유 기여는 "어떤 장이 인플라톤인가"에 대한 답(= 광명장)과 xi = alpha_s^{1/3}의 이론적 동기 부여이다.

## 5.5 재가열과 바리온 물질의 기원

인플레이션 종료 후 광명장이 VEV 주변에서 진동하면서 에너지를 SM 입자로 전달한다. 히그스 포탈 결합 lambda_HP = delta^2를 통해:

$$\Gamma(\Phi \to HH^*) \sim \frac{\lambda_{HP}^2 v_\Phi^2}{8\pi m_\Phi}$$

재가열 온도:

$$T_R \sim \sqrt{\Gamma \cdot M_P} \sim \sqrt{\frac{\delta^4 v_\Phi^2 M_P}{8\pi m_\Phi}}$$

CE 매개변수를 대입하면 T_R ~ O(10) GeV 수준으로, 전자약 스케일 이하의 낮은 재가열 온도를 예측한다. 이는 GUT 바리오제네시스를 배제하지만, 전자약 바리오제네시스 / 저에너지 렙토제네시스와 양립한다.

## 5.6 시뮬레이션 우주론과의 연결

본문 17.4절의 시뮬레이션 해석에서, 인플레이션은 부트스트랩 연산의 초기 반복에 대응한다:

- **인플레이션** = 연산 시작. 모든 경로가 열려 있다(epsilon = 1). 해가 수렴하지 않은 상태.
- **느린 구름** = 반복 수렴 과정. 각 반복마다 epsilon이 감소하며 접힘이 진행된다.
- **재가열** = 연산 완료. 접혀진 에너지가 바리온 물질로 방출된다.
- **현재 우주** = 고정점. 연산이 수렴한 정상 상태.

우주의 "시작"은 시간의 시작이 아니라 **연산의 시작**이다.

## 5.7 열린 문제

1. **e-folds 수의 유도.** N = 55-60은 입력이며 CE에서 유도되지 않는다. N이 D_eff나 d의 함수로 결정되는지 여부는 미해결이다.
2. **비가우스성.** 고원 모형은 극도로 작은 비가우스성(f_NL ~ 10^{-2})을 예측한다. CMB-S4 수준에서도 검출 어려울 수 있다.
3. **xi = alpha_s^{1/3}의 유도.** 현재 DESI w_0 정합에서 결정되지만, 이론적 유도는 미완성이다. 본문 17.2절 과제 8.
4. **멀티필드 효과.** 광명장과 히그스가 동시에 작용하는 2-field 인플레이션의 가능성.
