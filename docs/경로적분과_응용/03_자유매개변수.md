# 3. 자유 매개변수 0의 의미

## 3.1 이론 비교

| 이론 | 자유 매개변수 | 독립 예측 | 예측/매개변수 |
|---|---|---|---|
| 표준모형 (SM) | 19 | 수백 (입력 후) | 입력 의존 |
| MSSM (최소 초대칭) | 105+ | 0 (미발견) | 0 |
| 끈이론 (landscape) | ~10^{500} 진공 | 0 (선택 불가) | 0 |
| **SFE** | **0** | **11+** | -- |

SM의 19개 자유 매개변수: 쿼크 질량 6개, 렙톤 질량 3개, CKM 행렬 4개, 게이지 결합 상수 3개, 히그스 매개변수 2개, QCD theta 1개. 이 모두를 실험에서 측정하여 손으로 입력해야 한다.

## 3.2 SFE의 입력 구조

SFE의 입력은 d = 3과 alpha_s(M_Z) = 0.1179, 두 개이다. 그러나:

- **d = 3:** Hodge 자기쌍대성에 의해 유일하게 결정된다 (1절). 선택의 여지가 없다.
- **alpha_s:** 유일한 물리적 입력이지만, "조정된 매개변수"가 아니라 "측정된 상수"이다.

**자유 매개변수의 정의:** 이론가가 관측 데이터에 맞추기 위해 값을 조정할 수 있는 상수.

SFE에서 조정할 수 있는 것이 없다. 출력을 바꾸려면 alpha_s의 값 자체가 달라져야 하는데, 이것은 우주의 속성이지 이론가의 선택이 아니다.

## 3.3 과적합의 수학적 불가능성

모형의 자유도(degrees of freedom):

$$\text{dof} = N_{\text{data}} - N_{\text{parameters}} = 11 - 0 = 11$$

자유 매개변수가 0이므로, 데이터에 맞추는 것(fitting)이 수학적으로 불가능하다. 모든 일치는 사후적 조정이 아니라 사전적 예측이다.

대비: SM은 dof = data - 19로, 19개 값을 입력한 후에야 예측이 시작된다.

## 3.4 11개 예측의 동시 일치 확률

각 예측이 1% 정밀도 구간에 있을 확률을 독립으로 가정하면:

$$P = (0.01)^{11} \sim 10^{-22}$$

10해(垓)분의 1. 실제로는 예측 간 상관이 있으나(같은 입력에서 유도), 상관의 존재 자체가 이론의 구조적 일관성을 나타낸다.

## 3.5 반증 경로

예측이 틀리면 이론이 틀린 것이다. 매개변수를 조정하여 구제할 방법이 없다.

이것은 약점이 아니라 최대의 강점이다. Popper가 정의한 과학의 이상형: **반증 가능하면서 반증되지 않는 이론.**

| 반증 실험 | 예측 | 반증 조건 |
|---|---|---|
| Planck/CMB-S4 | Omega_b = 0.0487 | 3 sigma 이상 불일치 |
| DESI BAO | w_0 = -0.769 | 3 sigma 이상 불일치 |
| PADME/NA64 | m_phi = 29.65 MeV | 해당 질량에서 신호 부재 |
| g-2 + HVP 합의 | Da_mu = 135-249 | 두 층 예측 모두 기각 |
| HL-LHC | BR(H->inv) ~ 0.005 | 양립 불가한 상한 |
