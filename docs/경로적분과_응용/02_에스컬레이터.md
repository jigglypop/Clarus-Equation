# 2. 3+1 구조의 보편성

## 2.1 4개 힘의 기원: N_forces = d + 1

SFE에서 자연의 힘은 두 종류로 분류된다:

1. **게이지 힘 (d개):** d = 3 차원의 하강 분할 {3, 2, 1}에 의해 생성되는 SU(3) x SU(2) x U(1).
2. **중력 (1개):** 접힘 메커니즘 자체. 억압장 Phi = delta^2 S / delta gamma^2 = 곡률.

$$N_{\text{forces}} = d + 1 = 3 + 1 = 4$$

게이지 힘 3개는 접힘의 **결과**이고, 중력은 접힘 **자체**이다. 이것이 중력이 다른 세 힘과 근본적으로 다른 이유이다. 중력은 게이지장 위에 정의된 구조가 아니라, 게이지 구조를 생성하는 상위 메커니즘이다.

## 2.2 3+1 패턴의 반복

SFE 구조에서 3+1 분할은 단일한 기원(d = 3)에서 출발하지만, 물리의 여러 층위에서 반복 출현한다:

| 구조 | 3 | +1 | 기원 |
|---|---|---|---|
| 힘 | SU(3), SU(2), U(1) | 중력 | 하강 분할 + 접힘 |
| 시공간 | 공간 3차원 | 시간 1차원 | 로렌츠 서명 |
| 세대 | N_gen = 3 | 1개 커플링 (alpha_s) | Hodge + 곱적 분해 |
| 색 전하 | 3색 (r, g, b) | 1개 색중성 (바리온) | N_c = d = 3 |
| 유효 차원 | D_eff = 3 + delta | delta = 전자약 혼합 | Z 채널 기여 |

## 2.3 구조 대응표

SFE의 전체 구조를 단일 도표로 압축한다:

| 물리 | SFE 기원 |
|---|---|
| 건물이 3층인 이유 | d = 3 (Hodge 자기쌍대성, 1절) |
| 각 층의 정체 | SU(3) / SU(2) / U(1) (하강 분할, 본문 15.5절) |
| 층 사이의 메커니즘 | 중력 = 접힘 = Phi (본문 15.12절) |
| 기하학적 도출 | alpha_s = 0.11789 (1/(2pi) 자기일관 풀이에서 도출) |
| 접힘의 세기 | 억압장 Phi |
| 접히지 않은 분율 | 바리온 물질 (4.87%) |
| 접힘 과정의 양자 요동 | 암흑물질 (25.9%) |
| 접힘의 진공 에너지 | 암흑에너지 (69.2%) |

## 2.4 설계도의 최소 정보

SFE 우주의 전체 설계도를 읽는 데 필요한 정보:

1. **d = 3** -- Hodge 자기쌍대성에 의해 유일하게 결정. 선택의 여지 없음.
2. **alpha_s(M_Z) = 0.11789** -- alpha_total = 1/(2pi) 자기일관 연립계에서 기하학적으로 도출. 외부 입력이 아닌 내부 도출값.

이 두 가지로부터 유도되는 양:

$$d=3 \xrightarrow{\text{Hodge}} N_c = 3, \quad N_{\text{gen}} = 3, \quad SU(3) \times SU(2) \times U(1)$$

$$\alpha_s \xrightarrow{4\alpha_s^{4/3}} \sin^2\theta_W \xrightarrow{\delta} D_{\text{eff}} \xrightarrow{\text{bootstrap}} \Omega_b,\,\Omega_\Lambda,\,\Omega_{DM}$$

$$\delta^2 = \lambda_{HP} \xrightarrow{m_p\lambda_{HP}} m_\phi \xrightarrow{F = 1+\alpha_s D_{\text{eff}}} \Delta a_\mu,\,\Delta r_p^2,\,w_0$$

## 2.5 양자중력에 대한 함의

중력이 "4번째 게이지 힘"으로 양자화되지 않는 이유가 구조적으로 설명된다:

- 게이지 힘 3개: 접힘의 **각 층**이다. 고정된 대칭 구조(배경) 위에서 정의되므로 양자화가 가능하다.
- 중력: 접힘을 **만드는 메커니즘** 자체이다. 배경을 변형하는 연출이므로, 그 위에 정의된 양자장론의 틀 안에서 양자화하기가 본질적으로 어렵다.

이것은 기존 양자중력 접근(끈이론, 루프 양자중력)이 난관에 부딪히는 구조적 이유를 제공한다: 중력은 게이지 힘과 **같은 종류**가 아니기 때문이다.
