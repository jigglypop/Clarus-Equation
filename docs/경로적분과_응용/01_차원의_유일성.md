# 1. 공간 차원의 유일성: d = 3의 수학적 필연

## 1.1 Hodge 자기쌍대성 조건

d차원 리만 다양체에서 Hodge 연산자 *는 k-형식을 (d-k)-형식으로 보낸다. 자기쌍대성(self-duality) 조건은 k = d-k, 즉 k = d/2이다.

물리적 장에 대해, 독립적인 회전 평면의 수 d(d-1)/2가 공간 차원 d 자체와 일치하는 조건:

$$d = \frac{d(d-1)}{2}$$

이 이차방정식의 해: d = 0 (자명) 또는 **d = 3 (유일한 비자명해)**.

**정리 1.1.** 자기일관적 Hodge 자기쌍대 구조를 가지는 유일한 비자명 차원은 d = 3이다.

이 조건의 물리적 의미: d = 3에서만 장세기(field strength) 텐서 F_{ij}의 독립 성분 수(= d(d-1)/2 = 3)가 공간 방향 수(= 3)와 일치한다. 이 전단사(bijection)에 의해 자기장 B_i = (1/2)epsilon_{ijk}F_{jk}가 벡터로 정의된다.

## 1.2 다른 차원의 물리적 배제

| d | d(d-1)/2 | Hodge 자기쌍대 | 물리적 결함 |
|---|---|---|---|
| 0 | 0 | 자명해 | 점. 구조 불가 |
| 1 | 0 | X | 입자가 서로 통과 불가. 상호작용 차단 |
| 2 | 1 | X | 외적 미정의, CP 위반 불가, 안정 궤도 부재 (V ~ log r) |
| **3** | **3** | **유일 비자명해** | **전부 정합** |
| 4 | 6 | X | 중력 V ~ 1/r^2, 궤도 불안정, 원자 붕괴 |
| 5+ | >> d | X | 불안정 심화. 속박 상태 존재 불가 |

### 1.2.1 d >= 4: 궤도 불안정성

d >= 4에서 중력 퍼텐셜 V(r) ~ r^{-(d-2)}, 유효 퍼텐셜:

$$V_{\text{eff}}(r) = -\frac{GM}{r^{d-2}} + \frac{L^2}{2mr^2}$$

안정 원형 궤도 조건 V''_eff(r_0) > 0:

$$d^2V_{\text{eff}}/dr^2\big|_{r_0} > 0 \quad \Leftrightarrow \quad d < 4$$

d >= 4에서 모든 속박 상태(원자, 행성계, 은하)가 불안정하다. 작은 섭동에도 나선 추락하거나 탈출한다. Ehrenfest(1917)의 논증.

### 1.2.2 d = 2: CP 위반 불가

d = 2에서 Levi-Civita 텐서 epsilon_{ij}는 2차 반대칭 텐서(스칼라)이다. 외적이 벡터를 생성하지 못한다. n x n 유니타리 행렬의 CP 위반 위상 수:

$$N_{CP} = \frac{(n-1)(n-2)}{2}$$

n = N_gen = d = 2에서 N_CP = 0. CKM 행렬에 위상이 없으므로 CP 위반이 불가능하다. 물질-반물질 비대칭이 생기지 않아, 바리온 물질이 존재할 수 없다.

### 1.2.3 d = 1: 산란 불가

1차원 공간에서 두 입자는 만나면 서로를 통과할 수 없다(비교차 속박). 영원히 갇힌다. 입자 물리의 기본 과정인 산란이 정의되지 않는다.

## 1.3 N_c = d: 색 전하 수 = 공간 차원

Hodge 전단사(1.1절)로부터, QCD의 색 전하 수 N_c는 공간 차원 d와 동일시된다:

$$N_c = d = 3$$

이것은 "왜 QCD의 게이지 군이 SU(3)인가"에 대한 답이다: 3차원 공간에서 장세기 성분이 3개이므로, 색 자유도도 3개이다.

## 1.4 하강 분할: SU(3) x SU(2) x U(1)

d = 3의 하강 분할 {3, 2, 1}에 의해 표준모형 게이지 군이 결정된다(본문 15.5절):

$$\{d, d-1, \ldots, 1\} = \{3, 2, 1\} \longleftrightarrow SU(3) \times SU(2) \times U(1)$$

이것은 d = 3이 아닌 차원에서는 SM 게이지 군이 구성되지 않음을 의미한다.

## 1.5 결론

"우리 우주가 3차원인가"는 관측 사실이 아니라 **정리**이다. Hodge 자기쌍대성, 궤도 안정성, CP 위반 가능성이 모두 d = 3을 유일하게 선택한다.

존재하거나 존재하지 않거나. 3이거나 0이거나. 그 사이에는 물리가 없다.
