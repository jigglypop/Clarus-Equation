# 4층: 입자물리 -- 격자에서 SM으로

## 개관

격자 구조에서 표준모형(SM)의 핵심 관측량을 유도한다. QCD 진공 안정성, 히그스 질량, CKM 혼합, CP 위반. 이 층은 1-3층의 기반 위에서 입자물리의 구체적 예측을 생산한다.

---

## $\theta_{\text{QCD}} = 0$ -- 강한 CP 문제의 해결

### 문제의 배경

QCD 라그랑지안에는 원칙적으로 CP 위반 항이 존재할 수 있다:

$$\mathcal{L}_\theta = \theta \frac{g_s^2}{32\pi^2} G_{\mu\nu}^a \tilde{G}^{a\mu\nu}$$

$\theta \neq 0$이면 중성자가 전기 쌍극자 모멘트(EDM)를 가진다. 관측적으로 $|d_n| < 1.8 \times 10^{-26}\;e\cdot\text{cm}$이므로 $|\theta| < 10^{-10}$. 왜 이렇게 작은가? 이것이 "강한 CP 문제"이다.

표준모형에서 $\theta$는 자유 파라미터이다. Peccei-Quinn 대칭이나 액시온 등의 추가 메커니즘이 제안되어 왔다.

### CE의 해결: 세 가지 독립 논증

**1. 실수 스칼라 보호**

광명장 $\Phi$는 실수 게이지 단일항이다. 복소수가 아니므로 CP 위반 위상을 구조적으로 도입할 수 없다. $\Phi$의 어떤 결합도 $\theta$ 항을 생성하지 않는다.

**2. 부트스트랩 안정성**

$\theta \neq 0$이면 QCD 진공 에너지가 증가한다: $V_{\text{QCD}} \propto -\cos\theta$. 진공 에너지가 증가하면 부트스트랩 방정식의 고정점이 불안정해진다.

안정성 조건을 정량적으로 적용하면: 부트스트랩 해가 존재하려면 $|\theta| < 10^{-27}$. 관측 상한 $10^{-10}$보다 17자릿수 더 엄격하다.

**3. 루프 보호**

$\Phi$가 게이지 단일항이므로, QCD 게이지장과의 직접 결합이 없다. 따라서 루프 다이어그램을 통해 $\theta$가 재생성되는 것이 모든 루프 차수에서 구조적으로 금지된다.

$$\boxed{\theta_{\text{QCD}} = 0}$$

유도 레벨: **정리**(부트스트랩 안정성 논증). 액시온이 필요 없다.

## $M_H/M_Z = F$ -- 히그스 질량

### 관계식

$$F = 1 + \alpha_s D_{\text{eff}} = 1 + 0.11789 \times 3.17776 = 1.37473$$

### 유도

SM에서 히그스 질량은 tree-level 값에 복사 보정(radiative correction)이 더해진다. 지배적 보정: top 쿼크 루프 ($y_t \approx 1$, $N_c = d = 3$).

$$M_H^2 = M_Z^2 + \Delta M_H^2$$

CE에서 이 보정의 구조를 분석한다. 차원당 요동 비율이 $\alpha_s$이고, $D_{\text{eff}}$개 차원에서 기여한다. 인자 2는 top+bottom SU(2) 이중항 구조:

$$\Delta M_H^2|_{\text{1-loop}} = M_Z^2 \times 2\alpha_s D_{\text{eff}}$$

기하급수 재합산(leading-log resummation)에 의해 1-루프 보정이 반복되면:

$$M_H^2 = M_Z^2(1 + 2\alpha_s D_{\text{eff}} + (\alpha_s D_{\text{eff}})^2 + \cdots) = M_Z^2(1 + \alpha_s D_{\text{eff}})^2 = M_Z^2 F^2$$

$$\boxed{M_H = M_Z \times F = 91.1876 \times 1.37473 = 125.35\;\text{GeV}}$$

관측: $125.10 \pm 0.14$ GeV. 차이 **0.20%** ($1.8\sigma$).

### $\lambda_H$ 역산

히그스 자기결합상수:

$$\lambda_H = \frac{M_H^2}{2v_{\text{EW}}^2} = \frac{M_Z^2 F^2}{2v_{\text{EW}}^2} = 0.1296 \quad (\text{관측 } 0.1292, \; 0.3\%)$$

유도 레벨: **가설** (0.20% 정합, $F$가 양성자 반경에서도 동일하게 등장하는 교차 검증 존재).

## $m_W/m_Z$ -- W 보손 질량비

$$m_W/m_Z = \cos\theta_W = \sqrt{1 - \sin^2\theta_W} = \sqrt{1 - 4\alpha_s^{4/3}} = 0.8768$$

관측: $80.377 / 91.188 = 0.8815$. 이 차이는 1-루프 전자약 보정에 의한 것이며, SM 내에서 계산 가능하다.

## CKM 행렬 -- 세대 간 전이 진폭

### 물리적 배경

CKM(Cabibbo-Kobayashi-Maskawa) 행렬은 쿼크의 질량 고유상태와 약한 상호작용 고유상태 사이의 변환이다. 3x3 유니터리 행렬이며, 3개의 혼합각과 1개의 CP 위반 위상으로 매개변수화된다.

격자에서 CKM은 **비대각 전이**를 나타낸다. 1층에서 대각 원소(게이지 결합)를 확정했으므로, 이제 비대각 원소(세대 간 혼합)를 유도한다.

### $|V_{cb}| = \alpha_s^{d/2}$: QCD $d$차원 터널링

인접 세대 전이(2세대 → 3세대). 이것은 QCD 진공에서의 터널링이며, $d$개 공간 방향에서 독립적으로 일어난다.

전이 확률 $|V_{cb}|^2$은 $d$개 등방적 공간 방향에서 독립 기여의 곱:

$$|V_{cb}|^2 = \prod_{k=1}^{d} \alpha_s = \alpha_s^d$$

$$\boxed{|V_{cb}| = \alpha_s^{3/2} = 0.11789^{1.5} = 0.04049} \quad \text{관측 } 0.04053 \pm 0.00072 \quad (0.1\%,\; 0.06\sigma)$$

이것은 체계에서 가장 정밀한 예측 중 하나이다.

### $|V_{us}| \approx \sin^2\theta_W$: 전자약 혼합 지배

1-2 섹터의 쿼크($u, d, s$)는 $m_q \ll \Lambda_{\text{QCD}} \approx 200$ MeV이므로 QCD 효과보다 전자약 혼합이 지배한다:

$$\boxed{|V_{us}| = \sin^2\theta_W = 4\alpha_s^{4/3} = 0.23122} \quad \text{관측 } 0.22650 \quad (2.1\%)$$

2.1% 잔차: $m_s/\Lambda_{\text{QCD}} \approx 0.47$에 의한 QCD 보정이 이 크기이다.

### $|V_{ub}| = \alpha_s^{8/3}$: SU($d$) 최대거리 전이

1세대 → 3세대의 직접 전이. SU($d$) 수반 표현의 최대 전이 경로이다. Casimir 연산자 $C_2 = d^2 - 1 = 8$에 비례하는 억압:

$$|V_{ub}|^2 = \alpha_s^{2(d^2-1)/d} = \alpha_s^{16/3}$$

$$\boxed{|V_{ub}| = \alpha_s^{8/3} = 0.00335} \quad \text{관측 } 0.00382 \quad (12\%)$$

12% 잔차에 대한 보정 계수 $c_1 \simeq 1.19$는 $O(1)$ 크기이며, 1-2 세대 질량 효과로 설명 가능하다.

## Jarlskog 불변량

$J$는 CP 위반의 관례 불변적(convention-independent) 측도이다. 어떤 매개변수화를 사용하든 같은 값이다.

세 CKM 원소의 곱으로 표현:

$$J = |V_{us}| \times |V_{cb}| \times |V_{ub}| \times \sin\delta_{CP}$$

CE 값 대입:

$$|V_{us}| \times |V_{cb}| \times |V_{ub}| = 4\alpha_s^{4/3 + 3/2 + 8/3} = 4\alpha_s^{11/2}$$

$$\boxed{J = 4\alpha_s^{11/2} \cdot \sin\delta_{CP} = 3.12 \times 10^{-5}} \quad \text{관측 } 3.08 \times 10^{-5} \quad (1.3\%)$$

$J$는 매개변수화 독립(관례 불변) 관측량이다. 역으로:

$$\sin\delta_{CP} = \frac{J_{\text{obs}}}{4\alpha_s^{11/2}} = \frac{3.08}{3.124} = 0.986$$

$\sin\delta_{CP} \approx 1$이므로 $\delta_{CP} \approx \pi/2$. 잔차 1.4%는 1-2세대 질량 보정의 크기와 일치한다. PDG의 "$\delta_{CP} \approx 1.20$ rad"는 표준 Wolfenstein 매개변수화에서의 값이며, $J$의 관례 불변적 일치(1.3%)가 물리적 검증이다.

## $\delta_{CP}$ -- $i$의 기하학적 발현

### CKM CP 위상

$N_{\text{gen}} = d = 3$에서 CKM 행렬의 재규격화 불가능한 물리적 위상은 정확히 1개이다(Kobayashi-Maskawa 정리). CP 위반 위상의 수 = $(N_{\text{gen}} - 1)(N_{\text{gen}} - 2)/2 = 1$.

CE에서 이 단일 물리적 위상은 자의적인 각도를 가질 수 없다. 복소평면에서 실수축에 대해 구조적으로 구별되는 유일한 위상은 허수축 자체, 즉 $\pi/2$이다:

$$e^{i\pi/2} = i$$

$$\boxed{\delta_{CP}^{\text{CKM}} = \frac{\pi}{2}}$$

### PMNS CP 위상: 위상 켤레

실수 광명장 배경에서, 전체 유효작용의 복소 위상은 쿼크 블록과 렙톤 블록이 상쇄되어야 한다:

$$\arg\det Y_q + \arg\det Y_\ell = 0 \pmod{2\pi}$$

따라서:

$$\boxed{\delta_{CP}^{\text{PMNS}} = -\frac{\pi}{2} = \frac{3\pi}{2}}$$

$$e^{i\delta_{CP}^{\text{CKM}}} \cdot e^{i\delta_{CP}^{\text{PMNS}}} = e^{i\pi/2} \cdot e^{-i\pi/2} = 1$$

쿼크와 렙톤의 CP 위상이 $+i$와 $-i$의 켤레쌍이며, 전체 CP-odd 위상 흐름이 닫힌다.

관측: T2K는 $\delta_{CP}^{\text{PMNS}} \sim 270^\circ = 3\pi/2$를 선호. NuFIT 최적값은 $\sim 195^\circ$. 실험 오차가 아직 크므로 최종 판정은 JUNO/Hyper-K가 맡는다.

## 5개 상수 귀속

| 상수 | 역할 | 등장하는 식 |
|---|---|---|
| $\pi$ | CKM 지수, $J = 4\alpha_s^{11/2}$, $M_H/M_Z$ | $\alpha_s$의 거듭제곱 |
| $0$ | $d = 3$: CKM 지수 $d/2$, $(d^2-1)/d$; $N_{\text{gen}} = 3 \to$ CP 위상 1개 | 세대 구조 |
| $i$ | $\delta_{CP} = \pi/2$: 허수 단위의 직접 발현 | 최대 CP 위반 |
| $e$ | 부트스트랩 안정성 $\to \theta_{\text{QCD}} = 0$ | 진공 안정성 |
| $1$ | $F = 1 + \alpha_s D_{\text{eff}}$: 보정 없으면 $M_H = M_Z$ | 기준선 |
