# 4층: 입자물리 -- 격자에서 SM으로

## 개관

격자 구조에서 SM의 핵심 관측량을 유도한다. QCD 진공 안정성, 히그스 질량, CKM 혼합, CP 위반.

---

## $\theta_{\text{QCD}} = 0$ -- (1,1) 행의 안정성

세 가지 독립 논증:

1. **실수 스칼라 보호:** 억압장 $\Phi$는 실수 게이지 단일항. CP 위반 위상을 도입하지 않음.
2. **부트스트랩 안정성:** $\theta \neq 0$은 QCD 진공 에너지를 증가시키고 부트스트랩 고정점을 불안정화. 안정성 조건이 $|\theta| < 10^{-27}$을 강제.
3. **루프 보호:** $\Phi$가 게이지 단일항이므로 모든 루프 차수에서 $\theta$ 재생성이 구조적으로 금지.

$$\boxed{\theta_{\text{QCD}} = 0}$$

유도 레벨: **정리**(부트스트랩 안정성 논증).

---

## $M_H/M_Z = F$ -- 폼팩터

### 관계식

$$F = 1 + \alpha_s D_{\text{eff}} = 1 + 0.11789 \times 3.17776 = 1.37473$$

### 유도

SM에서 히그스 질량은 tree-level에 복사 보정이 더해진다. 지배적 보정: top 쿼크 루프 ($y_t \approx 1$, $N_c = d = 3$).

$$M_H^2 = M_Z^2 + \Delta M_H^2$$

차원당 요동 비율이 $\alpha_s$이고, $D_{\text{eff}}$개 차원에서 기여. 인자 2는 top+bottom SU(2) 이중항 구조:

$$\Delta M_H^2|_{\text{1-loop}} = M_Z^2 \times 2\alpha_s D_{\text{eff}}$$

기하급수 재합산(leading-log resummation):

$$M_H^2 = M_Z^2(1 + 2\alpha_s D_{\text{eff}} + (\alpha_s D_{\text{eff}})^2 + \cdots) = M_Z^2(1 + \alpha_s D_{\text{eff}})^2 = M_Z^2 F^2$$

$$\boxed{M_H = M_Z \times F = 91.1876 \times 1.37473 = 125.35\;\text{GeV}}$$

관측: $125.10 \pm 0.14$ GeV. 차이 **0.20%** ($1.8\sigma$).

### $\lambda_H$ 역산

$$\lambda_H = \frac{M_H^2}{2v_{\text{EW}}^2} = \frac{M_Z^2 F^2}{2v_{\text{EW}}^2} = 0.1296 \quad (\text{관측 } 0.1292, \; 0.3\%)$$

유도 레벨: **가설** (0.20% 정합, $F$가 양성자 반경에서도 동일하게 등장하는 교차 검증 존재).

## $m_W/m_Z$ -- (2,2) 행의 구조

$$m_W/m_Z = \cos\theta_W = \sqrt{1 - 4\alpha_s^{4/3}} = 0.8768$$

## CKM -- 행 간 전이 진폭

격자의 비대각 전이(세대 간 혼합)는 결합상수 거듭제곱으로 억압.

### $|V_{cb}| = \alpha_s^{d/2}$: QCD $d$차원 터널링

인접 세대 전이 확률 $|V_{cb}|^2$은 헤시안의 $d$개 등방적 공간 방향에서 독립 기여의 곱:

$$|V_{cb}|^2 = \prod_{k=1}^{d} \alpha_s = \alpha_s^d$$

$$\boxed{|V_{cb}| = \alpha_s^{3/2} = 0.04049} \quad \text{관측 } 0.04053 \pm 0.00072 \quad (0.1\%,\; 0.06\sigma)$$

### $|V_{us}| \approx \sin^2\theta_W$: 전자약 지배

1-2 섹터는 $m_u, m_s \ll \Lambda_{\text{QCD}}$이므로 EWSB 혼합이 지배:

$$\boxed{|V_{us}| = \sin^2\theta_W = 4\alpha_s^{4/3} = 0.23122} \quad \text{관측 } 0.22650 \quad (2.1\%)$$

2.1% 잔차: $m_s/\Lambda_{\text{QCD}} \approx 0.47$에 의한 QCD 보정.

### $|V_{ub}| = \alpha_s^{8/3}$: SU($d$) 최대거리 전이

SU($d$) 수반 표현의 최대 전이 경로. Casimir $C_2 = d^2 - 1$에 비례하는 억압:

$$|V_{ub}|^2 = \alpha_s^{2(d^2-1)/d} = \alpha_s^{16/3}$$

$$\boxed{|V_{ub}| = \alpha_s^{8/3} = 0.00335} \quad \text{관측 } 0.00382 \quad (12\%)$$

보정 계수 $c_1 \simeq 1.19$로 $O(1)$ 크기, 자연스러움.

---

## Jarlskog 불변량

세 CKM 원소의 곱:

$$|V_{us}| \times |V_{cb}| \times |V_{ub}| = 4\alpha_s^{4/3 + 3/2 + 8/3} = 4\alpha_s^{11/2}$$

$$\boxed{J = 4\alpha_s^{11/2} \cdot \sin\delta_{CP} = 3.12 \times 10^{-5}} \quad \text{관측 } 3.08 \times 10^{-5} \quad (1.3\%)$$

---

## $\delta_{CP}$ -- $i$의 기하학적 발현

### CKM

$N_{\text{gen}} = d = 3$에서 재규격화 불가능한 물리적 위상은 정확히 1개. $i$는 복소평면의 허수축 전체를 생성하는 단일 생성원이므로:

$$\boxed{\delta_{CP}^{\text{CKM}} = \frac{\pi}{2}}$$

$e^{i\pi/2} = i$. 최대 CP 위반.

### PMNS

실수 억압장 배경에서 전블록 실수성:

$$\arg\det Y_q + \arg\det Y_\ell = 0 \pmod{2\pi}$$

위상 켤레:

$$\boxed{\delta_{CP}^{\text{PMNS}} = -\frac{\pi}{2} = \frac{3\pi}{2}}$$

$$e^{i\delta_{CP}^{\text{CKM}}} \cdot e^{i\delta_{CP}^{\text{PMNS}}} = 1$$

## 5개 상수 귀속

| 상수 | 역할 |
|---|---|
| $\pi$ | $\alpha_s$: CKM 지수, $J = 4\alpha_s^{11/2}$, $M_H/M_Z$ |
| $0$ | $d = 3$: CKM 지수 $d/2$, $(d^2-1)/d$; $N_{\text{gen}} = 3 \to$ CP 위상 1개 |
| $i$ | $\delta_{CP} = \pi/2$: 허수 단위의 직접 발현 |
| $e$ | 부트스트랩 안정성 $\to \theta_{\text{QCD}} = 0$ |
| $1$ | $F = 1 + \alpha_s D_{\text{eff}}$: 보정 없으면 $M_H = M_Z$ |
