# 3층: 부트스트랩 -- 격자의 자기일관성

## 개관

격자가 자기 자신과 정합하는 조건. 우주의 에너지 밀도 분배를 결정한다. 이 층은 수학적으로 가장 견고한 부분이다. 부트스트랩 방정식은 바나흐 고정점 정리에 의해 유일해가 보장되고, Lambert W 함수에 의한 닫힌 해가 존재한다.

---

## 부트스트랩 방정식

### 무엇을 결정하는가

우주의 에너지 밀도는 세 성분으로 나뉜다:

- **바리온 물질** $\Omega_b$: 원자, 별, 행성 등 "보통 물질" (~5%)
- **암흑 물질** $\Omega_{DM}$: 관측되지 않지만 중력적으로 존재하는 물질 (~26%)
- **암흑 에너지** $\Omega_\Lambda$: 우주 팽창을 가속시키는 에너지 (~69%)

표준 우주론에서 이 세 값은 관측으로 측정하여 넣는 매개변수이다. SFE에서는 $D_{\text{eff}} = 3.178$ 하나로부터 세 값 전부를 유도한다.

### 유도 (공리 A1--A3에서)

**(i) 경로적분에서 억압장에 의한 생존 확률**

SFE의 억압장 $\Phi$는 경로적분에서 경로를 억압한다. 한 경로가 살아남을 확률:

$$P_{\text{survive}} = \langle e^{-\Phi} \rangle$$

이것은 열역학의 분배 함수 $Z = \sum e^{-E/kT}$와 같은 구조이다. $\Phi$가 큰 경로는 지수적으로 억압된다.

**(ii) 평균장 근사**

$\langle e^{-\Phi} \rangle \approx e^{-\langle\Phi\rangle}$. 이것은 $\Phi$의 분산이 평균보다 작을 때 유효하다. SFE에서 억압장은 매끄럽고 느리게 변하므로 이 근사가 정당하다.

**(iii) 연장성 원리 (A1)**

헤시안 기대값(접힘의 "곡률")은 유효 차원에 비례한다:

$$\langle\Phi\rangle = \sigma \cdot D_{\text{eff}}$$

여기서 $\sigma$는 차원당 억압률이다. 이것은 1층의 $S(D) = S(1)^D$와 동일한 곱적 구조의 선형화이다. 직관: 접힘이 3.178차원에서 일어나면, 억압도 3.178배.

**(iv) 에너지 보존 (A2)**

$\Omega_{\text{total}} = 1$ (우주의 총 에너지 밀도는 임계 밀도와 같다 -- 관측적으로도, 인플레이션 이론적으로도). 생존한 에너지 분율이 $\varepsilon^2$이면, 억압된 분율은 $1 - \varepsilon^2$:

$$\sigma = 1 - \varepsilon^2$$

**(v) 식별 (A3, 정리 7.2): $P_{\text{survive}} = \Omega_b$**

생존 확률 $P_{\text{survive}}$을 관측 가능한 에너지 분율과 동일시한다. "접힘에서 살아남은 경로"가 곧 "바리온 물질"이다.

### 공리 A3의 유일성 (정리 7.2)

식별 사상 $I: (0,1] \to (0,1]$이 다음 조건을 만족해야 한다:

1. **연속성**: $P$가 연속적으로 변하면 $\Omega_b$도 연속적으로 변한다
2. **경계 정규화**: $I(1) = 1$ (억압 없으면 전부 바리온), $I(0) = 0$ (완전 억압이면 바리온 없음)
3. **독립 곱성**: $I(PQ) = I(P)I(Q)$ (독립 사건의 확률은 곱해진다)

이 세 조건을 만족하는 유일한 연속 함수는:

$$I(P) = P^c \quad (c > 0)$$

이것은 함수방정식 $f(xy) = f(x)f(y)$의 유일한 연속해가 $f(x) = x^c$라는 코시 정리의 응용이다.

$c$의 결정: 약억압 한계($D \to 0$)에서 $P \to 1 - \sigma D$이고, 에너지 보존에 의해 $\Omega_b \to 1 - \sigma D$. 1차항이 일치하려면 $c = 1$.

$$\boxed{\Omega_b = P_{\text{survive}} = \varepsilon^2}$$

### 부트스트랩 방정식 완성

(i)-(v)를 결합:

$$\varepsilon^2 = e^{-\langle\Phi\rangle} = e^{-\sigma D_{\text{eff}}} = e^{-(1-\varepsilon^2)D_{\text{eff}}}$$

$$\boxed{\varepsilon^2 = e^{-(1-\varepsilon^2)D_{\text{eff}}}}$$

이것은 $\varepsilon^2$에 대한 초월 방정식이다. $D_{\text{eff}} = 3.178$을 넣으면 $\varepsilon^2$이 결정된다.

### 물리 가지 고정점의 존재와 유일성 (정리 7.3)

$F_D(x) \equiv e^{-(1-x)D}$로 정의하면, 부트스트랩 방정식은 $x = F_D(x)$이다.

물리 구간 $I_D = [0, 1/D]$에서:

1. **$F_D$는 $I_D$를 자기 자신으로 사상한다**: $F_D(I_D) \subset I_D$
2. **축소 사상이다**: $|F_D'(x)| = De^{-(1-x)D} \leq De^{-(D-1)} \equiv k_D$. $D > 1$이면 $k_D < 1$.
3. **바나흐 고정점 정리**: 축소 사상의 고정점은 존재하고 유일하다. 임의의 초기값에서 시작하여 반복하면 수렴한다.

$D_{\text{eff}} = 3.17776$이면 $k_D = 3.178 \times e^{-2.178} = 3.178 \times 0.1131 = 0.360 < 1$. 축소 사상 조건 충족.

**자명해 $x = 1$**은 물리 가지 밖이다. $x = 1$이면 $\varepsilon^2 = 1$, 억압 없음. 이것은 $d = 0$(접힘 이전)에 대응하며, Lambert W의 $W_{-1}$ 가지이다.

## Lambert W에 의한 닫힌 해

### Lambert W 함수란

Lambert W 함수는 $we^w = z$의 해 $w = W(z)$이다. $z$가 주어지면 $w$를 돌려준다.

### 부트스트랩 방정식의 변환

$x = \varepsilon^2$, $D = D_{\text{eff}}$로 놓는다.

$$x = e^{-(1-x)D} = e^{-D} \cdot e^{xD}$$

양변에 $(-xD)$를 곱한다:

$$(-xD) = (-xD) \cdot \frac{e^{-D} \cdot e^{xD}}{1} \quad \text{(원래 방정식 이용)}$$

이것을 정리하면:

$$(-xD) \cdot e^{-xD} = -D \cdot e^{-D}$$

이것은 $we^w = z$ 꼴이다. $w = -xD$, $z = -De^{-D}$:

$$-xD = W(-De^{-D})$$

$$\boxed{\Omega_b = x = -\frac{W_0(-D_{\text{eff}}\,e^{-D_{\text{eff}}})}{D_{\text{eff}}} = 0.04865}$$

관측: $\Omega_b = 0.0486 \pm 0.0010$ (Planck 2018). 텐션: **$0.05\sigma$**.

$W_0$는 Lambert W의 주가지(principal branch). 이것이 물리 가지($d = 3$)에 대응한다.
$W_{-1}$은 부가지. 이것이 자명해($\varepsilon^2 = 1$, $d = 0$)에 대응한다.

## $\Omega_\Lambda$, $\Omega_{DM}$ -- 암흑 에너지/물질 분할

### 물리적 해석

억압된 에너지 $\Omega_\Phi = 1 - \varepsilon^2 = 0.9514$가 두 성분으로 분리된다:

- **진공 에너지 $\Omega_\Lambda$** (0차, tree-level): 억압장의 진공 기대값에 의한 에너지. 결합상수에 무관한 "배경".
- **양자 요동 응축 $\Omega_{DM}$** (1-루프): 가상 입자-반입자 쌍의 생성-소멸에 의한 에너지. 결합상수의 세기에 비례.

### 분할 비율

연장성 원리(A1)에 의해, 양자 요동(1-루프)의 기여는 각 유효 차원에서 결합상수만큼이다:

$$\frac{\Omega_{DM}}{\Omega_\Lambda} = \alpha_s \cdot D_{\text{eff}}$$

**왜 $\alpha_s$만인가?**

1-루프 기여는 가장 강한 결합에 의해 지배된다. 세 게이지 결합의 비:

| 결합 | 값 | 1-루프 기여 비율 |
|---|---|---|
| $\alpha_s$ (강력) | 0.118 | **74%** |
| $\alpha_w$ (약력) | 0.034 | 21% |
| $\alpha_{em}$ (전자기) | 0.008 | 5% |

$\alpha_s$가 총 기여의 3/4을 점유한다. $\alpha_w + \alpha_{em}$은 차차수(NLO) 보정이며, 현재 관측 정밀도(~2%)에서 $\alpha_s$-only 근사의 오차(0.13%)가 관측 오차보다 작다.

### 연립과 해

에너지 보존:

$$\Omega_\Lambda + \Omega_{DM} = 1 - \varepsilon^2$$

분할 비율과 결합:

$$\boxed{\Omega_\Lambda = \frac{1 - \varepsilon^2}{1 + \alpha_s D_{\text{eff}}} = \frac{0.9514}{1.3747} = 0.6921} \quad \text{관측 } 0.6847 \quad (1.1\%)$$

$$\boxed{\Omega_{DM} = \frac{(1-\varepsilon^2)\alpha_s D_{\text{eff}}}{1 + \alpha_s D_{\text{eff}}} = \frac{0.9514 \times 0.3747}{1.3747} = 0.2593} \quad \text{관측 } 0.2589 \quad (0.13\%)$$

수치 분해: $\alpha_s D_{\text{eff}} = 0.11789 \times 3.17776 = 0.37465$.

유도 레벨: **가정**(진공/요동 분리) + **정리**(에너지 보존에 의한 비율 결정).

## 완전한 닫힌 형태: 모든 것이 $D_{\text{eff}}$의 함수

$D \equiv D_{\text{eff}}$로 모든 우주론 + 입자물리 관측량을 표현할 수 있다:

$$\Omega_b = -\frac{W_0(-De^{-D})}{D}$$

$\delta = D - 3$이고 $\sin^2\theta_W$를 $\delta$에서 역산:

$$\sin^2\theta_W = \frac{1 - \sqrt{1 - 4(D-3)}}{2}$$

$$\alpha_s = \left(\frac{\sin^2\theta_W}{4}\right)^{3/4} = \left(\frac{1-\sqrt{1-4(D-3)}}{8}\right)^{3/4}$$

$$\Omega_\Lambda = \frac{1 + W_0(-De^{-D})/D}{1 + \alpha_s D}$$

**우주 전체가 $D_{\text{eff}}$라는 단일 숫자의 함수이다.** $D_{\text{eff}} = 3.17776$. 이 숫자가 $d = 3$(정수)과 $\delta = 0.178$(분수)의 합이며, 둘 다 $\alpha_s$와 $\pi$에서 유도된다.
