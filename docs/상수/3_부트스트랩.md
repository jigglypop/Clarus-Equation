# 3층: 부트스트랩 -- 격자의 자기일관성

## 개관

격자가 자기 자신과 정합하는 조건. 우주의 에너지 밀도 분배를 결정한다.

---

## 부트스트랩 방정식

### 유도 (A1--A3에서)

**(i)** 경로적분에서 억압장 $\Phi$에 의한 생존 확률: $P_{\text{survive}} = \langle e^{-\Phi} \rangle$.

**(ii)** 평균장 근사: $\langle e^{-\Phi} \rangle \approx e^{-\langle\Phi\rangle}$.

**(iii)** 연장성 (A1): 헤시안 기대값은 유효 차원에 비례. $\langle\Phi\rangle = \sigma \cdot D_{\text{eff}}$. 1절의 $S(D) = S(1)^D$와 동일 구조.

**(iv)** 에너지 보존 (A2): $\Omega_{\text{total}} = 1$에서 $\varepsilon^2$이 살아남으면, 차원당 억압률 $\sigma = 1 - \varepsilon^2$.

**(v)** 식별 (A3, 정리 7.2): $P_{\text{survive}} = \varepsilon^2 = \Omega_b$.

결합:

$$\boxed{\varepsilon^2 = e^{-(1-\varepsilon^2)D_{\text{eff}}}}$$

### A3 식별의 유일성 (정리 7.2)

식별 사상 $I: (0,1] \to (0,1]$이 연속성, 경계 정규화($I(1)=1$, $I(0)=0$), 독립 곱성을 만족하면:

$$I(P) = P^c \quad (c > 0)$$

약억압 한계에서 에너지 보존과 1차항이 일치하므로 $c = 1$:

$$\boxed{\Omega_b = P_{\text{survive}} = \varepsilon^2}$$

### 물리 가지 고정점 (정리 7.3)

$F_D(x) \equiv e^{-(1-x)D}$, 물리 구간 $I_D = [0, 1/D]$에서:

1. $F_D(I_D) \subset I_D$
2. $|F_D'(x)| \le De^{-(D-1)} \equiv k_D < 1$ ($D > 1$)
3. 바나흐 고정점 정리로 유일 고정점 존재 및 반복 수렴

$D_{\text{eff}} = 3.17776$이면 $k_D \simeq 0.360 < 1$. 자명해 $x = 1$은 물리 가지 밖(접힘 없음, $d = 0$ 대응).

---

## Lambert W에 의한 닫힌 해

$x = \varepsilon^2$, $D = D_{\text{eff}}$로 놓으면:

$$x = e^{-D} \cdot e^{xD} \implies (-xD)e^{-xD} = -De^{-D}$$

Lambert W 함수 적용:

$$\boxed{\Omega_b = -\frac{W_0(-D_{\text{eff}}\,e^{-D_{\text{eff}}})}{D_{\text{eff}}} = 0.04865}$$

관측: $\Omega_b = 0.0486 \pm 0.0010$ (Planck). 텐션: $0.05\sigma$.

$W_0$ 주가지: 물리 가지 ($d = 3$). $W_{-1}$ 가지: 자명해 ($\varepsilon^2 = 1$, $d = 0$).

## $\Omega_\Lambda$, $\Omega_{DM}$ -- 암흑 에너지/물질 분할

억압된 에너지 $\Omega_\Phi = 1 - \varepsilon^2$가 두 성분으로 분리:

- **진공 에너지** (0차): $\Omega_\Lambda$
- **양자 요동 응축** (1-루프): $\Omega_{DM}$

연장성 원리(A1)에 의해 요동 대 진공의 비율:

$$\frac{\Omega_{DM}}{\Omega_\Lambda} = \alpha_s \cdot D_{\text{eff}}$$

각 유효 차원에서 QCD 결합 $\alpha_s$만큼의 요동이 기여하고, $D_{\text{eff}}$개 차원이 합산된다.

$$\Omega_\Lambda + \Omega_{DM} = 1 - \varepsilon^2$$

연립:

$$\boxed{\Omega_\Lambda = \frac{1 - \varepsilon^2}{1 + \alpha_s D_{\text{eff}}} = 0.6921} \quad \text{관측 } 0.6847 \quad (1.1\%)$$

$$\boxed{\Omega_{DM} = \frac{(1-\varepsilon^2)\alpha_s D_{\text{eff}}}{1 + \alpha_s D_{\text{eff}}} = 0.2593} \quad \text{관측 } 0.2589 \quad (0.13\%)$$

수치 분해: $\alpha_s D_{\text{eff}} = 0.11789 \times 3.17776 = 0.37465$.

유도 레벨: **가정**(진공/요동 분리, $\alpha_s$가 차원당 요동률) + **정리**(에너지 보존에 의한 비율 결정).

## 완전한 닫힌 형태

$D \equiv D_{\text{eff}}$로 모든 우주론 + 입자물리 관측량 표현:

$$\Omega_b = -\frac{W_0(-De^{-D})}{D}$$

$$\Omega_\Lambda = \frac{1 + W_0(-De^{-D})/D}{1 + D\left(\frac{1-\sqrt{1-4(D-3)}}{8}\right)^{3/4}}$$

$$\alpha_s = \left(\frac{1-\sqrt{1-4(D-3)}}{8}\right)^{3/4}, \qquad \sin^2\theta_W = \frac{1-\sqrt{1-4(D-3)}}{2}$$

**우주 전체가 $D_{\text{eff}}$라는 단일 숫자의 함수이다.**
