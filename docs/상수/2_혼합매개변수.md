# 2층: 혼합 매개변수

## 개관

1층의 격자 대각원소에서 파생되는 혼합 구조. EWSB(전자약 대칭 깨짐)가 여는 분수 차원을 정의한다. 이 층에서 확정되는 $\delta$와 $D_{\text{eff}}$가 3층 부트스트랩의 핵심 입력이다.

---

## $\sin^2\theta_W$ -- (2,2)-(3,3) 혼합각

### 물리적 의미

와인버그 각(Weinberg angle) $\theta_W$는 약한 힘과 전자기력이 얼마나 "섞여" 있는지를 나타낸다. 표준모형에서 약한 힘을 매개하는 $W^3$ 보손과 전자기력의 $B$ 보손이 혼합하여, 관측되는 $Z$ 보손과 광자 $\gamma$를 만든다.

$\sin^2\theta_W$가 크면 약한 힘과 전자기력의 혼합이 크고, 0이면 혼합 없음.

### 1층에서의 유도 (복습)

$$\sin^2\theta_W = 4\alpha_s^{4/3}$$

지수 $4/3 = 2N_w/N_c$: 격자의 (2,2) 행(SU(2), 차원 $N_w = 2$)과 (1,1) 행(SU(3), 차원 $N_c = 3$)의 비율이다.

$$\boxed{\sin^2\theta_W = 0.23122} \quad \text{관측 } 0.23122 \pm 0.00003 \quad (0.02\sigma)$$

---

## $\delta$ -- EWSB 유효 혼합 가중치

### 물리적 배경

전자약 대칭 깨짐(EWSB)은 히그스장이 진공 기대값 $v_{\text{EW}} = 246$ GeV를 가지면서 발생한다. 이 과정에서 $W^3$과 $B$가 혼합하여 질량을 가진 $Z$ 보손과 질량 없는 광자 $\gamma$를 만든다.

$Z$ 보손은 SU(2)와 U(1)의 혼합이므로, EWSB는 격자의 (2,2)와 (3,3) 행 사이에 "연결"을 만든다. 이 연결의 세기가 $\delta$이다.

### 정의: 질량 행렬에서

EWSB 후 중성 게이지 보손 질량 행렬 $(W^3, B)$ 기저:

$$M^2 = \frac{v^2}{4}\begin{pmatrix} g^2 & -gg' \\ -gg' & g'^2 \end{pmatrix}$$

여기서 $g$는 SU(2) 결합상수, $g'$은 U(1) 결합상수이다.

비대각 원소 $|M^2_{W^3B}| = gg'v^2/4$를 $M_Z^2 = (g^2+g'^2)v^2/4$로 정규화하면:

$$\frac{|M^2_{W^3 B}|}{M_Z^2} = \frac{gg'}{g^2+g'^2} = \sin\theta_W \cos\theta_W$$

### 유일성 증명 (정리 7.1)

중성 혼합 채널 가중치 $w_{\text{mix}}$가 "접힘 차원에 더해질 수 있는 양"으로서 만족해야 할 4가지 조건:

1. **무차원**: 차원 수($d = 3$)에 더해질 양이므로 무차원이어야 한다
2. **기저 회전 불변**: $(W^3, B)$의 직교변환에 대해 불변. 물리적 결과가 기저 선택에 의존하면 안 된다
3. **재위상/부호반전 불변**: $M^2_{W^3B} \to -M^2_{W^3B}$에서 불변. 혼합의 "세기"는 부호에 무관하다. 따라서 $w_{\text{mix}}$의 급수 전개에서 홀수차 항이 금지된다
4. **혼합 소거**: $M^2_{W^3B} = 0$이면 $w_{\text{mix}} = 0$. 혼합이 없으면 추가 차원도 없다

**증명:**

해석성(영점 근방 거듭제곱 급수 전개 가능)을 가정한다.

- 조건 3에 의해 급수는 $|M^2_{W^3B}|$의 짝수차($0, 2, 4, ...$)만 포함
- 조건 4에 의해 0차(상수항)는 0. 최소 차수는 2차
- 조건 1, 2에 의해 $|M^2_{W^3B}|^2$는 $(\text{Tr}\,M^2)^2 = M_Z^4$로 나누어야 무차원

따라서 최소 차수의 비자명 불변량은:

$$w_{\text{mix}} = \frac{|M^2_{W^3B}|^2}{(\text{Tr}\,M^2)^2} = \left(\frac{|M^2_{W^3B}|}{M_Z^2}\right)^2 = \sin^2\theta_W \cos^2\theta_W$$

뿐이다. 고차항은 $w^2, w^3, ...$이지만, "최소 차수"를 택하면(Occam) 유일하다. $\square$

### 수치

$$\boxed{\delta \equiv w_{\text{mix}} = \sin^2\theta_W(1 - \sin^2\theta_W) = 0.23122 \times 0.76878 = 0.17776}$$

$\delta$의 범위: $0 \leq \delta \leq 1/4$. $\delta = 1/4$이면 최대 혼합($\theta_W = \pi/4$). $\delta = 0$이면 혼합 없음. 실제 값 $0.178$은 최대값의 71%.

## $D_{\text{eff}}$ -- 유효 접힘 차원

### 물리적 의미

$D_{\text{eff}}$는 CE 경로적분에서 "유효적으로 몇 차원의 접힘이 일어나는가"를 나타낸다. 정수 공간 차원 $d = 3$에 EWSB에 의한 분수 보정 $\delta$가 더해진다.

### 구성

**(i) 공간 기여: $D_{\text{space}} = d = 3$**

자유 입자의 작용은 공간 3방향으로 등방적으로 분해된다:

$$S = \int dt \left(\frac{m}{2}\dot{x}^2 - V\right) = \int dt \left(\frac{m}{2}(\dot{x}_1^2 + \dot{x}_2^2 + \dot{x}_3^2) - V\right)$$

각 방향이 독립적으로 접힘에 기여하므로, 공간 기여는 $d = 3$.

**(ii) EWSB 기여: $\delta$**

$Z$ 보손이 SU(2)$\leftrightarrow$U(1) 혼합이라는 사실은, CE 접힘에서 "새로운 방향"을 연다. 정수 차원이 아니라 분수 차원 $\delta$만큼.

왜 $Z$만 기여하는가?
- 광자($M_\gamma = 0$): 질량이 없으므로 억압 스케일이 없다. 접힘에 기여하지 않는다
- $W^\pm$(하전 전류): 하전 보손은 $\{3, 2, 1\}$ 분할에 이미 포함된 SU(2) 기본 표현의 일부이다. 새 채널을 추가하지 않는다
- $Z$(중성 전류): SU(2)의 $W^3$과 U(1)의 $B$가 혼합한 것이므로, 대각 구조에 없던 **비대각 연결**을 제공한다. 이것만이 새 접힘 방향이다

**(iii) 결합: 덧셈의 물리적 정당성**

광명장의 헤시안(Hessian) $\langle\Phi\rangle$은 차원에 대해 선형이다(연장성 원리 A1). 이것은 $\Phi = \sigma \cdot D$의 선형 구조에서 오며, 두 독립적 기여(공간 + EWSB)는 합산된다:

$$\boxed{D_{\text{eff}} = d + \delta = 3 + \sin^2\theta_W\cos^2\theta_W = 3.17776}$$

### 경계 검증

- $\theta_W = 0$ (약한 혼합 없음): $\delta = 0$, $D_{\text{eff}} = 3$. 혼합 없으면 추가 차원 없다. 물리적으로 올바르다.
- $\theta_W = \pi/2$ (최대 혼합): $\delta = 0$, $D_{\text{eff}} = 3$. $\sin^2\theta_W = 1$이면 $B$ 보손이 완전히 약한 보손과 동일해져서 독립적 혼합 채널이 사라진다. 올바르다.
- $\theta_W = \pi/4$ (대칭점): $\delta = 1/4$, $D_{\text{eff}} = 3.25$. 최대 비대각 혼합.

유도 레벨: (i) **정리**, (ii) **정리**(SM 질량 행렬 + 불변량 최소차수 선택), (iii) **정리**(연장성 원리에 의한 선형성).

## 수치 체인 (1층 → 2층)

| 단계 | 입력 | 연산 | 출력 |
|---|---|---|---|
| 1 | $\alpha_s = 0.11789$ | $\alpha_s^{4/3} = 0.05781$ | |
| 2 | $\alpha_s^{4/3}$ | $\times 4$ | $\sin^2\theta_W = 0.23122$ |
| 3 | $\sin^2\theta_W$ | $\times(1 - \sin^2\theta_W)$ | $\delta = 0.17776$ |
| 4 | $\delta$ | $3 + \delta$ | $D_{\text{eff}} = 3.17776$ |

4단계. 모든 중간값이 5개 상수에서만 유도된다. 외부 측정값은 하나도 들어가지 않는다.

---

## 이 층에서 확정되는 것

2층의 출력(3층 이상의 입력):

- $\sin^2\theta_W = 0.23122$
- $\delta = 0.17776$
- $D_{\text{eff}} = 3.17776$

$D_{\text{eff}}$는 3층 부트스트랩 방정식의 핵심 매개변수이다. 이 단일 숫자가 우주의 에너지 구성 전체를 결정한다.

## 5개 상수 귀속

| 상수 | 역할 | 등장하는 식 |
|---|---|---|
| $\pi$ | $\alpha_s \to \sin^2\theta_W = 4\alpha_s^{4/3}$ | 지수 $4/3$의 근원 |
| $0$ | $d = 3$: $D_{\text{space}} = 3$ | 공간 기여 |
| $1$ | $D_{\text{eff}} = d + \delta$에서 $d$는 정수 | $\delta$는 분수 보정 |
