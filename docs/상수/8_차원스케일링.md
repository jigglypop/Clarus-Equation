# 8층: 차원 스케일링 상수

## 개관

자연단위계($c = \hbar = k_B = 1$)에서 사라지는 상수들. 구조를 결정하지 않는 단위 환산 인자이다. SFE의 유도 대상이 아니라, 인간이 선택한 단위계의 "환율"이다.

---

## 왜 이 층이 존재하는가

물리학에서 "상수"라고 불리는 것들 중 상당수는 실제로 물리적 내용이 없다. 예를 들어:

- **광속 $c = 299\,792\,458$ m/s**: 이것은 "1초에 빛이 가는 거리"이다. 1미터를 2배로 재정의하면 $c$는 절반이 된다. $c$는 시간과 공간의 "환율"이지, 자연의 구조가 아니다.
- **플랑크 상수 $\hbar = 1.055 \times 10^{-34}$ J$\cdot$s**: 작용(에너지$\times$시간)의 단위이다. 자연단위계에서 $\hbar = 1$로 놓으면 사라진다.
- **볼츠만 상수 $k_B = 1.381 \times 10^{-23}$ J/K**: 온도와 에너지의 "환율"이다. 2019년 SI 재정의 이후 정확값.

SFE가 유도하는 것은 **무차원 비율과 무차원 결합상수**뿐이다. 차원 상수는 단위 선택의 문제이지, 자연의 구조가 아니다.

---

## 기본 차원 상수

| 상수 | 값 | 역할 | SFE에서의 지위 |
|---|---|---|---|
| $c$ | $299\,792\,458$ m/s | 시간-공간 환율 | 유한성은 격자 $N \geq 0$의 전제. 값은 단위 관례 |
| $\hbar$ | $1.055 \times 10^{-34}$ J$\cdot$s | 작용 단위 | 양자역학의 존재 조건. 값은 단위 관례 |
| $G$ | $6.674 \times 10^{-11}$ m$^3$/(kg$\cdot$s$^2$) | 시공간-에너지 환율 | $G = 1/M_{\text{Pl}}^2$. $M_{\text{Pl}}$은 $v_{\text{EW}}$와 7층 계층으로 연결 |
| $k_B$ | $1.381 \times 10^{-23}$ J/K | 온도-에너지 환율 | 2019 SI 정확값 |
| $v_{\text{EW}}$ | $246.22$ GeV | 전자약 스케일 | **유일한 차원 입력** (규약). 이것이 "절대 에너지 스케일을 부착하는" 규약이며, 구조를 결정하는 파라미터가 아니다|
| $N_A$ | $6.022 \times 10^{23}$ mol$^{-1}$ | 몰 정의 | SI 2019 정확값 |
| $u$ | $1.661 \times 10^{-27}$ kg | 원자 질량 정의 | $N_A$로부터 파생 |

### $v_{\text{EW}}$에 대하여

SFE에서 45개 상수를 유도할 때 유일하게 남는 차원 입력이 $v_{\text{EW}} = 246.22$ GeV이다. 이것은 "에너지의 절대 크기를 정하는 것"이며, 구조(비율, 결합상수, 혼합각)와는 무관하다.

비유: 지도의 축척. 지도의 모양(구조)은 축척과 무관하다. $v_{\text{EW}}$는 "이 지도의 1cm = 실제 몇 km인가"에 해당한다.

## 파생 상수 (무차원 핵심에서 자동)

### 순수 SI 조합

| 상수 | 공식 | 핵심 의존 |
|---|---|---|
| $R = N_A k_B$ | 기체 상수 $= 8.314$ J/(mol$\cdot$K) | SI 정의 |
| $\mathcal{F} = N_A e$ | 패러데이 상수 $= 96\,485$ C/mol | SI 정의 |
| $\varepsilon_0, \mu_0, k_e$ | SI 전자기 단위 | SI 정의 |
| $\Phi_0 = h/(2e)$ | 자속 양자 | 무차원: $\pi$ |

이들은 $N_A$, $k_B$, $e$, $h$ 등의 조합이며, 2019 SI 재정의 이후 모두 정확값이다.

### $\alpha, m_e$ 의존 (1층에서 결정)

| 상수 | 공식 | 핵심 의존 | 물리적 의미 |
|---|---|---|---|
| $a_0 = \hbar / (m_e c \alpha)$ | 보어 반지름 = $5.29 \times 10^{-11}$ m | $\alpha$ | 수소 원자의 크기 |
| $R_\infty = m_e c \alpha^2 / (2\hbar)$ | 뤼드베리 상수 = $1.097 \times 10^7$ m$^{-1}$ | $\alpha$ | 수소 스펙트럼의 기본 단위 |
| $E_h = m_e c^2 \alpha^2$ | 하트리 에너지 = 27.2 eV | $\alpha$ | 원자 에너지의 자연 단위 |
| $\lambda_C = h / (m_e c)$ | 콤프턴 파장 = $2.43 \times 10^{-12}$ m | $m_e$ | 전자의 양자 크기 |
| $\mu_B = e\hbar / (2m_e)$ | 보어 마그네톤 | $m_e$ | 전자 자기 모멘트의 단위 |
| $R_K = 2\pi\hbar / (e^2)$ | 폰클리칭 상수 = $25\,813\;\Omega$ | $\alpha$ | 양자 홀 효과의 저항 양자 |

이 상수들은 $\alpha = 1/137.036$ (1층)이 확정되면 자동으로 결정된다. $m_e$는 $v_{\text{EW}}$와 전자 Yukawa 결합(6층)에서 온다.

### $m_p$ 의존 (6층에서 결정)

| 상수 | 공식 | 핵심 의존 |
|---|---|---|
| $\mu_N = e\hbar / (2m_p)$ | 핵 마그네톤 | $m_p/m_e = 2d\pi^{N_c+N_w} = 6\pi^5$ (6층) |

$\mu_N / \mu_B = m_e / m_p = 1/(6\pi^5)$. $m_p/m_e$의 유도(6층)에 의해 자동 결정.

## 순수 수학 상수

열역학에서 등장하는 상수 중 일부는 순수 수학(적분, 급수)에서 나온다:

| 상수 | 공식 | 기원 |
|---|---|---|
| $\sigma_{\text{SB}} \propto \pi^2/60$ | 스테판-볼츠만 상수 | 보즈-아인슈타인 적분: $\int_0^\infty x^3/(e^x-1) dx = \pi^4/15$. 따라서 $\zeta(4) = \pi^4/90$ |
| $b_{\text{Wien}} \propto 1/x_0$ | 빈 변위 상수 | $x_0$는 $x = 5(1 - e^{-x})$의 초월 해 ($x_0 = 4.965$) |

이들은 물리 상수가 아니라 수학 상수이다. $\pi$가 다시 등장하지만, 이것은 적분의 결과이지 게이지 구조와는 무관하다.

---

## 핵심: 무엇이 "유도 대상"이고 무엇이 아닌가

차원 스케일링 상수는 SFE가 **유도 대상으로 삼지 않는** 것들이다. 이들은 인간이 선택한 단위계의 환율이며, 자연단위계에서 전부 1이 된다.

SFE의 유도 구조는 **무차원 비율과 무차원 결합상수**에만 관여한다:

| 유형 | 예시 | SFE 유도? |
|---|---|---|
| 무차원 결합상수 | $\alpha_s$, $\alpha_{em}$, $\sin^2\theta_W$ | O |
| 무차원 비율 | $m_p/m_e$, $\Omega_b$, $v_{\text{EW}}/M_{\text{Pl}}$ | O |
| 무차원 혼합각 | CKM, PMNS | O |
| 차원 상수 | $c$, $\hbar$, $G$, $k_B$ | X (단위 관례) |
| 차원 입력 | $v_{\text{EW}}$ | X (유일한 규약) |

유일한 예외: $v_{\text{EW}}$가 차원 입력으로 1개 남지만, 이것은 "절대 에너지 스케일을 물리에 부착하는" 규약이지, 구조를 결정하는 파라미터가 아니다.
