# 8층: 차원 스케일링 상수

## 개관

자연단위계($c = \hbar = k_B = 1$)에서 사라지는 상수들. 구조를 결정하지 않는 단위 환산 인자.

---

## 기본 차원 상수

| 상수 | 역할 | 비고 |
|---|---|---|
| $c$ | 시간-공간 환율 | 유한성은 격자 $N \geq 0$의 전제 |
| $\hbar$ | 작용 단위 | |
| $G$ | 시공간-에너지 환율 | $M_{\text{Pl}}^{-2}$ |
| $k_B$ | 온도-에너지 환율 | |
| $v_{\text{EW}}$ | 전자약 스케일 | 유일한 차원 입력 (규약) |
| $N_A$ | 몰 정의 | SI 2019 정확값 |
| $u$ | 원자 질량 정의 | |

---

## 파생 상수 (무차원 핵심에서 자동)

### 순수 조합

| 상수 | 공식 | 핵심 의존 |
|---|---|---|
| $R = N_A k_B$ | 기체 상수 | SI 정의 |
| $\mathcal{F} = N_A e$ | 패러데이 상수 | SI 정의 |
| $\varepsilon_0, \mu_0, k_e$ | SI 전자기 단위 | SI 정의 |
| $\Phi_0 = h/(2e)$ | 자속 양자 | 무차원: $\pi$ |

### $\alpha, m_e$ 의존 (1층에서 결정)

| 상수 | 공식 | 핵심 의존 |
|---|---|---|
| $a_0 = \hbar / (m_e c \alpha)$ | 보어 반지름 | $\alpha$ |
| $R_\infty = m_e c \alpha^2 / (2\hbar)$ | 뤼드베리 상수 | $\alpha$ |
| $E_h = m_e c^2 \alpha^2$ | 하트리 에너지 | $\alpha$ |
| $\lambda_C = h / (m_e c)$ | 콤프턴 파장 | $m_e$ |
| $\mu_B = e\hbar / (2m_e)$ | 보어 마그네톤 | $m_e$ |
| $R_K = 2\pi / (\alpha e^2/\hbar)$ | 폰클리칭 상수 | $\alpha$ |

### $m_p$ 의존 (클리어)

| 상수 | 공식 | 핵심 의존 |
|---|---|---|
| $\mu_N = e\hbar / (2m_p)$ | 핵 마그네톤 | $m_p/m_e = 2d\pi^{N_c+N_w}$ (6층) |

$\mu_N / \mu_B = m_e / m_p = 1/(6\pi^5)$. $m_p/m_e$의 유도(6층)에 의해 자동 결정.

## 핵심

차원 스케일링 상수는 SFE가 **유도 대상으로 삼지 않는** 것들이다. 이들은 인간이 선택한 단위계의 환율이며, 자연단위계에서 전부 1이 된다. SFE의 유도 구조는 무차원 비율과 무차원 결합상수에만 관여한다.

유일한 예외: $v_{\text{EW}}$가 차원 입력으로 1개 남지만, 이것은 "절대 에너지 스케일을 물리에 부착하는" 규약이지, 구조를 결정하는 파라미터가 아니다.
