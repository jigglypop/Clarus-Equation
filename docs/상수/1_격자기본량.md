# 1층: 격자 기본량 -- 3x3 대각원소

## 개관

3x3+1 게이지-중력 전이 격자의 대각원소. 세 게이지 결합상수를 외부 입력 없이 결정한다.

---

## 출발점: 총 결합상수

게이지장의 위상 회전 $\psi \to e^{i\theta}\psi$에서, 완전한 한 주기는 $\Delta\phi = 2\pi$이다. SFE 접힘 생존율 $S(D) = e^{-D}$에서, 한 게이지 주기를 1 e-fold로 정의하면(규약 H2):

$$P_{\text{gauge}}(\Delta\phi = 2\pi) = e^{-\alpha_{\text{total}} \cdot 2\pi} = e^{-1}$$

$$\boxed{\alpha_{\text{total}} = \frac{1}{2\pi} = 0.15916}$$

유도 레벨: **정리**(원의 기하학 $\Delta\phi = 2\pi$) + **규약**(H2: 게이지 주기 = 1 e-fold).

---

## 연립계: 3방정식, 3미지수

$\alpha_s + \alpha_w + \alpha_{em} = \frac{1}{2\pi}$에서 세 결합상수를 분리하려면 두 개의 추가 관계식이 필요하다.

### 관계식 1: 차원확률 (SFE 통합 관계식)

$d = 3$ 공간의 $\{3, 2, 1\}$ 게이지 분할에서:

- SFE 경로적분은 $d$차원에서 곱적으로 분해(P1)
- 전자약 혼합은 이중선형(2-vertex) 게이지 과정(P2)
- SU(2) 기본 표현 차원 $N_w = 2$(P3)
- 색 전하 수 $N_c = d = 3$(P4)

이로부터:

$$\sin\theta_W = N_w \cdot (\alpha_s^{1/N_c})^{N_w} = 2\alpha_s^{2/3}$$

$$\boxed{\sin^2\theta_W = 4\alpha_s^{4/3}}$$

지수 $4/3 = 2N_w/N_c$는 차원의 비율이다. 계수 $4 = N_w^2$는 약력 차원의 제곱이다.

### 관계식 2: 전자약 구조

SM의 전자약 통일 조건 $e = g\sin\theta_W$에서:

$$\alpha_{em} = \alpha_w \cdot \sin^2\theta_W$$

### 연립과 풀이

$$\begin{cases}
\alpha_s + \alpha_w + \alpha_{em} = \dfrac{1}{2\pi} \\[6pt]
\sin^2\theta_W = 4\alpha_s^{4/3} \\[6pt]
\alpha_{em} = \alpha_w \cdot \sin^2\theta_W
\end{cases}$$

(2), (3)을 (1)에 대입:

$$\alpha_s + \alpha_w(1 + 4\alpha_s^{4/3}) = \frac{1}{2\pi}$$

$$\therefore \quad \alpha_w = \frac{1/(2\pi) - \alpha_s}{1 + 4\alpha_s^{4/3}}, \qquad \alpha_{em} = \alpha_w \cdot 4\alpha_s^{4/3}$$

$\sin^2\theta_W = 0.23122$ (관측)를 만족하는 해:

## 저에너지 미세구조상수: 직접 유도

고에너지($M_Z$) 유도와 별도로, 저에너지 $\alpha^{-1}(0)$에 대한 직접 공식이 존재한다:

$$\boxed{\alpha^{-1}(0) = 4\pi^3 + \pi^2 + \pi = \pi(4\pi^2 + \pi + 1) = 137.036}$$

관측: $\alpha^{-1}(0) = 137.036$ (0.002%).

계수 $\{4, 1, 1\}$의 격자 해석:

| 항 | 계수 | 위상공간 차원 | 격자 대응 |
|---|---|---|---|
| $4\pi^3$ | 4 | $\pi^3$ | $d = 3$: 3차원 위상공간 |
| $\pi^2$ | 1 | $\pi^2$ | $N_w = 2$: SU(2) 위상공간 |
| $\pi$ | 1 | $\pi^1$ | $1$: U(1) 위상공간 |

이 공식으로부터 QED running을 역으로 돌리면 $\alpha^{-1}(M_Z) = 127.95$를 정확히 재현한다.

## 비대각 원소 = 0

$$\mathbf{T} = \begin{pmatrix} \mathcal{T}_1(\text{SU}(3)) & 0 & 0 \\ 0 & \mathcal{T}_2(\text{SU}(2)) & 0 \\ 0 & 0 & \mathcal{T}_3(\text{U}(1)) \end{pmatrix}$$

게이지 섹터 간 트리 수준 혼합 없음. $\det(\mathbf{T}) = \prod_i \mathcal{T}_i$. 인과성의 근거.

## 5개 상수 귀속

| 상수 | 역할 |
|---|---|
| $\pi$ | $\alpha_{\text{total}} = 1/(2\pi)$: 결합의 척도 |
| $0$ | $d = 3 \to \{3, 2, 1\}$ 분할: 지수 $4/3 = 2N_w/N_c$ |
| $e$ | $P_{\text{gauge}} = e^{-1}$: 접힘 생존율 |
| $1$ | 규약 H2: 게이지 주기 = 1 e-fold |
| $i$ | 게이지 변환 $\psi \to e^{i\theta}\psi$: 위상 회전의 원천 |
