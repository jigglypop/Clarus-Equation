# 뇌 이론 제1장: 해부학 계층

> 역할: `05_실험근거.md`에서 쓰는 해부학-제어 계층을 정리한다.
>
> 핵심 앵커: $p_r$, $s_r$, $q_n$.
>
> 검증 기준: `06_검증기준.md`

---

## 1. 계층 개요

뇌 모델은 최소 네 층으로 나눈다.

| 층 | 포함 구조 | 주된 기능 | 주로 연결되는 항 | 현재 판정 |
|---|---|---|---|---|
| $L_1$ | 감각/운동/연합 피질, DMN, thalamo-cortical relay | 입력 라우팅, 과제 반응, intrinsic background | $x_a$, $x_b$ | bridge |
| $L_2$ | hippocampus, entorhinal cortex, amygdala, basal ganglia, salience hub | 기억 재활성화, 위험/가치 표지, switching | $x_a$, $x_s$ | bridge |
| $L_3$ | hypothalamus, brainstem, LC, raphe, NTS, DMV | 수면-각성, 자율신경 set-point, endocrine/autonomic control | $x_b$, $s_r$ | bridge |
| $L_4$ | autonomic output, endocrine, immune tone, metabolic reserve, vascular tone | 몸 전체 완충력과 장기별 stress coupling | $q_n$, $\ell_r$ | bridge |

중요한 점은 $L_3$과 $L_4$가 단순 배경이 아니라, 실제로 $x_b$와 복구 여력을 떠받치는 제어층이라는 것이다.

---

## 2. 정점 가족

지역 인덱스 $r$는 임의 voxel이 아니라, 아래 정점 가족 중 하나에 속한 node로 읽는다.

$$
V_{\text{brain}}
=
V_{\text{ctx}}
\sqcup
V_{\text{thal}}
\sqcup
V_{\text{hip}}
\sqcup
V_{\text{sal}}
\sqcup
V_{\text{hyp}}
\sqcup
V_{\text{stem}}
\sqcup
V_{\text{aut}}
$$

| 정점 집합 | 포함 구조 | 가장 먼저 봐야 하는 역할 |
|---|---|---|
| $V_{\mathrm{ctx}}$ | sensory cortex, motor cortex, association cortex, DMN | task recruitment, intrinsic background |
| $V_{\mathrm{thal}}$ | relay thalamus, intralaminar thalamus | gating, bandwidth reallocation |
| $V_{\mathrm{hip}}$ | hippocampus, entorhinal loop | replay, episodic indexing, memory binding |
| $V_{\mathrm{sal}}$ | amygdala, basal ganglia, salience hub | gain control, switching, value/risk tagging |
| $V_{\mathrm{hyp}}$ | hypothalamus, SCN, PVN, LH/VLPO 계열 | circadian, endocrine, metabolic set-point |
| $V_{\mathrm{stem}}$ | LC, raphe, NTS, DMV 등 brainstem nuclei | arousal tone, autonomic correction |
| $V_{\mathrm{aut}}$ | vagal/sympathetic output side | body-coupling output |

---

## 3. 지역별 해석 규칙

같은 $p_r=(x_{a,r},x_{s,r},x_{b,r})$라도 정점 가족에 따라 해석이 다르다.

| 정점 가족 | $x_a$가 크다는 뜻 | $x_s$가 크다는 뜻 | $x_b$가 크다는 뜻 |
|---|---|---|---|
| $V_{\mathrm{ctx}}$ | 과제 반응, sensory recruitment, network engagement | local maintenance/plasticity burden | resting/DMN background 유지 |
| $V_{\mathrm{thal}}$ | relay overload, excessive gating burden | relay reweighting burden | tonic relay baseline |
| $V_{\mathrm{hip}}$ | replay/encoding activation | plasticity + replay burden | memory baseline context |
| $V_{\mathrm{sal}}$ | gain up, switching demand, value tagging | gating/policy update burden | tonic salience baseline |
| $V_{\mathrm{hyp}},V_{\mathrm{stem}},V_{\mathrm{aut}}$ | emergency drive, arousal/autonomic overdrive | set-point correction burden | homeostatic reserve |

따라서 $r$는 단지 위치가 아니라, 어떤 제어층의 일부인가를 포함하는 해부학 태그다.

---

## 4. 최소 읽기 순서

1. `01_해부학계층.md`: 어떤 node family가 있는지 본다.
2. `02_관측정의.md`: 그 node에서 $p_r$, $s_r$를 어떻게 읽을지 본다.
3. `03_항상성제어축.md`: $q_n$과 $\ell_r$가 어떻게 들어오는지 본다.
4. `04_그래프결합과이완.md`: node 사이 spread/buffering을 어떻게 쓸지 본다.
5. `06_검증기준.md`: 각 정점 가족이 실제로 증명 가능한지 본다.

즉 해부학은 CE를 대체하는 층이 아니라, 기존 기호에 실제 기관을 붙여 주는 층이다.
