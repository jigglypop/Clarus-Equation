# 뇌 이론 제5장: 실험 근거

이 파일은 실험 근거 장의 짧은 입구다. 세부 내용은 `05_실험근거/` 아래에 나누어 둔다.

## 요약

이 장은 뇌 이론에서 실제로 검증 가능한 항목을 정리한다. 기준은 세 가지다.

```text
수식이 있는가
관측 proxy가 있는가
baseline보다 예측력이 좋은가
```

현재 실험 프로그램은 전역 뇌 방정식의 각 항을 한 번에 믿지 않는다. 과제, 항상성, 그래프, 시냅스, 느린 회복 항을 따로 검증한 뒤, 마지막에 full equation이 ablation을 이기는지 본다.
