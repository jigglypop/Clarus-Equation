# 뇌 이론 제5장: 실험 근거

이 장은 뇌 이론에서 실제로 검증 가능한 연구 항목을 모은다. 기준은 단순하다. 수식이 있고, 관측 proxy가 있으며, 개입으로 부호를 바꿀 수 있고, baseline보다 예측력이 좋아야 한다.

---

## 1. 판정 기준

| 등급 | 의미 |
|---|---|
| supported | 독립적인 신경과학 근거가 충분히 있고, 관측 방향이 안정적이다 |
| bridge | 생물학 근거는 있으나 CE 변수로 압축하는 검증이 더 필요하다 |
| testable | 현재 식과 실험 설계가 있어 바로 검증할 수 있다 |
| excluded | 현재 수식이나 실험 설계로 닫히지 않는다 |

---

## 2. 검증 가능한 핵심 주장

| 주장 | 유도식 | 관측 proxy | 개입 | 판정 |
|---|---|---|---|---|
| 지역 활성 부담은 $x_{a,r}$로 잡힌다 | $p_r=\operatorname{softmax}(A_r,S_r,B_r)$ | task fMRI, EEG/MEG evoked response | 과제 난이도 증가 | supported, testable |
| 과제 민감도는 역추정 가능하다 | $\widehat\beta_r=(\widehat A_r(u_1)-\widehat A_r(u_0))/(u_1-u_0)$ | 과제 부하별 $x_{a,r}$ | graded task load | testable |

| 배경 reserve 민감도는 역추정 가능하다 | $\widehat\chi_r=(\widehat B_r(u_0)-\widehat B_r(u_1))/(u_1-u_0)$ | resting burden별 $x_{b,r}$ | 수면박탈, resting instability | testable |

| 수면 민감도는 역추정 가능하다 | $\widehat\alpha_r=\Delta x_{a,r}/z_{\mathrm{sleep}}$ | 수면박탈 전후 $p_r$ | 수면박탈 | testable |
| 회복 수면은 잔차를 줄인다 | $\widehat\kappa=\lVert p_{\mathrm{recovery}}-p^*\rVert/\lVert p_{\mathrm{deprived}}-p^*\rVert$ | longitudinal EEG/fMRI/PVT | 회복 수면 | testable |
| 급성 PVT 회복은 CE 복구율과 맞는다 | $\rho_B=\rho_{\mathrm{night}}^{1.6}$ | PVT recovery curves | 회복 밤 수 | supported, testable |
| 만성 수면부채는 느린 회복 모드가 필요하다 | $e_n=e_n^{\mathrm{fast}}+e_n^{\mathrm{slow}}$ | long recovery PVT proxy | 반복 수면 제한 | bridge, testable |
| 그래프 결합은 예측력을 높인다 | $p_{n+1}=\Pi_{\Delta^2}((1-\rho)p^*+\rho p_n+\gamma\Delta_Gp_n+H(q-q^*))$ | structural or functional connectivity | graph model과 flat model 비교 | bridge, testable |
| 그래프 항은 flat baseline을 이겨야 한다 | $\mathcal L_{\mathrm{graph}}<\mathcal L_{\mathrm{flat}}$ | holdout regional state | connectivity perturbation | testable |
| 그래프 결합은 수면 민감도 차이를 평탄화한다 | $e^*=((1-\rho_B)I+\gamma L_G)^{-1}z_{\mathrm{sleep}}\alpha$ | 회로별 $\Delta x_a$ 지도 | 수면박탈 | testable |
| 시간축 회복은 $\rho_B$와 $\gamma$를 분리한다 | $\mu_k=\rho_B-\gamma\lambda_k$ | 회복 밤별 회로 잔차 지도 | 회복 수면 반복 측정 | testable |
| STDP는 시냅스 방향을 정한다 | $\Delta w_{ij}=\eta(A_+p_is_j-A_-s_iq_j)$ | paired spike plasticity | spike timing 조작 | supported |

| 미시 가소성은 거시 구조 좌표로 투영된다 | $p_r=\operatorname{softmax}(A_r,S_r,B_r)$ | 지역별 plasticity load, PET, glial marker | 학습 또는 자극 protocol | testable |
| 전체 상태 모델은 baseline보다 예측력이 좋아야 한다 | $\mathcal L_{\mathrm{state}}<\min(\mathcal L_{\mathrm{mean}},\mathcal L_{\mathrm{flat}},\mathcal L_{\mathrm{single}})$ | holdout $p_r$ | subject, session, task split | testable |
| 전역 뇌 방정식은 ablation을 이겨야 한다 | $\mathcal L_{\mathrm{full}}<\mathcal L_{\mathrm{best\;ablation}}$ | holdout $P_{n+1}$ | subject, session, task split | testable |

## 3. 전역 방정식 연구 프로토콜

목표는 모든 뇌 부위의 상태를 하나의 전역 방정식으로 찾는 것이다.

$$
P_{n+1}
=
\Pi_{(\Delta^2)^{|V_{\mathrm{brain}}|}}
\left[
(1-\rho_B)P^*
+\rho_B P_n
+\gamma\Delta_G P_n
+U_{\mathrm{task},n}
+H(Q_n-Q^*)
+F_{\mathrm{syn}}(E_n,M_n,W_n)
+F_{\mathrm{slow}}(D_{\mathrm{sleep},n})
+\Xi_n
\right]
$$

연구는 다음 순서로 진행한다.

| 단계 | 찾는 대상 | 필요한 자료 | 성공 기준 |
|---|---|---|---|
| 1 | 관측 사상 $\mathcal O:Y\to P$ | task, rest, EEG, metabolic proxy | $P_n$이 반복 측정에서 안정 |
| 2 | 과제 forcing $U_{\mathrm{task}}$ | graded task load | $\widehat\beta_r$ 순위가 task family 안에서 재현 |
| 3 | 항상성 투사 $H$ | sleep, arousal, metabolic intervention | 부호 제약과 subject split 재현 |
| 4 | 그래프 결합 $\gamma\Delta_G$ | structural 또는 functional connectivity | 실제 graph가 flat/shuffled graph보다 낮은 손실 |
| 5 | 시냅스 항 $F_{\mathrm{syn}}$ | plasticity, glial, metabolic proxy | $x_s$ 순위와 proxy 순위가 일치 |
| 6 | 느린 회복 항 $F_{\mathrm{slow}}$ | 반복 수면 제한과 장기 회복 | fast-only 모델의 잔차 floor를 줄임 |
| 7 | full equation | 위 자료의 holdout split | ablation 중 가장 좋은 모델보다 낮은 손실 |

최종 비교는 full equation과 항 제거 모델의 ablation으로 한다.

$$
\mathcal L_{\mathrm{full}}
<
\min
\left(
\mathcal L_{\mathrm{no\;task}},
\mathcal L_{\mathrm{no\;homeostasis}},
\mathcal L_{\mathrm{no\;graph}},
\mathcal L_{\mathrm{no\;syn}},
\mathcal L_{\mathrm{no\;slow}},
\mathcal L_{\mathrm{inertial}}
\right)
$$

이 부등식이 subject split, session split, task split에서 모두 유지될 때만 전역 방정식 후보를 결론으로 올린다.

`examples/physics/brain_equation_integrated_gate.py`의 synthetic pilot에서는 문서의 전역식 항을 모두 넣고 다음 ablation을 비교한다.

| 모델 | holdout loss |
|---|---:|
| full | $0.00000980$ |
| no task | $0.00030620$ |
| no graph | $0.00005922$ |
| no homeostasis | $0.06531250$ |
| no synaptic | $0.00031443$ |
| no slow recovery | $0.00466675$ |
| inertial | $0.11573503$ |

가장 강한 ablation은 no graph이고, full equation의 상대 손실은 다음이다.

$$
\frac{
\mathcal L_{\mathrm{full}}
}{
\mathcal L_{\mathrm{best\;ablation}}
}
=0.165477
$$

따라서 코드 gate에서는 $U_{\mathrm{task}}$, $H(Q-Q^*)$, $\Delta_G$, $F_{\mathrm{syn}}$, $F_{\mathrm{slow}}$가 모두 들어간 식이 항 제거 모델보다 낮은 손실을 낸다. 단, 이 결과는 synthetic holdout이므로 실제 결론은 위 split 규칙을 통과한 공개 또는 실험 데이터에서만 올린다.

같은 스크립트의 split inverse gate는 synthetic 전이를 subject, session, task로 나눈 뒤, 보지 않은 split에서 full equation과 ablation을 다시 비교한다.

| holdout split | holdout loss | full/best ablation | max coefficient error | 통과 |
|---|---:|---:|---:|---|
| subject = s3 | $0.00000125$ | $0.006867$ | $0.002005$ | yes |
| session = recovery | $0.00000043$ | $0.002437$ | $0.004964$ | yes |
| task = task_c | $0.00000152$ | $0.008687$ | $0.003923$ | yes |

이 gate는 “전역 계수가 split 밖에서도 회수될 수 있는 형식”을 확인하는 단계다. 실제 데이터에서 이 표가 깨지면 전역 방정식은 결론이 아니라 synthetic 구조 검증으로만 남긴다.

CE-뇌 bridge gate는 보편 안정 작용의 문법이 뇌 전역식에서도 유지되는지 보는 보조 지표다.

| 지표 | 값 | 의미 |
|---|---:|---|
| modal radius | $0.197441$ | $\max_k|\rho_B-\gamma\lambda_k(L_G)|$ |
| stability margin | $0.802559$ | $1-\mathrm{modal\;radius}$ |
| coupling energy | $0.00413704$ | $\|\Delta_G P\|^2$ |
| forcing energy | $0.11533617$ | $\|U_{\mathrm{task}}+H+F_{\mathrm{syn}}+F_{\mathrm{slow}}\|^2$ |
| ablation gain | $0.834523$ | $1-\mathcal L_{\mathrm{full}}/\mathcal L_{\mathrm{best\;ablation}}$ |

이 표는 CE의 안정 functional 문법과 뇌 전역식이 같은 구조를 공유한다는 긍정 신호다. 그러나 우주론 계수와 뇌 계수가 수치적으로 같다는 뜻은 아니다.

## 4. 최소 데이터셋 설계

전역 방정식을 실제로 찾으려면 단일 실험 하나가 아니라 같은 피험자의 반복 측정이 필요하다. 최소 데이터셋은 다음 구조를 가져야 한다.

| 축 | 최소 조건 | 이유 |
|---|---|---|
| subject | 여러 subject | subject split 검증 |
| session | baseline, deprivation, recovery 1, recovery 2 이상 | $\rho_B$와 $\gamma$ 분리 |
| condition | rest, graded task, sleep deprivation, recovery | $x_a$, $x_b$, $q_{\mathrm{sleep}}$ 동시 검증 |
| region | $V_{\mathrm{brain}}$의 node family 전체 | 전역 상태 $P_n$ 구성 |
| graph | subject별 connectivity 또는 공통 template | $\Delta_G$ 항 검증 |
| proxy | EEG/fMRI/PVT/metabolic 중 둘 이상 | 단일 proxy 과적합 방지 |
| plasticity | 학습 또는 자극 전후 proxy | $F_{\mathrm{syn}}$ 검증 |

한 행의 전이 데이터는 다음처럼 읽는다.

$$
\left(
P_n,
Q_n,
G_{\mathrm{brain}},
E_n,
M_n,
W_n,
D_{\mathrm{sleep},n}
\right)
\mapsto
P_{n+1}
$$

가장 작은 실험 블록은 다음이다.

$$
\mathrm{baseline}
\rightarrow
\mathrm{task}
\rightarrow
\mathrm{sleep\;deprivation}
\rightarrow
\mathrm{recovery\;1}
\rightarrow
\mathrm{recovery\;2}
$$

각 단계에서 resting proxy와 task proxy를 모두 찍으면, $U_{\mathrm{task}}$, $H(Q-Q^*)$, $\Delta_G$, $B$, $F_{\mathrm{slow}}$를 같은 subject 안에서 분리할 수 있다. $F_{\mathrm{syn}}$는 같은 블록 안에 학습 과제나 stimulation 전후 측정을 추가해야 한다.

split은 세 가지로 고정한다.

| split | 학습 | 검증 |
|---|---|---|
| subject split | 일부 subject 전체 | 보지 않은 subject |
| session split | 일부 회복/반복 session | 보지 않은 session |
| task split | 일부 과제군 | 보지 않은 과제군 |

전역 방정식은 세 split에서 모두 ablation을 이겨야 한다.

실제 전이 JSON이 준비되면 다음 명령으로 synthetic gate와 실제 데이터 gate를 함께 실행한다.

```bash
PYTHONIOENCODING=utf-8 ./.venv/Scripts/python.exe examples/physics/brain_equation_integrated_gate.py --real-transitions path/to/transitions.json
```

데이터 파일을 만들기 전에는 코드 기준 입력 사양을 먼저 출력한다.

```bash
PYTHONIOENCODING=utf-8 ./.venv/Scripts/python.exe examples/physics/brain_equation_integrated_gate.py --print-real-schema
```

초기 JSON 골격은 다음 명령으로 만든다.

```bash
PYTHONIOENCODING=utf-8 ./.venv/Scripts/python.exe examples/physics/brain_equation_integrated_gate.py --print-real-template
```

데이터 파일을 만든 뒤에는 본 실행 전에 점검 모드로 먼저 본다.

```bash
PYTHONIOENCODING=utf-8 ./.venv/Scripts/python.exe examples/physics/brain_equation_integrated_gate.py --check-real-transitions path/to/transitions.json
```

입력 파일은 JSON list이거나 `{ "region_order": [...], "transitions": [...] }` 형태다. `region_order`를 넣으면 canonical 순서와 정확히 일치해야 한다. 각 전이는 `current_p`, `observed_next_p`, `subject`, `session`, `task`를 포함해야 한다. 입력 항은 원시 control 또는 전처리 scale 중 하나를 쓴다.

| 항 | 원시 control | 전처리 scale |
|---|---|---|
| 과제 | `task_load` | `task_input` |
| 항상성 | `q_delta` | `homeostasis_input` |
| 시냅스 | `plasticity_input` | `syn_input` |
| 느린 회복 | `d_sleep` | `slow_input` |
| 그래프 | `graph_edges` 또는 `edges` | 생략 시 canonical graph |

split 값이 둘 이상 있는 축만 holdout 평가를 실행하고, 가능한 모든 split에서 full equation이 best ablation보다 낮은 손실을 내야 실제 데이터 gate를 통과한다.

실제 graph가 들어오면 결과 JSON의 `real_data_gate.graph_diagnostics`에 전이별 modal radius, stability margin, coupling energy가 기록된다. 모든 graph가 안정성 gate를 통과해야 `passed_real_gate`가 참이 된다.

실제 데이터 gate의 ablation에는 `shuffled_graph`와 `degree_preserving_graph`도 포함한다. 즉 입력 graph는 graph가 없는 모델, region label을 순환 치환한 shuffled graph, node별 edge 개수를 보존한 rewired graph보다도 낮은 holdout 손실을 내야 한다.

graph gate가 실패하면 `top_regions`와 `top_edges`를 먼저 본다. 이 값은 결론용 지표가 아니라, 어떤 region 관측이나 edge connectivity가 전역식을 깨뜨렸는지 추적하는 디버깅 지표다.

또한 실제 split의 train design matrix가 여섯 계수 $\rho_B,\gamma,$ task, homeostasis, syn, slow scale을 모두 분리할 수 있어야 한다. rank가 $6$보다 작거나 condition number가 너무 크면, holdout 손실이 낮아도 계수 역추정 결론으로 올리지 않는다.

실제 데이터 파일을 넣으면 결과 JSON의 `real_data_gate.readiness`를 먼저 본다. 이 블록은 `case_count`, subject/session/task별 전이 수, 평가 가능한 split 수, 식별성 실패, 예측 실패, graph 안정성 실패를 요약한다. `ready_for_claim`이 거짓이면 전역 방정식 결론이 아니라 데이터 보강 또는 원인 추적 단계로 남긴다. 실패가 있으면 `next_actions`가 split 추가, 독립 control 변화 추가, holdout 사례 점검, graph edge 점검 중 무엇을 먼저 해야 하는지 제시한다.

검증 장치 확장은 여기서 멈추고, 식의 표현력은 region-family 계수식으로 한 단계 올린 뒤 다시 압축한다. `family_inverse`는 42개 자유계수를 상한선으로 두고, 최종 후보인 anatomical generator는 base 여섯 개와 slope 여섯 개, 총 12개 계수만 사용한다. 현재 synthetic 결과는 subject, session, task split 모두에서 12개 generator가 global scalar 여섯 계수보다 낮은 holdout 손실을 냈고, train design rank도 $12/12$를 채웠다.

## 5. 먼저 실행할 실험

### 5.1 과제 활성 실험

과제 부하 $u_{\mathrm{task}}$는 $x_{a,r}$ 정의를 가장 직접적으로 검증한다.

$$
A_r(u)
=
A_{r,0}
+\beta_r u_{\mathrm{task}}
$$

$$
p_r(u)
=
\operatorname{softmax}
\begin{pmatrix}
A_{r,0}+\beta_r u_{\mathrm{task}}\\
S_r\\
B_r
\end{pmatrix}
$$

관측된 $x_{a,r}$에서 활성 점수는 다음처럼 역산한다.

$$
\widehat A_r
=
\log
\frac{x_{a,r}}{1-x_{a,r}}
+\log
\left(
\exp(S_r)+\exp(B_r)
\right)
$$

따라서 두 부하 조건에서 지역 과제 민감도는 다음이다.

$$
\widehat\beta_r
=
\frac{
\widehat A_r(u_1)-\widehat A_r(u_0)
}{u_1-u_0}
$$

`examples/physics/task_active_state.py`의 pilot 결과는 다음이다.

| 지역 | $\Delta x_a$ | $\Delta x_b$ | $\widehat\beta_r$ |
|---|---:|---:|---:|
| visual | $+0.198744$ | $-0.125106$ | $0.420000$ |
| motor | $+0.158631$ | $-0.098368$ | $0.340000$ |
| prefrontal | $+0.118829$ | $-0.072844$ | $0.260000$ |
| hippocampus | $+0.080173$ | $-0.048575$ | $0.180000$ |

성공 조건은 과제 부하가 커질수록 $x_{a,r}$가 증가하고, 같은 과제군에서 $\widehat\beta_r$ 순위가 subject split에서도 유지되는 것이다. 실패 조건은 과제 부하를 올려도 $x_{a,r}$가 오르지 않거나, 단일 EEG/fMRI 지표보다 $p_r$ 상태 모델의 holdout 예측력이 낮은 경우다.

### 5.2 배경 reserve 실험

resting baseline 부담을 $u_{\mathrm{rest}}$로 두면 배경 점수는 다음처럼 둔다.

$$
B_r(u)
=
B_{r,0}
-\chi_r u_{\mathrm{rest}}
$$

$$
p_r(u)
=
\operatorname{softmax}
\begin{pmatrix}
A_r\\
S_r\\
B_{r,0}-\chi_ru_{\mathrm{rest}}
\end{pmatrix}
$$

관측된 $x_{b,r}$에서 배경 점수는 다음처럼 역산한다.

$$
\widehat B_r
=
\log
\frac{x_{b,r}}{1-x_{b,r}}
+\log
\left(
\exp(A_r)+\exp(S_r)
\right)
$$

따라서 두 부담 조건에서 지역 reserve 민감도는 다음이다.

$$
\widehat\chi_r
=
\frac{
\widehat B_r(u_0)-\widehat B_r(u_1)
}{u_1-u_0}
$$

`examples/physics/background_reserve_state.py`의 pilot 결과는 다음이다.

| 지역 | $\Delta x_b$ | $\Delta x_a$ | $\widehat\chi_r$ |
|---|---:|---:|---:|
| default mode | $-0.170165$ | $+0.090181$ | $0.360000$ |
| thalamus | $-0.142615$ | $+0.076646$ | $0.300000$ |
| brainstem | $-0.115504$ | $+0.060349$ | $0.240000$ |
| prefrontal | $-0.096088$ | $+0.051401$ | $0.200000$ |

성공 조건은 $u_{\mathrm{rest}}$가 커질수록 $x_{b,r}$가 하락하고, resting fMRI 안정성, posterior alpha reactivity, PET baseline metabolism이 같은 방향으로 움직이는 것이다. 실패 조건은 배경 proxy가 흔들려도 $x_b$가 하락하지 않거나, $\widehat\chi_r$가 반복 측정에서 불안정한 경우다.

### 5.3 수면박탈 실험

가장 먼저 닫을 수 있는 검증은 수면박탈이다.

$$
z_n(t)=\left(q_{\mathrm{sleep}}(t)-q_{\mathrm{sleep}}^*\right)_+
$$

검증할 부호는 다음이다.

$$
\frac{\partial x_{a,r}}{\partial z_n}>0
$$

$$
\frac{\partial x_{b,r}}{\partial z_n}<0
$$

$$
\frac{\partial \mathrm{RT}_{\mathrm{pvt}}}{\partial z_n}>0
$$

실패 조건은 세 부호 중 둘 이상이 반복 측정에서 깨지는 경우다.

ds004902 pilot summary에서는 세 proxy가 모두 부담 방향으로 움직인다.

$$
z_\alpha=0.4115,\qquad
z_{\mathrm{PVT}}=0.1219,\qquad
z_{\mathrm{SSS}}=0.5000
$$

복합 부담 좌표는 다음이다.

$$
z_{\mathrm{mean}}=0.3445,\qquad
z_{\mathrm{geom}}=0.2927
$$

따라서 현재 관측으로는 $q_{\mathrm{sleep}}$의 부호가 닫힌다. 다음 연구 단계는 이 복합 좌표가 실제 지역 상태 $p_r$의 $x_a$ 증가와 $x_b$ 감소를 얼마나 설명하는지 보는 것이다.

최소 상태 전이 pilot에서는

$$
p_r'
=
\Pi_{\Delta^2}
\left(
p^*
+z_{\mathrm{sleep}}
\begin{pmatrix}
\alpha_r\\
0\\
-\alpha_r
\end{pmatrix}
\right)
$$

를 쓴다. $z_{\mathrm{mean}}=0.3445$일 때 $\alpha_r=0.10$이면,

$$
\Delta x_a=+0.0344,\qquad
\Delta x_b=-0.0344
$$

이다. $z_{\mathrm{geom}}=0.2927$을 쓰면,

$$
\Delta x_a=+0.0293,\qquad
\Delta x_b=-0.0293
$$

이다. 따라서 수면박탈 조건에서 $x_a$와 $x_b$가 이 정도 방향으로 움직이지 않으면, 현재 forcing 식은 약하거나 틀린 것이다.

이제 관측값이 들어오면 민감도는 곧바로 역추정한다.

$$
\widehat\alpha_r=
\frac{\Delta x_{a,r}}{z_{\mathrm{sleep}}}
$$

작은 상태 이동 $0.01$만 관측되어도 필요한 민감도는 다음 정도다.

$$
\widehat\alpha_r\approx0.029
\quad
\mathrm{to}
\quad
0.034
$$

상태 이동 $0.03$이 관측되면,

$$
\widehat\alpha_r\approx0.087
\quad
\mathrm{to}
\quad
0.102
$$

따라서 다음 실험의 핵심 산출물은 수면박탈 효과의 유무가 아니라, 회로별 $\widehat\alpha_r$ 지도다.

다중 항상성 축을 함께 보려면 개입 조건을 쌓아 $d_r$를 역추정한다.

$$
Q
=
\begin{pmatrix}
(q_n^{(1)}-q^*)_+^\top\\
(q_n^{(2)}-q^*)_+^\top\\
\cdots\\
(q_n^{(m)}-q^*)_+^\top
\end{pmatrix}
$$

$$
\widehat d_r
=
\arg\min_{d_r\ge0}
\left\|
Qd_r-y_r
\right\|^2
$$

`examples/physics/homeostatic_sensitivity_map.py`의 pilot 결과는 다음이다.

| 지역 | $\widehat d_{\mathrm{sleep}}$ | $\widehat d_{\mathrm{arousal}}$ | $\widehat d_{\mathrm{metabolic}}$ |
|---|---:|---:|---:|
| cortex | $0.1040$ | $0.1497$ | $0.0766$ |
| hippocampus | $0.1273$ | $0.1193$ | $0.0875$ |
| hypothalamus | $0.1865$ | $0.1390$ | $0.0965$ |
| brainstem | $0.1369$ | $0.1596$ | $0.1051$ |
| default mode | $0.1127$ | $0.0773$ | $0.1324$ |

이 pilot은 수면 축 하나로 모든 몸 상태를 설명하려는 시도가 틀렸다는 점을 보여 준다. 수면, 각성, 대사 부담의 지역 순위가 다르므로, 실제 실험도 다중 개입 설계로 가야 한다.

### 5.4 회복 수면 실험

회복 수면은 잔차 감쇠로 검증한다.

$$
\widehat \kappa
=
\frac{\lVert p(t_{\mathrm{recovery}})-p^*\rVert}
{\lVert p(t_{\mathrm{deprived}})-p^*\rVert}
$$

성공 조건은 다음이다.

$$
0\le \widehat\kappa<1
$$

그리고 회복 모델이 baseline보다 좋아야 한다.

$$
\mathcal L_{\mathrm{sleep}}<\mathcal L_{\mathrm{baseline}}
$$

현재 문헌 PVT 회복 곡선을 맞추면 급성 회복 자료는 하룻밤 잔차율이 다음 구간에 모인다.

$$
\rho_{\mathrm{night}}^{\mathrm{acute}}
\in
[0.305,0.318]
$$

$B$의 한 번 적용을 $1.6$밤으로 환산하면,

$$
\rho_B
=
\left(\rho_{\mathrm{night}}^{\mathrm{acute}}\right)^{1.6}
\in
[0.149,0.160]
$$

이 구간은 CE 중심값 $0.155$를 포함한다. 따라서 수면 연구의 첫 결론은 $B$의 복구율이 급성 PVT 회복 자료와 정합한다는 것이다. 단, 장기 누적 수면 부채 proxy에서는 $\rho_{\mathrm{night}}\approx0.61$이 나와 느린 회복 모드를 별도로 두어야 한다.

급성 회복과 만성 회복을 분리하면,

$$
\rho_{\mathrm{night}}^{\mathrm{fast}}
=0.3119\pm0.0055
$$

$$
\rho_B^{\mathrm{fast}}
\in
[0.1496,0.1602]
$$

이고 만성 proxy는

$$
\rho_{\mathrm{night}}^{\mathrm{slow}}
=0.6121
$$

이다. 세 밤 뒤 잔차는 fast mode에서 약 $0.0304$, slow mode에서 약 $0.2293$이다. 따라서 만성 수면부채는 회복 수면 횟수만으로 닫히지 않고 누적 입력 $u_{\mathrm{debt}}$를 포함해야 한다.

또 하나의 느린 모드 표지는 잔류 바닥값이다.

$$
y(n)=y^*+F+A\rho_{\mathrm{night}}^n
$$

$$
\phi_{\mathrm{floor}}=\frac{F}{A+F}
$$

Belenky 3시간 제한에서는 $\phi_{\mathrm{floor}}=0.483$으로 초기 결손의 거의 절반이 빠른 회복 채널로 제거되지 않는다. Banks 2010에서는 $\phi_{\mathrm{floor}}=0.050$으로 대부분 빠른 회복 채널에 실린다. 따라서 제한 강도와 반복 기간에 따라 fast mode의 진폭 $A$와 slow floor $F$가 달라진다.

누적 수면손실을 $D_{\mathrm{sleep}}=(8-T_{\mathrm{IB}})_+N_{\mathrm{days}}$로 두면, PVT lapse 계열 pilot에서는 다음 최소식이 나온다.

$$
\phi_{\mathrm{floor}}
\approx
-0.4162
+0.02579D_{\mathrm{sleep}}
$$

$$
R^2=0.9471
$$

이 식은 최종 dose-response가 아니라 작은 문헌 요약 pilot이다. 그래도 연구 방향은 분명하다. 누적 수면손실이 약 $20$시간을 넘으면 floor가 보이기 시작하고, 약 $35.5$시간에서는 초기 결손의 절반 정도가 빠른 복구 채널 밖에 남는다.

### 5.5 그래프 결합 실험

그래프 결합은 flat model과 비교한다.

$$
\mathcal L_{\mathrm{graph}}
=
\sum_{r,t}
\left\|
p_{r,t+1}
-
\widehat p_{r,t+1}^{\mathrm{graph}}
\right\|^2
$$

$$
\mathcal L_{\mathrm{flat}}
=
\sum_{r,t}
\left\|
p_{r,t+1}
-
\widehat p_{r,t+1}^{\mathrm{flat}}
\right\|^2
$$

성공 조건은 다음이다.

$$
\mathcal L_{\mathrm{graph}}<\mathcal L_{\mathrm{flat}}
$$

`examples/physics/graph_vs_flat_prediction.py`의 pilot에서는 다음 값이 나온다.

$$
\mathcal L_{\mathrm{graph}}=0.00000623
$$

$$
\mathcal L_{\mathrm{flat}}=0.00007377
$$

$$
\frac{\mathcal L_{\mathrm{graph}}}{\mathcal L_{\mathrm{flat}}}
=0.084453
$$

따라서 pilot에서는 graph term이 flat baseline보다 약 $91.6\%$ 낮은 손실을 낸다. 실제 연구에서는 structural connectivity, functional connectivity, shuffled graph를 모두 비교해서 진짜 연결 구조가 맞는지 확인해야 한다.

수면박탈 전파 pilot에서는 다음 고정점 식을 쓴다.

$$
e^*
=
\left((1-\rho_B)I+\gamma L_G\right)^{-1}
z_{\mathrm{sleep}}\alpha
$$

canonical graph의 $\lambda_{\max}$는 다음이다.

$$
\lambda_{\max}=2.9370
$$

따라서 안정한 결합 상한은 다음이다.

$$
\gamma<0.3933
$$

$\gamma$가 $0$에서 $0.30$으로 커지면 최대/최소 지역 반응 비율은 $2.50$에서 $1.88$로 줄어든다. 따라서 실제 수면박탈 자료에서 회로별 $\Delta x_a$ 지도가 얼마나 평탄한지를 보면 $\gamma$를 역추정할 수 있다.

관측 반응 지도 $e^{\mathrm{obs}}$가 있으면 각 $\gamma$ 후보에서 민감도 지도는 다음처럼 역산한다.

$$
\widehat\alpha(\gamma)
=
\frac{
\left((1-\rho_B)I+\gamma L_G\right)e^{\mathrm{obs}}
}{z_{\mathrm{sleep}}}
$$

현재 synthetic inverse pilot에서는 안정성과 민감도 상한을 만족하는 $\gamma$ 구간이 $0$부터 $0.38$까지 남는다. 해부학 prior를 함께 쓰면 생성값 $\gamma=0.20$과 민감도 prior를 정확히 복원한다. 실제 자료에서는 단일 지도만으로 부족하므로, 여러 수면박탈 강도와 회복 시점에서 같은 $\gamma$가 반복되는지 확인해야 한다.

시간축 회복 자료가 있으면 이 비유일성이 크게 줄어든다. 회복 구간의 잔차를 그래프 고유모드로 투영하면,

$$
c_{k,n+1}=\mu_kc_{k,n}
$$

$$
\mu_k=\rho_B-\gamma\lambda_k
$$

가 된다. 따라서 두 개 이상의 비영 고유모드에서 감쇠율을 얻으면 다음 역추정이 가능하다.

$$
\widehat\gamma
=
\frac{\widehat\mu_i-\widehat\mu_j}
{\lambda_j-\lambda_i}
$$

$$
\widehat\rho_B
=
\widehat\mu_i+\widehat\gamma\lambda_i
$$

`examples/physics/sleep_temporal_modes.py`의 pilot에서는 생성값 $\rho_B=0.155$, $\gamma=0.030$을 넣고 회복 모드 감쇠율을 만든 뒤 다시 역추적했다. 1% 모드 감쇠 노이즈를 넣은 결과는 다음이다.

$$
\widehat\rho_B=0.1550\pm0.0016
$$

$$
\widehat\gamma=0.0300\pm0.0006
$$

따라서 실제 실험의 우선순위는 회로별 $\Delta x_a$ 단일 지도에서 끝나지 않는다. 회복 1밤, 2밤, 3밤의 잔차 지도를 모아 $\mu_k$의 직선 관계를 검증해야 한다.

### 5.6 시냅스 가소성 실험

미시 가소성은 이미 강한 실험 전통이 있다. 이 문서에서 새로 검증할 것은 거시 변수 $x_{s,r}$와의 연결이다.

$$
x_{s,r}(t)
\propto
\sum_{i,j\in r}
|\Delta w_{ij}(t)|
+\lambda_{\mathrm{struct}}
\sum_{i,j\in r}
|w_{ij}(t)|
$$

이를 실제 관측 정의로 올리려면 먼저 지역 구조 점수를 계산한다.

$$
S_r(t)
:=
\sum_{i,j\in r}
\left|
\Delta w_{ij}(t)
\right|
+\lambda_{\mathrm{struct}}
\sum_{i,j\in r}
\left|
w_{ij}(t)
\right|
$$

그 다음 다음 사상으로 $x_{s,r}$를 얻는다.

$$
p_r(t)
=
\operatorname{softmax}
\begin{pmatrix}
A_r(t)\\
S_r(t)\\
B_r(t)
\end{pmatrix}
$$

`examples/physics/synapse_structural_burden.py`의 pilot에서는 STDP 시간차, 제3요인 $m$, 기존 가중치 $w_{ij}$를 넣어 $S_r$와 $x_{s,r}$를 계산했다.

| 지역 | plasticity load | $S_r$ | $x_{s,r}$ |
|---|---:|---:|---:|
| hippocampus | $0.040343$ | $0.065743$ | $0.256829$ |
| cortex | $0.029146$ | $0.050946$ | $0.250266$ |
| thalamus | $0.012623$ | $0.031423$ | $0.244881$ |

성공 조건은 plasticity proxy와 metabolic maintenance proxy가 $S_r$ 또는 $x_{s,r}$ 추정량과 같은 방향으로 움직이는 것이다. 반대로 학습량이 큰 지역에서 $S_r$가 오르지 않거나, $S_r$ 순위가 PET 대사량과 glial marker 순위와 반복적으로 어긋나면 이 연결은 실패한다.

### 5.7 예측 게이트

각 축의 역추정이 끝나도 결론은 아직 아니다. 상태 모델이 단일 proxy보다 예측력이 좋아야 한다.

$$
\mathcal L_{\mathrm{state}}
<
\min
\left(
\mathcal L_{\mathrm{mean}},
\mathcal L_{\mathrm{flat}},
\mathcal L_{\mathrm{single}}
\right)
$$

`examples/physics/brain_state_prediction_gate.py`의 pilot에서는 다음 결과가 나온다.

| 모델 | holdout loss |
|---|---:|
| state model | $0.000804$ |
| mean baseline | $0.073444$ |
| single active baseline | $0.011654$ |

가장 강한 baseline은 single active baseline이고, 상태 모델의 상대 손실은 다음이다.

$$
\frac{
\mathcal L_{\mathrm{state}}
}{
\mathcal L_{\mathrm{single}}
}
=0.068989
$$

따라서 pilot에서는 예측 게이트를 통과한다. 실제 연구에서는 같은 비교를 subject split, session split, task split에서 반복해야 한다. 이 부등식이 깨지면 $p_r=(x_a,x_s,x_b)$는 설명용 좌표일 수는 있어도 예측 모델로는 결론이 아니다.

| 작업 | 필요한 데이터 | 산출물 |
|---|---|---|
| 수면박탈 EEG 검증 | eyes-open, eyes-closed EEG와 PVT | $r_\alpha$, $\mathrm{RT}_{\mathrm{pvt}}$, $z_n$ 부호 |
| resting background 검증 | resting fMRI 또는 resting EEG | $x_{b,r}$ 신뢰도 |
| 과제 활성 검증 | task fMRI 또는 EEG/MEG | $x_{a,r}$ 부호 |

| 전역 방정식 ablation | 반복 측정, connectivity, 복수 proxy | $\mathcal L_{\mathrm{full}}/\mathcal L_{\mathrm{best\;ablation}}$ |

### 6.1 데이터 사냥 지도

다음 단계는 새 검증기를 만드는 것이 아니라, 공개 데이터에서 $P_n\to P_{n+1}$ 전이를 뽑는 것이다. 데이터 후보는 식의 항별로 나눈다.

| 우선순위 | 데이터 후보 | 바로 검증할 항 | 장점 | 약점 |
|---:|---|---|---|---|
| 1 | OpenNeuro `ds004902` resting-state EEG sleep deprivation | $H(Q-Q^*)$, $F_{\mathrm{slow}}$, $x_b$, PVT proxy | 71명, normal sleep vs sleep deprivation, EEG와 PVT가 같이 있음 | graph와 fMRI region mapping은 약함 |
| 2 | OpenfMRI/OpenNeuro `ds000201` Sleepy Brain Project | $U_{\mathrm{task}}$, $H$, $\Delta_G$, $F_{\mathrm{slow}}$ | partial sleep deprivation, rest/task fMRI, T1/T2, DTI가 같이 있음 | 용량이 크고 전처리 비용이 큼 |
| 3 | Human Connectome Project young adult | $\Delta_G$, $U_{\mathrm{task}}$, anatomical generator $z_{\tau}$ | 대규모 rest/task fMRI, diffusion, behavioral data | 수면박탈 개입이 없어 $F_{\mathrm{slow}}$ 검증은 약함 |
| 4 | Allen Human Brain Atlas | anatomical generator $z_{\tau}$, plasticity/homeostasis prior | region별 gene expression, brainstem/subcortex 포함, 다운로드 가능 | 전이 데이터가 아니라 계수 prior용 |
| 5 | 공개 HD-EEG/PSG 수면 데이터 | sleep stage, recovery, slow floor | 수면 단계와 spectral 변화가 직접적 | task와 graph 항은 별도 데이터가 필요 |

가장 먼저 팔 데이터는 `ds004902`다. 이미 sleep deprivation, PVT, SSS, resting EEG가 같은 subject 안에 들어 있으므로 다음 전이를 만들 수 있다.

$$
P_{\mathrm{NS}}
\rightarrow
P_{\mathrm{SD}}
$$

여기서 $Q_n$은 sleepiness, PVT, EEG spectral burden으로 만들고, $P_n$은 전극 topography를 정점 가족별 proxy로 접는다. 이 데이터는 full equation 전체가 아니라 $H(Q-Q^*)$와 $F_{\mathrm{slow}}$의 부호, $x_b$ 감소, $x_a$ 증가를 먼저 검증한다.

두 번째 후보는 `ds000201`이다. 이 데이터는 partial sleep deprivation, resting fMRI, task fMRI, DTI가 함께 있으므로 graph term까지 들어간 전이를 만들 수 있다.

$$
\left(P_{\mathrm{normal}},G_{\mathrm{DTI}},Q_{\mathrm{normal}}\right)
\rightarrow
P_{\mathrm{deprived}}
$$

세 번째 후보인 HCP는 수면박탈 검증용이 아니라 $\Delta_G$와 anatomical generator의 배경 좌표를 고정하는 데 쓴다. Allen Human Brain Atlas는 더 직접적으로 $z_{\tau}$를 만드는 데 쓴다. 즉 task recruitment, relay graph, plasticity axis, homeostatic depth, sleep recovery axis를 손으로 쓰지 않고 공개 anatomical feature로 치환하는 경로다.

### 6.2 데이터별 산출물

| 데이터 | 1차 산출물 | gate 투입 여부 |
|---|---|---|
| `ds004902` | subject별 `NS -> SD` 전이, $z_{\mathrm{sleep}}$, PVT burden, EEG spectral proxy | partial, graph 없이 check 가능 |
| `ds000201` | normal/partial deprivation fMRI state, DTI graph, task/rest contrast | full real gate 후보 |
| HCP | canonical graph, task/rest reference response, generator axis reference | real gate 보조 입력 |
| Allen Human Brain Atlas | $z_{\tau}$ anatomical prior, synaptic/homeostatic gene proxy | generator prior |

`ds004902`의 첫 adapter는 raw EEG를 직접 처리하지 않는다. 먼저 subject/session별 요약표를 만든 뒤, 다음 컬럼을 읽어 `NS -> SD` 전이 JSON으로 접는다.

| 필드 | 허용 컬럼명 |
|---|---|
| subject | `subject`, `participant_id`, `participant`, `sub` |
| session | `session`, `condition` |
| posterior alpha reactivity | `posterior_alpha_reactivity_ratio`, `alpha_reactivity`, `alpha_ratio` |
| PVT mean RT | `pvt_mean_rt_ms`, `pvt_mean_rt`, `mean_rt_ms` |
| sleepiness | `stanford_sleepiness_scale`, `sss`, `sleepiness` |

region별 관측 상태를 직접 만들려면 같은 표에 다음 컬럼을 추가한다.

| 필드 | 허용 컬럼명 |
|---|---|
| region | `region` |
| active state proxy | `active_proxy`, `x_a`, `active` |
| structural state proxy | `structural_proxy`, `x_s`, `structural` |
| background state proxy | `background_proxy`, `x_b`, `background` |

실행은 다음 순서다.

```bash
PYTHONIOENCODING=utf-8 ./.venv/Scripts/python.exe examples/physics/ds004902_transition_adapter.py path/to/ds004902_summary.tsv --output path/to/ds004902_transitions.json
PYTHONIOENCODING=utf-8 ./.venv/Scripts/python.exe examples/physics/brain_equation_integrated_gate.py --check-real-transitions path/to/ds004902_transitions.json
```

region 컬럼이 없으면 adapter의 출력은 `control_proxy_only`다. 즉 `observed_next_p`가 region-resolved EEG/fMRI 관측에서 온 것이 아니라, sleepiness, PVT, alpha reactivity의 방향성을 이용한 1차 proxy-derived transition이다. 이 경우 full real gate 결론으로 쓰지 않는다.

region별 `active_proxy`, `structural_proxy`, `background_proxy`가 들어오면 출력은 `region_resolved_state`가 된다. 이때 `current_p`와 `observed_next_p`는 각 region의 관측 proxy에서 직접 정규화되므로, 비로소 실제 전이 후보로 본다.

따라서 첫 판정은 full real gate가 아니라 출력 JSON의 `adapter.burden_diagnostics`다. 여기서 `alpha_direction_rate`, `pvt_direction_rate`, `sleepiness_direction_rate`, `all_proxy_direction_rate`를 먼저 본다. 이 값들이 낮으면 전역 방정식 이전에 수면박탈 proxy 자체가 일관되지 않은 것이다.

region-resolved 입력에서는 `adapter.state_delta_diagnostics`도 본다. 핵심 값은 `mean_delta_active`, `mean_delta_background`, `mean_sleep_deprivation_direction_rate`다. 수면박탈 1차 가설은 $x_a$가 증가하고 $x_b$가 감소하는 것이므로, 이 방향률이 낮으면 관측 사상 $P=\mathcal O(Y)$부터 다시 잡아야 한다.

scalp EEG만으로는 hypothalamus, brainstem, autonomic 같은 canonical region을 직접 채우기 어렵다. 이때 탐색용으로만 `--fill-missing-regions`를 쓸 수 있다.

```bash
PYTHONIOENCODING=utf-8 ./.venv/Scripts/python.exe examples/physics/ds004902_transition_adapter.py path/to/ds004902_region_summary.tsv --fill-missing-regions --output path/to/ds004902_transitions.json
```

이 옵션은 빠진 region을 $P^*$로 채우고 `adapter.region_coverage`에 `observed_regions`, `missing_regions`, `complete_subject_rate`를 남긴다. 보완 region이 있으면 full brain 결론으로 올리지 않고, EEG가 직접 관측한 region subset의 탐색 결과로만 둔다.

이 순서로 가면 바로 결론을 주장하지 않고도, 먼저 “수면박탈이 $p_r$를 실제로 미는가”를 공개 EEG에서 확인하고, 그 다음 fMRI/DTI 데이터로 graph와 task 항까지 확장할 수 있다.

## 7. 현재 결론

현재 연구해서 증명 가능한 뇌 모델은 다음까지다.

> 뇌는 지역 상태 $p_r$, 느린 제어축 $q_n$, 회로 결합 $\Delta_G$, 시냅스 가소성 $\Delta w_{ij}$가 결합된 동역학계이며, 수면박탈과 회복 수면은 이 동역학을 가장 직접적으로 검증하는 개입이다.

이번 연구에서 실제로 설명한 것은 아래 전역 방정식의 각 항과 관측 사상이다.

$$
P_{n+1}
=
\Pi_{(\Delta^2)^{|V_{\mathrm{brain}}|}}
\left(
(1-\rho_B)P^*
+\rho_BP_n
+\gamma\Delta_GP_n
+U_{\mathrm{task},n}
+H(Q_n-Q^*)
+F_{\mathrm{syn}}(E_n,M_n,W_n)
+F_{\mathrm{slow}}(D_{\mathrm{sleep},n})
\right)
$$

현재 결과값으로 역추적된 계수는 다음이다.

| 항 | 역추적 결과 |
|---|---|
| 수면박탈 입력 $z_n$ | $z_{\mathrm{mean}}=0.3445$, $z_{\mathrm{geom}}=0.2927$ |
| 수면 forcing 부호 | $h_r^{\mathrm{sleep}}=(+\alpha_r,0,-\alpha_r)^\top$ |
| 관측 가능한 $\alpha_r$ 범위 | $\Delta x=0.01$이면 $\alpha_r\approx0.03$, $\Delta x=0.03$이면 $\alpha_r\approx0.09$부터 $0.10$ |
| 다축 항상성 민감도 | sleep top: hypothalamus $0.1865$, arousal top: brainstem $0.1596$, metabolic top: default mode $0.1324$ |
| 급성 복구율 | $\rho_B^{\mathrm{fast}}\in[0.1496,0.1602]$ |
| 야간 fast 잔차율 | $\rho_{\mathrm{night}}^{\mathrm{fast}}=0.3119\pm0.0055$ |
| slow 잔차율 | $\rho_{\mathrm{night}}^{\mathrm{slow}}=0.6121$ |
| graph 안정 상한 | $\gamma<0.3933$ |
| graph 평탄화 효과 | 최대/최소 반응비 $2.50\to1.88$ as $\gamma:0\to0.30$ |
| graph-vs-flat pilot | $\mathcal L_{\mathrm{graph}}/\mathcal L_{\mathrm{flat}}=0.084453$ |
| 시간축 inverse pilot | $\widehat\rho_B=0.1550\pm0.0016$, $\widehat\gamma=0.0300\pm0.0006$ |
| slow floor 시작 | $D_{\mathrm{sleep}}\approx20.0$ h |
| 큰 floor 기준 | $D_{\mathrm{sleep}}\approx35.5$ h에서 $\phi_{\mathrm{floor}}\approx0.50$ |
| 예측 게이트 pilot | $\mathcal L_{\mathrm{state}}/\mathcal L_{\mathrm{single}}=0.068989$ |
| 통합 전역식 gate | $\mathcal L_{\mathrm{full}}/\mathcal L_{\mathrm{best\;ablation}}=0.165477$ |
| 전역 계수 inverse gate | $\widehat\rho_B=0.161930$, $\widehat\gamma=0.110599$, max error $0.101397$ | synthetic 전이에서 계수 회수 |
| split inverse gate | subject/session/task 모두 통과, max split error $0.004964$ | synthetic split 밖 예측 |
| CE-뇌 bridge gate | modal radius $0.197441$, ablation gain $0.834523$ | 안정 functional 구조 대응 |

즉 지금 결과는 뇌 전체를 끝낸 것이 아니라, 과제 활성, 배경 reserve, 시냅스 구조 부담, 수면 회복을 이용해 전역 상태 방정식의 일부 계수와 synthetic ablation gate를 역추적한 것이다.

다음에 모아야 할 결과값은 회로별 $\Delta x_a$, $\Delta x_b$다. 이것이 들어오면

$$
\widehat\alpha_r=
\frac{\Delta x_{a,r}}{z_{\mathrm{sleep}}}
$$

로 $h_r^{\mathrm{sleep}}$의 공간 지도를 얻고, 회복 시점별 잔차를 고유모드로 분해해 $\rho_B$와 $\gamma$를 다시 추정한다. 이 값들이 같은 subject와 조건에서 반복되면 $q_{\mathrm{sleep}}\rightarrow p_r\rightarrow B$에 해당하는 뇌 구조와 방정식의 첫 블록이 닫힌다.
