# 뇌 이론 제9장: 생명에서 지능까지

이 장이 생명-뇌-지능 진화 trace의 정본이다. 실행 스크립트와 자동 리포트는 근거 자료이고, 최종 해석, 수식, 판정, 다음 병목은 이 장을 기준으로 읽는다.

## 초록

이 장의 목적은 "뇌 방정식"을 인간 뇌에서 갑자기 시작하는 식으로 두지 않고, 생명 최소 동역학에서 C. elegans, Drosophila, zebrafish, mouse로 이어지는 진화 사다리 위에서 역추적하는 것이다. 핵심 질문은 각 단계에서 방정식의 어떤 항이 새로 필요해지는가다. 현재까지의 결과는 네 가지로 요약된다.

첫째, C. elegans 수준의 최소 원시 신경계에서는 binary topology가 아니라 weighted chemical connectivity가 감각-domain을 출력-domain으로 보내는 routing operator로 닫힌다.

둘째, Drosophila에서는 memory 하나가 단독으로 추가되는 것이 아니라 cell type, descending/action, mushroom body, central complex 계열이 함께 분화하고, adult FlyWire 규모에서는 이 공동 분화가 permutation gate를 통과한다.

셋째, zebrafish에서는 실제 calcium activity state가 assembly/recurrent 구조와 행동 frame/direction association까지 닫히지만, continuous speed/turn decoding은 공개 partial/Others 자료에 e2-behavior timestamp bridge가 없어서 data-boundary로 남는다.

넷째, mouse IBL/OpenAlyx에서는 첫 strict Neuropixels session과 motor/striatal multi-probe session이 통과했고, 이어 5개 후보 cross-session panel에서 4개 후보가 choice, first-movement speed, wheel action direction을 모두 통과했다.

마지막으로 strict cluster acronym에서 생기던 26.9703% unknown spike caveat는 channel-level CCF id fallback으로 0%까지 줄였고, hybrid region-id decoder도 choice 4/5, speed 5/5, wheel 4/5 반복성을 유지했다. 추가 baseline gate에서는 timing-only 반례가 세 target 모두 5/5로 깨졌지만, task-history increment는 speed 5/5, wheel 4/5, choice 2/5라서 choice 항은 stimulus/task-table 성분이 강한 caveat를 남겼다. 이어 flat-unit gate에서는 top-unit readout이 choice 5/5, wheel 5/5, speed 2/5에서 hybrid region을 이겼지만, same-window region interaction과 lagged region coupling은 모두 음성이었다. 마지막 nested regularization gate에서는 \(task+region\) 뒤의 all-unit residual이 choice 0/5, speed 1/5, wheel 0/5로 반복되지 않았고, 오히려 작은 \(region\mid task+unit\) residual이 3/5, 3/5, 4/5로 남았다. 따라서 mouse region 항은 maximal neural decoder가 아니라 task/history 위에서 살아남는 anatomically compressed readout으로 해석해야 한다.

따라서 이 장의 현재 결론은 다음이다.

$$
\boxed{
\mathrm{life}
\rightarrow
\mathrm{weighted\ neural\ routing}
\rightarrow
\mathrm{role\ differentiation}
\rightarrow
\mathrm{activity\ state\ closure}
\rightarrow
\mathrm{mammalian\ region\ loop}
}
$$

이 결론은 지능을 단순한 규모 증가가 아니라, 보존되는 제어 동역학 위에 새 연산자가 추가되는 과정으로 읽게 한다.

## 중심 질문

인간 뇌는 갑자기 생긴 장치가 아니라, 원시 생명, 원시 신경계, 감각-운동 회로, 학습 가능한 내부 loop, 척추동물 활동 상태공간, 포유류 영역 loop가 층층이 쌓인 진화체다. 따라서 뇌 방정식은 다음 질문을 받아야 한다.

$$
\boxed{
\text{생명에서 지능까지 올라갈 때, 방정식의 어떤 항이 새로 필요해지는가?}
}
$$

단순히 neuron 수, synapse 수, brain volume만 비교하지 않는다. 각 단계에서 실제로 닫히는 항을 본다.

| 축 | 질문 | 닫힘의 의미 |
|---|---|---|
| self-maintenance | 열린 반응계가 선택 가능한 자기유지 동역학으로 닫히는가 | 생명 최소항 |
| routing | 자극 domain이 같은 행동/output domain으로 안정적으로 흐르는가 | 원시 신경계 항 |
| differentiation | cell type, action, memory 역할이 분화되는가 | 곤충 단계 항 |
| activity closure | 실제 neural activity state가 다음 상태를 예측하는가 | 척추동물 activity 항 |
| behavior bridge | activity가 trial/frame 행동을 예측하는가 | activity-to-behavior 항 |
| region loop | 포유류 영역이 decision, action, internal state를 나누는가 | mouse/human 전 단계 항 |
| workspace | 장기 memory, replay, abstract task state가 전역적으로 묶이는가 | 고등 지능 후보항 |

현재까지 얻은 핵심 결론은 다음이다.

$$
\boxed{
\text{지능은 신경계 양의 단순 증가가 아니라, 보존되는 weighted routing 위에 역할 분화와 닫힌 activity state가 쌓인 결과다.}
}
$$

## 주장과 기여

이 장은 관측 자료를 곧바로 하나의 거대 식에 맞추려는 문서가 아니다. 먼저 종별로 가장 작은 닫힌 gate를 만들고, 통과한 항만 상위 방정식으로 승격한다. 그래서 주장은 네 층으로 나뉜다.

| 층 | 주장 | 현재 지위 |
|---|---|---|
| 생명 최소식 | boundary/copy/autocatalysis가 선택 가능한 열린 자기유지 동역학을 만든다 | 아직 gate 없음 |
| 원시 신경계 | weighted chemical graph가 최초 neural routing operator다 | C. elegans에서 supported |
| 곤충 단계 | C. elegans 다음 항은 memory 단독이 아니라 celltype/action/memory 공동 분화다 | adult FlyWire에서 closed |
| 척추동물 단계 | 실제 neural activity state는 recurrent/assembly 구조와 discrete behavior bridge를 가진다 | zebrafish에서 closed with caveat |
| 포유류 단계 | region loop가 decision/action/internal state를 분리해야 한다 | mouse/IBL candidate panel passed with registration caveat reduced |

이 문서의 기여는 "어떤 생물이 더 지능적이다"라는 서술이 아니라, 각 진화 단계에서 방정식에 새로 들어가는 항을 검증 가능한 손실함수로 분리한 점이다. 따라서 아래의 식들은 생물학적 장식이 아니라 실험 gate를 생성하는 압축 표현이다.

## 공통 상태 묶음

종 또는 발달 단계 \(s\)의 상태를 다음 네 묶음으로 둔다.

$$
Z_n^{(s)}
=
\left(
P_n^{(s)},q_n^{(s)},m_n^{(s)},W_n^{(s)}
\right)
$$

| 기호 | 의미 |
|---|---|
| \(P_n^{(s)}\) | 빠른 biochemical, neural, calcium activity state |
| \(q_n^{(s)}\) | 몸 상태, 에너지, 각성, 항상성 변수 |
| \(m_n^{(s)}\) | 느린 내부 상태, short memory, synaptic trace, eligibility trace |
| \(W_n^{(s)}\) | weighted chemical, structural, effective graph |

원시 생명 단계에는 \(P\) 대신 chemical population \(X\)가 주 상태다. C. elegans 구조 gate에서는 \(W\)가 고정 connectome으로 관측된다. Zebrafish부터는 \(P_t\)가 실제 calcium activity matrix로 들어온다. Mouse/IBL부터는 \(P_t\)를 spike-rate/region-binned activity로 접는다.

## 공통 고도화식

빠른 상태식은 다음 controlled dynamical family로 둔다.

$$
P_{n+1}^{(s)}
=
\Pi_{\mathcal S_s}
\left[
\rho_sP_n^{(s)}
+(1-\rho_s)P_*^{(s)}
+\gamma_s\mathcal L(W_{\mathrm{chem/effective}}^{(s)})P_n^{(s)}
+\sum_{d\in\mathcal D_s}a_{d,n}^{(s)}U_d^{(s)}(z_n,q_n)
+H_s(q_n^{(s)}-q_*^{(s)})
+A_sb_n^{(s)}
+M_sm_n^{(s)}
+\eta_n^{(s)}
\right]
$$

행동 관측식은 다음으로 둔다.

$$
b_{n+1}^{(s)}
=
\Omega_s(P_n^{(s)},q_n^{(s)},E_n)
+\xi_{b,n}^{(s)}
$$

항상성식은 다음으로 둔다.

$$
q_{n+1}^{(s)}
=
q_n^{(s)}
+B_s(E_n)
-C_s(b_n^{(s)},P_n^{(s)})
-\chi_s(q_n^{(s)}-q_*^{(s)})
$$

느린 memory/synaptic trace는 다음으로 둔다.

$$
m_{n+1}^{(s)}
=
\lambda_sm_n^{(s)}
+\Psi_s(P_n^{(s)},b_n^{(s)},r_n)
$$

학습 가능한 graph 또는 synaptic weight 변화는 다음으로 둔다.

$$
W_{n+1}^{(s)}
=
\Pi_{\mathcal W_s}
\left[
W_n^{(s)}
+\epsilon_s\Phi_s(P_n^{(s)},b_n^{(s)},r_n)
-\mu_sW_n^{(s)}
\right]
$$

여기서 가장 중요한 제약은 다음이다.

$$
\boxed{
\mathcal L(W)\neq \mathcal L(A_{\mathrm{binary}})
\quad\text{and}\quad
\mathcal L(W)=\mathcal L(W_{\mathrm{weighted}})
}
$$

C. elegans gate에서 반복적으로 확인된 것은 binary adjacency가 아니라 weighted chemical graph가 원시 신경계의 routing operator라는 점이다. 따라서 포유류나 인간으로 올라가도 graph term을 검증할 때 단순 edge existence보다 weight/effective connectivity를 우선한다.

## 식의 유도

공통 고도화식은 임의로 쓴 큰 식이 아니다. 출발점은 모든 단계에서 관측 가능한 상태가 다음 세 힘 사이에서 움직인다는 가정이다.

1. 자기 상태를 유지하려는 residual term
2. 구조 또는 graph를 따라 퍼지는 coupling term
3. 외부 자극, 몸 상태, 행동 결과, 느린 memory가 밀어 넣는 control term

이를 가장 작은 affine-controlled map으로 쓰면 다음이다.

$$
\tilde P_{n+1}
=
\rho P_n
+(1-\rho)P_*
+G(P_n;W)
+U_n
+H_n
+A_nb_n
+M_nm_n
+\eta_n
$$

생물학적 상태는 임의의 유클리드 벡터가 아니라 확률, firing proportion, calcium activity score, population fraction처럼 제한된 상태공간에 놓인다. 그래서 마지막에 projection을 둔다.

$$
P_{n+1}
=
\Pi_{\mathcal S}(\tilde P_{n+1})
$$

여기서 \(\Pi_{\mathcal S}\)의 역할은 세 가지다.

| 역할 | 의미 |
|---|---|
| boundedness | activity, probability, fraction이 물리적으로 가능한 범위를 벗어나지 않게 한다 |
| conservation | coarse state를 simplex나 정규화된 score space에 남긴다 |
| comparability | 종마다 다른 관측량을 같은 gate 문법으로 비교하게 한다 |

### Graph term

graph coupling은 처음에는 일반 \(G(P;W)\)로 둘 수 있다. 하지만 C. elegans에서 실제로 통과한 것은 edge existence가 아니라 weighted chemical synapse였으므로, 최소 operator는 Laplacian 또는 row-normalized transition의 형태로 좁힌다.

$$
G(P;W)
=
\gamma\mathcal L(W)P
$$

Laplacian형으로 쓰면

$$
[\mathcal L(W)P]_i
=
\sum_j w_{ij}(P_j-P_i)
$$

transition형으로 쓰면

$$
T_{ij}(W)
=
\frac{w_{ij}}{\sum_k w_{ik}+\varepsilon},
\qquad
P_{n+1}\supset \gamma T(W)P_n
$$

두 표현의 공통 의미는 "상태가 연결 강도에 따라 이동한다"는 것이다. C. elegans 결과가 중요한 이유는 여기서 \(W\)를 binary adjacency로 두면 gate가 깨지고, chemical weighted \(W\)로 둘 때 gate가 닫히기 때문이다.

### Stimulus-output term

자극 domain \(d\)가 들어오면 상태는 해당 domain의 input subspace에서 output subspace로 흐른다. 이를

$$
U_n
=
\sum_{d\in\mathcal D}a_{d,n}U_d
$$

로 둔다. \(a_{d,n}\)은 그 순간 domain \(d\)가 켜져 있는 정도이고, \(U_d\)는 그 domain이 밀어 넣는 방향이다. connectome proxy에서는 이 항을 직접 시간열로 보지 못하므로, row-normalized flow로 다음 비율을 검증한다.

$$
R_d
=
\frac{\mathrm{Flow}(L1_d\to L3_d)}
{\mathrm{mean}_{d'\ne d}\mathrm{Flow}(L1_d\to L3_{d'})}
$$

\(R_d>1\)이면 같은 domain output으로 흐르는 경로가 더 강하다는 뜻이고, \(R_d>1.5\) 및 permutation \(p<0.05\)면 domain-preserving channel로 승격한다. 이 단계는 실제 행동 trial은 아니지만, 원시 신경계가 이미 "자극을 행동 출력 계열로 보존해서 보내는 구조"를 가진다는 것을 보여준다.

### Homeostatic term

원시 생물부터 포유류까지 nervous system은 몸 밖에 떠 있는 계산기가 아니다. 에너지, 각성, 대사, 내부 상태가 activity를 밀고 당긴다. 그래서

$$
H_n
=
H_s(q_n-q_*^{(s)})
$$

를 둔다. 이 항은 C. elegans connectome gate에서는 직접 측정하지 않았지만, 상위 뇌 문서의 항상성 축과 연결되는 자리다. 수학적으로는 기준 몸 상태 \(q_*\)에서 벗어난 만큼 neural state를 특정 방향으로 이동시키는 forcing이다.

### Behavior and memory terms

행동이 단순 output label이면

$$
b_{n+1}
=
\Omega(P_n,q_n,E_n)
$$

로 충분하다. 그러나 Drosophila부터는 output이 단순 reflex가 아니라 action selection과 memory loop에 의해 바뀐다. 그래서 행동 항 \(A_sb_n\)과 느린 memory 항 \(M_sm_n\)을 분리한다.

$$
m_{n+1}
=
\lambda_sm_n
+\Psi_s(P_n,b_n,r_n)
$$

\(\lambda_s\)는 흔적이 얼마나 오래 남는지, \(\Psi_s\)는 neural state와 행동 결과, reward 또는 reinforcement signal이 memory trace를 어떻게 갱신하는지 나타낸다. Adult FlyWire에서 닫힌 것은 이 memory 항 하나가 아니라, memory 구조가 action/central-complex/descending 구조와 함께 분화한다는 점이다.

### Activity closure와 behavior bridge

Zebrafish부터는 \(W\)만 보지 않고 실제 activity \(P_t\)를 본다. 이때 최소 동역학 gate는 다음이다.

$$
\hat P_{t+1}
=
f_\theta(P_t),
\qquad
\mathcal L_{\mathrm{dyn}}
=
\frac{\mathrm{MSE}(P_{t+1},\hat P_{t+1})}
{\mathrm{MSE}(P_{t+1},\bar P)}
$$

이 값이 1보다 작으면 평균 상태로 예측하는 것보다 현재 상태가 다음 상태를 더 잘 설명한다. 즉 activity state가 단순 noise가 아니라 닫힌 상태공간 위에 있다는 뜻이다.

행동 bridge는 한 단계 더 강하다.

$$
\hat y_t
=
g_\theta(P_{t-\ell},q_t),
\qquad
y_t\in\{\mathrm{bout},\mathrm{direction},\mathrm{speed},\mathrm{turn},\mathrm{heading}\}
$$

discrete bout/direction은 현재 zebrafish 자료에서 통과했다. 그러나 continuous speed/turn/heading은 \(P_t\)와 \(y_t\)의 timestamp alignment가 필요하다. alignment가 없으면 \(g_\theta\)를 아무리 잘 맞춰도 물리적 검증이 아니라 임의 대응이 된다. 그래서 zebrafish continuous는 실패가 아니라 data-boundary로 판정한다.

## 검증 문법

모든 단계는 같은 판정 문법으로 비교한다.

Graph structure:

$$
\mathcal L_{\mathrm{graph}}
=
\frac{
\mathcal L(\mathrm{structured\ block})
}{
\mathcal L(\mathrm{flat})
}
$$

성공 조건:

$$
\mathcal L_{\mathrm{graph}}<1
\quad\text{and}\quad
p_{\mathrm{perm}}<0.05
$$

Domain routing:

$$
\mathcal L_{\mathrm{route}}
=
\frac{
\mathrm{Flow}(d\rightarrow d)
}{
\mathrm{mean}_{d'\ne d}\mathrm{Flow}(d\rightarrow d')
}
$$

성공 조건:

$$
\mathcal L_{\mathrm{route}}>1.5
\quad\text{and}\quad
p_{\mathrm{perm}}<0.05
$$

Activity closure:

$$
\mathcal L_{\mathrm{dyn}}
=
\frac{
\mathrm{MSE}(P_{t+1}\mid P_t)
}{
\mathrm{MSE}(P_{t+1}\mid \bar P)
}
$$

성공 조건:

$$
\mathcal L_{\mathrm{dyn}}<1
$$

Continuous behavior:

$$
\mathcal L_{\mathrm{beh}}
=
\frac{
\mathrm{MSE}(y_t\mid P_{t-\ell},q_t)
}{
\mathrm{MSE}(y_t\mid \bar y)
}
$$

성공 조건:

$$
\mathcal L_{\mathrm{beh}}<1,\qquad R^2>0,\qquad p_{\mathrm{shift}}<0.05
$$

Loop enrichment:

$$
\mathcal E_{\mathrm{loop}}
=
\frac{
\mathrm{LoopFraction}_{\mathrm{observed}}
}{
\mathbb E[\mathrm{LoopFraction}_{\mathrm{random}}]
}
$$

성공 조건:

$$
\mathcal E_{\mathrm{loop}}>2
\quad\text{and}\quad
p_{\mathrm{random}\ge\mathrm{observed}}<0.05
$$

자료 경계 판정:

$$
\mathrm{data\ boundary}
\Longleftrightarrow
\left(
\text{필요한 time alignment 또는 behavior bridge가 없음}
\right)
$$

이 경우는 "방정식 실패"로 올리지 않는다. 특히 zebrafish continuous movement는 여기에 해당한다.

## 통계 판정과 모델 비교

각 gate는 "숫자가 좋아 보인다"만으로 통과하지 않는다. 최소한 다음 세 가지를 분리한다.

| 비교 | 목적 | 예 |
|---|---|---|
| flat baseline | 구조를 넣는 것이 평균/단일 모델보다 나은가 | block/flat, recurrent/baseline |
| random label baseline | 같은 label 수의 무작위 구조보다 나은가 | permutation p |
| competing model | 우리가 제안한 항 말고 더 간단한 설명이 있는가 | action split, module family, all-one |

block model에서는 관측된 weighted adjacency \(W\)를 label pair별 평균 block matrix \(\hat W\)로 재구성한다. 손실은 다음으로 둔다.

$$
\mathcal L_{\mathrm{block}}
=
\sum_{i,j}
\left(
W_{ij}-\hat W_{\ell_i,\ell_j}
\right)^2
$$

flat model은 모든 edge를 하나의 평균으로 예측한다.

$$
\mathcal L_{\mathrm{flat}}
=
\sum_{i,j}
\left(
W_{ij}-\bar W
\right)^2
$$

따라서 \(\mathcal L_{\mathrm{block}}/\mathcal L_{\mathrm{flat}}<1\)이면 label 구조가 아무 구조 없는 평균보다 낫다. 하지만 label 수가 늘면 손실은 기계적으로 줄어들 수 있으므로 permutation과 BIC-like penalty를 같이 본다.

Permutation p는 같은 label count를 유지한 채 label assignment를 무작위로 섞어 계산한다.

$$
p_{\mathrm{perm}}
=
\frac{
1+\#\{r:\mathcal L_{\mathrm{rand},r}\le \mathcal L_{\mathrm{obs}}\}
}{
1+N_{\mathrm{perm}}
}
$$

여기서 \(p_{\mathrm{perm}}<0.05\)이면 관측 label이 같은 크기의 무작위 label보다 유의하게 낫다.

BIC-like 점수는 포화모델을 경계하기 위한 보조 기준이다.

$$
\mathrm{BIC}^*
=
N\log
\left(
\frac{\mathcal L}{N}+\varepsilon
\right)
+k\log N
$$

\(k\)는 block parameter 수다. 이 값은 엄밀한 likelihood BIC가 아니라 "label을 너무 많이 쪼갠 모델이 과도하게 이기는지"를 보는 안전장치다. 그래서 C. elegans에서는 layer가 살아남고, Drosophila larva에서는 memory 단독 가설이 충분히 강하지 않다는 판정이 가능했다.

Activity gate에서는 같은 원칙을 시간열에 적용한다.

$$
\mathcal L_{\mathrm{recurrent}}
=
\frac1T
\sum_t
\|P_{t+1}-\hat P_{t+1}(P_t)\|^2
$$

baseline은 다음 시점을 전체 평균 \(\bar P\)로 예측하는 모델이다.

$$
\mathcal L_{\mathrm{baseline}}
=
\frac1T
\sum_t
\|P_{t+1}-\bar P\|^2
$$

따라서 recurrent/baseline이 1보다 작으면 activity가 단순 평균 주변 noise가 아니라 자기 상태를 가진 동역학이라는 뜻이다.

## 단계 0: 생명 최소 동역학

신경계 이전의 생명은 "선택 가능한 열린 자기유지 동역학"으로 둔다. 상태는 molecule population \(X_n\)다.

$$
X_{n+1}
=
\Pi_{\mathcal X}
\left[
X_n
+R_{\mathrm{autocat}}(X_n,E_n)
-D_{\mathrm{decay}}(X_n)
-L_{\mathrm{leak}}(X_n)
+B_{\mathrm{boundary}}(X_n)
+C_{\mathrm{copy}}(X_n)
\right]
$$

항의 의미:

| 항 | 의미 |
|---|---|
| \(R_{\mathrm{autocat}}\) | 자기촉매 반응망 |
| \(D_{\mathrm{decay}}\) | 분해와 열적 손실 |
| \(L_{\mathrm{leak}}\) | boundary 밖으로 빠지는 leakage |
| \(B_{\mathrm{boundary}}\) | 막, compartment, vesicle에 의한 유지 |
| \(C_{\mathrm{copy}}\) | template copying 또는 복제 bias |

이 단계의 의의는 boundary와 copy가 열린 화학 반응계를 선택 가능한 자기유지 동역학으로 바꾼다는 점이다. 아직 직접 gate는 만들지 않았다. 다음에 이 층을 열려면 LUCA 비교유전체학, RNA world/ribozyme 실험, autocatalytic set 모델, protocell/lipid vesicle 실험을 따로 잡아야 한다.

## 단계 3: Zebrafish larva activity

Zebrafish부터는 connectome proxy를 넘어 실제 calcium activity state를 본다. 목표는 두 가지다.

1. 실제 activity state가 다음 activity state를 예측하는가.
2. activity 또는 perturbation이 행동과 연결되는가.

축약식:

$$
P_{t+1}
=
\Pi
\left[
\rho P_t
+L_{\mathrm{lowrank}}(P_t)
+C_{\mathrm{assembly}}(P_t)
+U_{\mathrm{laser},t}
+H(q_t-q_*)
\right]
$$

행동 관측식:

$$
y_t
=
\Omega(P_{t-\ell},q_t)
+\xi_t,
\qquad
y_t\in
\{
\mathrm{bout},
\mathrm{left/right},
\mathrm{speed},
\mathrm{turn},
\mathrm{heading}
\}
$$

\(C_{\mathrm{assembly}}\)는 두 뉴런이 같은 assembly를 공유하는지에 따른 correlation block 항이고, \(L_{\mathrm{lowrank}}\)는 저차원 recurrent state 예측 항이다.

### Zebrafish 종합 의의

Zebrafish 단계에서 닫힌 것은 네 가지다.

1. assembly 공유 쌍은 random membership보다 상관구조를 훨씬 잘 설명했다.
2. 저차원 recurrent state는 평균 baseline보다 다음 시점을 잘 예측했다.
3. 방향성 perturbation은 방향성 회전 행동으로 강하게 닫혔다.
4. neural activity는 행동 bout frame과 baseline frame을 구분했고, left/right 방향 조건도 leave-one-trial-out에서 구분했다.

닫히지 않은 것은 하나다.

1. neural activity frame을 speed/heading/turn angle의 continuous tracking frame에 timestamp-certified로 붙이는 최종 bridge.

따라서 zebrafish 판정은 "activity state closure는 통과, continuous movement는 data-boundary"다.

## 단계 4: Mouse Neuropixels/IBL

Zebrafish 다음 단계는 mouse Neuropixels/IBL이다. 이유는 다음이다.

1. C. elegans에서 weighted routing을 닫았다.
2. Drosophila adult FlyWire에서 celltype/action/memory co-differentiation을 닫았다.
3. Zebrafish에서 activity state, perturbation behavior, discrete activity-behavior association을 닫았다.
4. 다음에는 포유류의 cortex, thalamus, striatum, midbrain, hippocampus가 decision/action/internal state를 어떻게 나누는지 봐야 한다.

후보식:

$$
P_{n+1}
=
\Pi
\left[
\rho P_n
+(1-\rho)P_*
+\gamma\Delta_G P_n
+U_{\mathrm{task},n}
+H(Q_n-Q^*)
+M_{\mathrm{internal},n}
+W_{\mathrm{workspace},n}
\right]
$$

Mouse/IBL 첫 gate:

| gate | 목적 | 성공 조건 |
|---|---|---|
| trial table audit | choice, feedback, stimulus, wheel timing이 한 세션에 있는지 | `_ibl_trials.table.pqt`, wheel datasets present |
| spike-region audit | spikes, clusters, channels, atlas region이 한 세션에 있는지 | spike times and region labels present |
| region activity closure | region-binned firing이 trial state를 예측하는지 | heldout decoding beats baseline |
| decision/action bridge | visual, thalamus, striatum, motor 계열이 choice/action timing을 분리하는지 | region-class model beats flat model |
| graph/flat comparison | 영역 묶음 모델이 flat neuron model보다 나은지 | \(\mathcal L_{\mathrm{region}}/\mathcal L_{\mathrm{flat}}<1\) |

trial \(i\), spike \(k\), spike time \(t_k\), cluster id \(c_k\), region/group map \(G(c_k)\)를 두면 region-binned feature는 다음처럼 정의했다.

$$
x_{ig}
=\frac{1}{b_i-a_i}
\sum_k
\mathbf 1[t_k\in[a_i,b_i]]
\mathbf 1[G(c_k)=g].
$$

이 식은 단순 spike count가 아니라 duration-normalized rate다. 만약 \(b_i-a_i\)가 trial마다 달라지면 단순 count는 response duration을 그대로 leak한다. 그래서 이 gate에서는 다음 두 고정 window를 사용했다.

| window | interval | target |
|---|---|---|
| stimulus window | `stimOn_times + 20 ms` to `stimOn_times + 320 ms` | choice sign, first-movement speed |
| movement window | `firstMovement_times - 100 ms` to `firstMovement_times + 200 ms` | wheel action direction |

비교한 feature model은 세 개다.

| model | definition | 의미 |
|---|---|---|
| `region_family` | visual cortex, visual thalamus, somatosensory thalamus, hippocampus, cingulate, other, unknown | 넓은 anatomical family |
| `acronym_region` | high-spike CCF acronyms plus unknown | 더 세분한 region acronym model |
| `global_rate` | \(\sum_g x_{ig}\) one scalar | 영역 구조를 지운 flat firing-rate baseline |

decoder는 z-scored ridge linear classifier다.

$$
\hat w
=
\arg\min_w
\|y-Xw\|_2^2
+\lambda\|w\|_2^2.
$$

각 fold에서 train set 평균과 표준편차로 \(X\)를 표준화하고, held-out fold의 score sign으로 binary label을 예측했다. 판정 지표는 class imbalance에 덜 민감한 balanced accuracy다.

$$
\mathrm{BA}
=
\frac12
\left(
\frac{TP}{P}
+
\frac{TN}{N}
\right).
$$

null은 label permutation이다. 즉 관측 feature \(X\)는 그대로 두고 \(y\)만 섞어 같은 cross-validation을 반복한다.

$$
p
=
\frac{
1+\#\{\mathrm{BA}_{\mathrm{perm}}\ge \mathrm{BA}_{\mathrm{obs}}\}
}{
1+N_{\mathrm{perm}}
}.
$$

$$
\boxed{
\mathrm{region/acronym\ activity}
\;>\;
\mathrm{global\ firing\ rate}
\;>\;
\mathrm{label\ permutation}
}
$$

따라서 첫 strict IBL session에서는 mouse region-binned activity가 choice, action direction, action timing class를 실제로 지지한다.

의의는 세 가지다.

첫째, Drosophila/Zebrafish에서 얻은 \(P_t\rightarrow b_t\) 항이 포유류에서는 단일 neural population이 아니라 region-indexed population vector로 올라간다.

$$
R_t
=
\left[
r_{\mathrm{VIS}}(t),
r_{\mathrm{TH}}(t),
r_{\mathrm{HIP}}(t),
r_{\mathrm{CG}}(t),
\ldots
\right].
$$

둘째, behavior variable은 더 이상 단일 \(b_t\)가 아니라 choice, wheel direction, first-movement latency처럼 분해된다.

$$
y_t
=
\left[
y_t^{\mathrm{choice}},
y_t^{\mathrm{wheel}},
y_t^{\mathrm{speed}}
\right],
\qquad
\hat y_t
=
h(BR_t).
$$

셋째, global firing rate가 유의한 target도 있지만, region/acronym model이 consistently 더 높다. 그러므로 mouse 단계에서 새로 승격되는 항은 "많이 발화한다"가 아니라 "어느 영역 묶음이 어느 행동 변수를 나누는가"다.

$$
\boxed{
\Phi_{\mathrm{mammal}}(t)
=
B_{\mathrm{region}}R_t
+
B_{\mathrm{action}}R_{t-\tau:t}
}
$$

단, 이 gate는 mouse 전체를 닫지 않는다. 첫 probe의 강한 영역은 thalamus, visual cortex, hippocampus 쪽이고 motor/striatal loop가 충분히 포함되지 않았다. 따라서 다음 mouse gate는 다중 probe 또는 motor/striatum 포함 세션에서 같은 decoder를 반복해, \(B_{\mathrm{region}}\)이 특정 probe 위치의 우연한 readout이 아니라 일반적인 region/action bridge인지 확인해야 한다. 이 병목을 아래 audit와 multi-probe gate로 바로 이어서 검사했다.

### Mouse IBL/OpenAlyx channel-region rescue gate

Cross-session gate를 통과한 뒤에도 가장 큰 약점은 `unknown` bin이었다. 특히 NYU-30 multi-probe gate에서 probe01은 strict acronym rows가 209인데 observed cluster slots가 561이라, spike의 74.4123%가 strict acronym 밖으로 빠졌다. 이 상태에서 decoder가 통과해도 결론은 "region code가 충분하다"가 아니라 "큰 censored bin을 보존한 상태에서도 decoding이 된다"에 머문다.

그래서 다음 gate는 strict cluster acronym만 쓰던 region map을 channel-level CCF id fallback으로 보강했다. IBL/OpenAlyx에는 다음 두 파일이 함께 있다.

| `clusters.channels.npy` | cluster \(c\)가 놓인 recording channel \(\chi(c)\) |
| `channels.brainLocationIds_ccf_2017.npy` | channel \(\chi\)의 Allen CCF 2017 numeric region id \(I(\chi)\) |

핵심은 strict acronym label을 버리는 것이 아니라, strict acronym이 없는 cluster에만 fallback을 적용하는 것이다. Session \(s\), probe \(p\), cluster \(c\)에 대해 strict acronym을 \(A_{sp}(c)\), cluster-channel map을 \(\chi_{sp}(c)\), channel CCF id를 \(I_{sp}(\chi)\)라고 두면 hybrid map은 다음처럼 정의한다.

$$
G_{\mathrm{hybrid},sp}(c)
=
\begin{cases}
A_{sp}(c),
&A_{sp}(c)\neq\varnothing,\\
I_{sp}(\chi_{sp}(c)),
&A_{sp}(c)=\varnothing\ \mathrm{and}\ I_{sp}(\chi_{sp}(c))>0,\\
\varnothing,
&\mathrm{otherwise}.
\end{cases}
$$

여기서 \(A_{sp}(c)\)는 CCF acronym이고, \(I_{sp}(\chi_{sp}(c))\)는 numeric CCF id다. 둘은 같은 종류의 label이 아니므로 생물학적 의미를 동일하게 읽으면 안 된다. 이 gate에서 hybrid bin의 역할은 "정확한 region acronym 복원"이 아니라, strict acronym row 부족 때문에 censored 되던 cluster를 channel-registered anatomical id bin으로 되살리는 것이다.

Feature 식은 기존 region-window rate와 같은 형태를 유지한다.

$$
x_{ipg}^{\mathrm{hybrid}}
=
\frac{1}{b_i-a_i}
\sum_k
\mathbf 1[t_{pk}\in[a_i,b_i]]
\mathbf 1[G_{\mathrm{hybrid},sp}(c_{pk})=g].
$$

따라서 비교한 model은 다음 세 해부학 표현과 하나의 flat baseline이다.

| model | definition | 목적 |
|---|---|---|
| `strict_acronym_by_probe` | strict CCF acronym만 사용, 나머지는 `unknown` | 이전 cross-session gate와 같은 censored reference |
| `channel_region_id_by_probe` | cluster-channel에서 얻은 numeric CCF id만 사용 | acronym 없이 channel registration만으로 되는지 확인 |
| `hybrid_acronym_channel_id_by_probe` | acronym이 있으면 acronym, 없으면 channel CCF id | unknown-bin rescue model |
| `global_rate` | hybrid feature 전체 rate 합 | region/probe 구조를 지운 flat firing-rate baseline |

### Mouse IBL/OpenAlyx task-baseline comparison gate

Channel-region rescue 다음에는 다른 반례를 검사했다. Region decoder가 실제 neural state를 읽는 것이 아니라, 단순 trial clock, stimulus table, previous-trial history만 복원하는 경우다. 이 반례를 다음처럼 분리했다.

$$
X_i^{\mathrm{timing}}
=
\left[
\mathrm{trial\ index},
\mathrm{session\ time},
\mathrm{stim\ delay},
\mathrm{go\ cue\ delay}
\right]_i
$$

$$
X_i^{\mathrm{task}}
=
\left[
X_i^{\mathrm{timing}},
\mathrm{signed\ contrast}_i,
|\mathrm{contrast}_i|,
\mathrm{stimulus\ side}_i,
\mathrm{probabilityLeft}_i,
H_{i-1}
\right].
$$

여기서 \(H_{i-1}\)는 previous choice, previous feedback, previous reward, previous signed contrast, previous reaction latency, previous response latency다. Current choice, current first movement, current response, current feedback은 baseline feature에서 제외했다. 이유는 이 값들이 이번 target 자체이거나 target 직후 outcome이라서 leakage가 되기 때문이다.

비교한 모델은 다음이다.

| model | feature | leakage rule |
|---|---|---|
| `timing_only` | trial clock and pre-response timing | current behavior/outcome excluded |
| `task_history` | timing, current stimulus, previous-trial history | current choice/movement/outcome excluded |
| `hybrid_region_by_probe` | channel-rescued probe-region spike rates | neural window feature |
| `task_history_plus_hybrid_region` | task history plus hybrid region | neural increment over task table |
| `global_rate` | one scalar total hybrid firing rate | flat firing-rate baseline |

두 차이를 명시했다.

$$
\Delta_{\mathrm{timing}}
=
\mathrm{BA}(R^{\mathrm{hybrid}})
-
\mathrm{BA}(X^{\mathrm{timing}}),
$$

$$
\Delta_{\mathrm{task}}
=
\mathrm{BA}
\left(
\left[X^{\mathrm{task}},R^{\mathrm{hybrid}}\right]
\right)
-
\mathrm{BA}(X^{\mathrm{task}}).
$$

\(\Delta_{\mathrm{timing}}>0\)이면 단순 session drift/trial clock 반례가 약해진다. \(\Delta_{\mathrm{task}}>0\)이면 current stimulus와 previous-trial history만으로 설명되지 않는 neural increment가 남는다.

### Mouse IBL/OpenAlyx flat-unit versus hybrid-region comparison gate

Task-baseline gate 다음에는 representational counterexample을 검사했다. 같은 neural window 안에서 anatomical region/probe bin을 쓰지 않고, 개별 cluster identity를 flat feature로 쓰면 더 잘 되는가다. 이 반례가 중요하다. Region decoder가 global-rate와 timing을 이기더라도, flat unit이 훨씬 잘 되면 region 항은 "최대 예측기"가 아니라 "해부학적 압축 표현"으로 내려가야 한다.

Probe \(p\)에서 spike count 상위 \(K=96\) cluster를 골라 unit map을 만든다.

=
\begin{cases}

\mathrm{other\ units},&\mathrm{otherwise}.
\end{cases}
$$

$$

=
\frac{1}{b_i-a_i}
\sum_k
\mathbf 1[t_{pk}\in[a_i,b_i]]
\mathbf 1[U_p(c_{pk})=u].
$$

\mathrm{BA}(U^{\mathrm{top}})
-
\mathrm{BA}(R^{\mathrm{hybrid}}),
$$

$$
\Delta_{\mathrm{task+unit}}
=
\mathrm{BA}
\left(
\left[X^{\mathrm{task}},U^{\mathrm{top}}\right]
\right)
-
\mathrm{BA}
\left(
\left[X^{\mathrm{task}},R^{\mathrm{hybrid}}\right]
\right).
$$

이유는 count만 보면 choice와 speed에서 interaction이 3/5 후보에서 additive region을 이기지만, mean delta는 세 target 모두 음수이기 때문이다.

$$
\overline{\Delta}_{\mathrm{int-region}}
=
\begin{cases}
-0.007047,&\mathrm{choice},\\
-0.006616,&\mathrm{speed},\\
-0.016091,&\mathrm{wheel}.
\end{cases}
$$

또한 interaction은 top-unit을 거의 이기지 못한다.

$$

=
\mathrm{choice}\ 0/5,
\mathrm{speed}\ 1/5,
\mathrm{wheel}\ 0/5.
$$

따라서 \(K_{\mathrm{region}}(R\otimes R)\) 항은 현재 식에 승격하지 않는다.

$$
\boxed{
K_{\mathrm{same\ window}}^{(s)}
\left(
R_t^{(s,\mathrm{hybrid})}
\otimes
R_t^{(s,\mathrm{hybrid})}
\right)
\quad
\mathrm{not\ promoted.}
}
$$

이 음성 결과가 weighted interaction 전체를 부정하지는 않는다. 같은 window 안의 곱 \(r_a(t)r_b(t)\)는 방향성과 지연을 잃는다. 그래서 다음 gate는 아래처럼 source window와 target window를 분리한 lagged coupling으로 진행했다.

$$
z_{iab}^{(\delta)}
=
r_{ia}(t-\delta)r_{ib}(t),
\qquad
\delta>0.
$$

### Mouse IBL/OpenAlyx lagged region-coupling proxy gate

Lagged proxy에서는 source window와 target window를 분리했다. Choice와 first-movement speed는 `stimOn -300 ms`부터 `stimOn`까지를 source로 두고, target은 기존 stimulus window `stimOn +20 ms`부터 `+320 ms`까지로 둔다. Wheel action direction은 `firstMovement -400 ms`부터 `-100 ms`까지를 source로 두고, target은 기존 movement window `firstMovement -100 ms`부터 `+200 ms`까지로 둔다.

식은 다음이다.

$$
z_{iab}^{\mathrm{lag}}
=
r_{ia}^{\mathrm{source}}
r_{ib}^{\mathrm{target}}.
$$

Target-window region과 결합한 lagged model은

$$
R_i^{\mathrm{lag}}
=
\left[
R_i^{\mathrm{target}},
Z_i^{\mathrm{lag}}
\right].
$$

### Mouse IBL/OpenAlyx all-unit nested-regularization gate

Nested gate의 질문은 단순히 unit decoder가 region decoder보다 높은가가 아니다. 더 좁은 질문은 다음이다.

1. Task/history \(X_i\)가 이미 들어간 뒤 region \(R_i\)가 남는가.
2. Task/history와 region \(R_i\)가 이미 들어간 뒤 unit identity \(U_i\)가 남는가.
3. 반대로 task/history와 unit \(U_i\)가 이미 들어간 뒤 region compression \(R_i\)가 남는가.

여기서 \(X_i\)는 current stimulus, previous choice/reward, trial timing history로 구성한 task/history covariate이고, \(R_i\)는 channel-rescued hybrid region bin, \(U_i\)는 probe별 high-spike unit identity bin이다. 이번 실행에서는 `min_unit_spikes=1000`, `max_units_per_probe=192`를 사용했다. 이 cap은 계산량을 제한하기 위한 guard이며, 이전 top-unit 96개보다 넓은 unit panel이다.

비교한 nested model은 다음이다.

$$
M_X:
y_i\sim X_i,
\qquad
M_{XR}:
y_i\sim [X_i,R_i],
$$

$$
M_{XU}:
y_i\sim [X_i,U_i],
\qquad
M_{XRU}:
y_i\sim [X_i,R_i,U_i].
$$

각 model은 같은 deterministic stratified fold 위에서 ridge linear decoder로 평가한다.

$$
\hat\beta_m^{(-k)}
=
\arg\min_{\beta}
\left\|
y_{\mathrm{train}(k)}
-
Z_{m,\mathrm{train}(k)}\beta
\right\|_2^2
+
\lambda\|\beta\|_2^2,
\qquad
\lambda=1.
$$

Test fold \(k\)에서 얻은 score를 합쳐 balanced accuracy를 계산한다.

$$
\mathrm{BA}(M_m)
=
\frac12
\left(
\frac{TP_m}{P}
+
\frac{TN_m}{N}
\right).
$$

핵심 residual은 두 개다.

$$
\Delta_{U\mid X,R}
=
\mathrm{BA}(M_{XRU})
-
\mathrm{BA}(M_{XR}),
$$

$$
\Delta_{R\mid X,U}
=
\mathrm{BA}(M_{XRU})
-
\mathrm{BA}(M_{XU}).
$$

\(\Delta_{U\mid X,R}>0\)가 후보 session들에서 반복되면, unit identity는 task와 anatomical region을 넘어 독립적으로 살아남는다. 반대로 \(\Delta_{R\mid X,U}>0\)가 반복되면, high-dimensional unit readout 위에서도 region compression이 작지만 안정적인 정규화 항으로 남는다.

$$

\quad

\quad
[X_t,R_t].
}
$$

그리고 더 중요한 정리:

$$
\boxed{
R_t^{(\mathrm{hybrid})}
\quad
\mathrm{survives\ as\ a\ compact\ anatomical\ regularizer\ over}
\quad
[X_t,U_t].
}
$$

이 결과는 flat-unit gate와 모순이 아니다. Flat-unit gate는 \(U_t\)와 \(R_t\)를 단독 또는 task와 각각 결합해 비교했다. 그 결과 top-unit은 choice/wheel에서 region보다 강했다. 하지만 nested gate는 \(R_t\)를 이미 포함한 뒤 \(U_t\)가 더해지는지를 본다. 고차원 unit bin은 candidate panel 평균에서 cross-validated generalization을 오히려 낮췄다. 즉 mouse 단계의 현재 식은 "unit identity가 중요하지 않다"가 아니라, "공개 IBL 5-candidate panel의 ridge nested decoder에서는 region-compressed state가 더 안정적인 중간 표현이다"로 읽어야 한다.

다음 강한 버전은 common ridge가 아니라 block-regularized unit penalty 또는 GLM coupling이다.

$$
\lambda_X\|\beta_X\|_2^2
+
\lambda_R\|\beta_R\|_2^2
+
\lambda_U\|\beta_U\|_2^2,
\qquad
\lambda_U
\gg
\lambda_R
\quad
\mathrm{or}
\quad
y_t\sim f(U_{t-\ell},R_t,X_t).
$$

## 현재 방정식 판정

현재까지의 결과를 refined equation에 대입하면 다음처럼 나뉜다.

| 단계 | 판정 | 닫힌 항 | 아직 남은 항 |
|---|---|---|---|
| 생명 최소 동역학 | not tested | autocatalysis/boundary/copying 후보식 | LUCA/ribozyme/protocell gate |
| C. elegans primitive neural control | closed enough | weighted chemical \(\mathcal L(W)\), stimulus-output \(U_d\rightarrow b_d\), 발달 안정성 | 실제 trial behavior |
| Drosophila next term | closed | adult FlyWire에서 celltype/action/memory block과 memory/action loop 통과 | 행동 trial dynamics |
| Zebrafish activity closure | closed with caveat | assembly, low-rank recurrent state, perturbation-behavior, discrete behavior association | timestamp-certified continuous movement |
| Mouse Neuropixels/IBL metadata bridge | ready | 29 strict sessions, 483 channel-fallback sessions | region-binned decision/action decoding |
| Mouse Neuropixels/IBL first strict gate | single-session passed | choice BA 0.632548, speed BA 0.847971, wheel direction BA 0.709179 with acronym-region model | multi-probe motor/striatal generalization |
| Mouse IBL motor-striatal multi-probe | single-session passed | NYU-30 choice BA 0.836748, speed BA 0.673460, wheel BA 0.851462 | cross-session repetition |
| Mouse Neuropixels/IBL cross-session candidate panel | candidate-panel passed | 4/5 candidates pass all three targets; choice 4/5, speed 5/5, wheel 4/5 | unknown-bin reduction |
| Mouse IBL channel-region rescue | candidate-panel passed | strict unknown 26.9703% to hybrid unknown 0%; hybrid choice 4/5, speed 5/5, wheel 4/5 | larger panel and stronger baselines |

| Mouse IBL all-unit nested regularization | negative/region survives | \(U\mid X,R\) is 0/5, 1/5, 0/5; \(R\mid X,U\) is 3/5, 3/5, 4/5 | block ridge or GLM coupling |

따라서 현재 고정 가능한 원시 신경계 최소식은 다음이다.

$$
\boxed{
P_{n+1}
=
\Pi
\left[
\rho P_n
+(1-\rho)P_*
+\gamma\mathcal L(W_{\mathrm{chem}})P_n
+\sum_d a_{d,n}U_d
+H(q_n-q_*)
\right]
}
$$

Drosophila 이후에는 다음 항을 추가한다.

$$
\boxed{
D_{\mathrm{celltype}}
+A_{\mathrm{action}}
+M_{\mathrm{memory}}
}
$$

Zebrafish 이후에는 실제 activity state 항을 추가한다.

$$
\boxed{
L_{\mathrm{lowrank}}(P_t)
+C_{\mathrm{assembly}}(P_t)
+\Omega(P_{t-\ell},q_t)
}
$$

단, \(\Omega\)의 continuous movement component는 아직 data-boundary다.

Mouse/IBL 이후에는 포유류 task-history와 region compression 항을 우선 추가하고, unit-detail은 독립 additive residual이 아니라 강한 decoder/caveat 항으로 남긴다.

$$
\boxed{
B_{\mathrm{stim/history}}^{(s)}
X_t^{(s,\mathrm{task})}
+
B_{\mathrm{region\ compression}}^{(s)}
R_t^{(s,\mathrm{hybrid})}
+
B_{\mathrm{action\ increment}}^{(s)}
\left[
X_t^{(s,\mathrm{task})},
R_t^{(s,\mathrm{hybrid})}
\right]
+
\varepsilon_{\mathrm{unit}}^{(s)}
\left(
U_t^{(s)}
\mid
X_t^{(s,\mathrm{task})},
R_t^{(s,\mathrm{hybrid})}
\right)
}
$$

단, 이 항은 현재 5-candidate IBL panel 수준의 closure다. Timing-only baseline은 깼지만, choice는 task-history가 강하고 flat-unit이 choice/wheel에서 region을 이긴다. Same-window region interaction과 lagged region coupling proxy도 통과하지 못했다. All-unit nested gate에서는 \(\varepsilon_{\mathrm{unit}}\)이 독립 additive residual로 승격되지 않았고, 오히려 small region residual이 \(task+unit\) 위에서 남았다. Human/AGI 쪽 workspace 항으로 넘기기 전에는 block-regularized unit penalty, GLM coupling, larger registered panel이 남아 있다.

## 논의

현재 결과는 "생명에서 지능으로 가는 하나의 완성된 증명"이 아니다. 더 좁고 강한 결론이다. 공개 자료가 있는 각 층에서 가장 작은 gate를 만들었고, 그 gate를 통과한 항만 다음 단계 방정식에 남겼다.

항 승격 규칙은 다음이다.

$$
\mathrm{candidate}
\xrightarrow{
\mathrm{flat\ baseline\ pass}
}
\mathrm{supported}
\xrightarrow{
\mathrm{permutation/holdout\ pass}
}
\mathrm{closed}
$$

단, 필요한 관측 bridge가 없으면 다음처럼 분리한다.

$$
\mathrm{not\ closed}
\neq
\mathrm{false}
$$

즉 데이터 정렬이 없어서 검증할 수 없는 항은 false가 아니라 data-boundary다. Zebrafish continuous movement가 여기에 해당한다. 이 구분은 중요하다. data-boundary를 false로 읽으면 실제로는 아직 관측하지 못한 동역학을 부정하게 되고, false를 data-boundary로 읽으면 실패한 모델을 계속 끌고 가게 된다.

### 왜 weighted routing이 첫 항인가

C. elegans 결과에서 가장 강한 신호는 L1/L2/L3라는 이름 자체가 아니다. 핵심은 같은 구조를 binary graph로 바꾸면 gate가 깨지고, chemical weighted graph에서는 통과한다는 점이다. 따라서 최초 신경계 항은 "층이 있다"가 아니라 다음 명제로 읽어야 한다.

$$
\boxed{
\text{원시 neural computation은 edge existence가 아니라 synaptic weight distribution에 실린다.}
}
$$

이 명제는 포유류로 올라갈 때도 직접적인 방법론적 제약을 준다. Mouse/IBL에서 영역 graph를 만들 때도 단순히 어느 영역이 연결되었다는 이진 구조보다, spike coupling, effective connectivity, trial-conditioned interaction strength처럼 weighted quantity를 우선해야 한다.

### 왜 Drosophila 항은 memory 단독이 아닌가

Larva 결과만 보면 mushroom body loop가 강한 후보로 보인다. 하지만 competing model에서 action split도 강했고, strict permutation은 통과하지 못했다. 이 때문에 "C. elegans 다음은 memory다"라는 문장은 너무 강하다.

Adult FlyWire에서 닫힌 것은 다음 항이다.

$$
D_{\mathrm{celltype}}
+A_{\mathrm{action}}
+M_{\mathrm{memory}}
$$

여기서 \(D_{\mathrm{celltype}}\)는 neuron type과 circuit role의 분화, \(A_{\mathrm{action}}\)은 descending/central-complex 계열 action routing, \(M_{\mathrm{memory}}\)는 mushroom body 계열 internal trace를 뜻한다. 세 항은 독립 장식이 아니라 함께 닫힌다. memory가 행동으로 연결되지 않으면 internal loop에 머물고, action이 memory와 분리되면 단순 reflex에 가까워진다. 따라서 곤충 단계의 도약은 memory storage가 아니라 memory-action closed loop다.

### 왜 zebrafish는 activity closure인가

Zebrafish는 구조 connectome만으로 설명할 수 없는 단계다. 실제 calcium activity가 들어오면 \(W\)의 block 구조보다 다음 식이 중요해진다.

$$
P_t
\mapsto
P_{t+1}
$$

이 mapping이 baseline보다 좋아야 activity state라는 말을 쓸 수 있다. Optic tectum과 freely swimming 자료에서 recurrent/baseline이 1보다 작게 나왔기 때문에, 이 단계에서는 "회로가 있다"보다 "상태가 닫힌다"가 핵심이다.

다만 activity closure와 behavior closure는 다르다.

$$
P_t\rightarrow P_{t+1}
\quad\not\Rightarrow\quad
P_t\rightarrow y_t
$$

현재 zebrafish에서 통과한 behavior bridge는 discrete bout frame과 left/right direction association이다. continuous speed/turn/heading은 timestamp bridge가 없어 최종 gate가 아니다.

### Mouse/IBL에서 깨야 할 반례

Mouse 단계에서 가장 위험한 반례는 region loop가 필요 없고, 단순 전체 firing rate, trial timing, task table, 또는 flat neuron feature만으로 choice/action이 설명되는 경우다. 현재까지 global firing-rate와 timing-only 반례는 깨졌고, task-history 반례는 action 쪽에서만 약해졌다. Flat-unit 반례는 일부 성립했다. Choice와 wheel에서는 top-unit readout이 region readout을 반복적으로 이겼다. 따라서 남은 비교는 다음처럼 읽는다.

| 모델 | 깨야 하는 이유 |
|---|---|
| global firing baseline | 전체 activity 양만으로 choice가 설명되는지 배제 |
| trial timing baseline | session clock만으로 decoding되는지 배제. 현재 5/5로 배제됨 |
| task-history baseline | stimulus/probability/previous history만으로 충분한지 확인. choice는 caveat |
| flat neuron model | 영역 구조 없이 개별 unit feature만으로 충분한지 비교. 현재 choice/wheel에서 unit 우세 |
| same-window interaction | region pair \(r_a(t)r_b(t)\)가 additive region보다 나은지 확인. 현재 mean delta 음수 |
| lagged coupling model | \(r_a(t-\delta)r_b(t)\)가 unit-detail residual을 설명하는지 확인. 현재 mean delta 음수 |
| all-unit nested residual | \(task+region\) 뒤에 unit residual이 남는지 확인. 현재 unit residual은 음수, region residual은 작게 양수 |

Mouse gate가 통과하려면 적어도 다음 부등식이 필요하다.

$$
\mathcal L_{\mathrm{probe/hybrid\ region}}
<
\min
\left(
\mathcal L_{\mathrm{global}},
\mathcal L_{\mathrm{timing}}
\right),
\qquad
\Delta_{\mathrm{task/action}}>0.
$$

Flat-neuron 비교 뒤에는 다음 식으로 내려간다.

$$
\mathcal L_{\mathrm{top\ unit}}
<
\mathcal L_{\mathrm{probe/hybrid\ region}}
\quad
\mathrm{for\ choice/wheel}.
$$

따라서 포유류 영역 loop를 "unit보다 강한 decoder"로 승격할 수는 없다. 현재는 region/probe readout이 timing artifact가 아니라는 점까지 닫혔고, choice는 task-history dominated, action은 neural increment supported, choice/wheel은 flat 비교에서 unit-detail dominated 상태다. 그러나 same-window region interaction, lagged region coupling, all-unit residual after \(task+region\)은 모두 통과하지 못했다. 남는 것은 \(R_t\)가 high-dimensional \(U_t\) 위에서도 작게 살아나는 compressed anatomical regularizer라는 판정이다.

## 한계

현재 문서의 한계는 명확히 남긴다.

| 한계 | 의미 | 다음 보완 |
|---|---|---|
| 생명 최소식 미검증 | autocatalysis/boundary/copying은 아직 직접 gate가 없다 | LUCA/ribozyme/protocell 자료 |
| C. elegans behavior proxy | connectome flow이지 실제 trial behavior가 아니다 | stimulus-behavior recording |
| Drosophila behavior dynamics 없음 | adult FlyWire는 구조 connectome이다 | fly behavior trial 또는 calcium/ephys |
| Zebrafish continuous data-boundary | e2-behavior timestamp bridge가 없다 | synchronized raw 또는 e2-resampled tracking |
| Mouse/IBL panel 한계 | choice/wheel은 flat 비교에서 top-unit이 region을 이기지만 nested unit residual은 음성이고 region residual은 작게 남는다 | larger panel, block-regularized unit penalty, GLM coupling |

이 한계 때문에 이 장의 판정은 "하위 사다리의 방정식 항을 다음 단계로 넘길 만큼 닫았다"이지, "생명에서 인간 지능까지 전부 닫았다"가 아니다.

| 필요한 것 | 이유 |
