# 뇌 이론 제9장: 생명에서 지능까지

이 장이 생명-뇌-지능 진화 trace의 정본이다. 실행 스크립트와 자동 리포트는 근거 자료이고, 최종 해석, 수식, 판정, 다음 병목은 이 장을 기준으로 읽는다.

## 초록

이 장의 목적은 "뇌 방정식"을 인간 뇌에서 갑자기 시작하는 식으로 두지 않고, 생명 최소 동역학에서 C. elegans, Drosophila, zebrafish, mouse로 이어지는 진화 사다리 위에서 역추적하는 것이다. 핵심 질문은 각 단계에서 방정식의 어떤 항이 새로 필요해지는가다. 현재까지의 결과는 네 가지로 요약된다.

첫째, C. elegans 수준의 최소 원시 신경계에서는 binary topology가 아니라 weighted chemical connectivity가 감각-domain을 출력-domain으로 보내는 routing operator로 닫힌다.

둘째, Drosophila에서는 memory 하나가 단독으로 추가되는 것이 아니라 cell type, descending/action, mushroom body, central complex 계열이 함께 분화하고, adult FlyWire 규모에서는 이 공동 분화가 permutation gate를 통과한다.

셋째, zebrafish에서는 실제 calcium activity state가 assembly/recurrent 구조와 행동 frame/direction association까지 닫히지만, continuous speed/turn decoding은 공개 partial/Others 자료에 e2-behavior timestamp bridge가 없어서 data-boundary로 남는다.

넷째, mouse IBL/OpenAlyx에서는 첫 strict Neuropixels session과 motor/striatal multi-probe session이 통과했고, 이어 5개 후보 cross-session panel에서 4개 후보가 choice, first-movement speed, wheel action direction을 모두 통과했다.

마지막으로 strict cluster acronym에서 생기던 26.9703% unknown spike caveat는 channel-level CCF id fallback으로 0%까지 줄였고, hybrid region-id decoder도 choice 4/5, speed 5/5, wheel 4/5 반복성을 유지했다. 추가 baseline gate에서는 timing-only 반례가 세 target 모두 5/5로 깨졌지만, task-history increment는 speed 5/5, wheel 4/5, choice 2/5라서 choice 항은 stimulus/task-table 성분이 강한 caveat를 남겼다. 이어 flat-unit gate에서는 top-unit readout이 choice 5/5, wheel 5/5, speed 2/5에서 hybrid region을 이겼지만, same-window region interaction과 lagged region coupling은 모두 음성이었다. common-ridge nested gate에서는 \(task+region\) 뒤의 all-unit residual이 반복되지 않았지만, 새 block-regularized unit gate에서 \(\lambda_U\)를 train fold 내부에서 고르면 \(U_{\mathrm{block}}\mid X,R\)가 choice 4/5, speed 5/5, wheel 5/5로 살아난다. 반대로 temporal GLM strict gate에서는 \(U_L\mid X,R,U_0\)가 반복되지 않았다. 따라서 mouse 항은 region-only closure가 아니라 task/history, anatomically compressed region, block-regularized current-window unit detail이 함께 남는 mixed readout으로 해석해야 하며, directed temporal coupling은 아직 승격하지 않는다.

따라서 이 장의 현재 결론은 다음이다.

$$
\boxed{
\mathrm{life}
\rightarrow
\mathrm{weighted\ neural\ routing}
\rightarrow
\mathrm{role\ differentiation}
\rightarrow
\mathrm{activity\ state\ closure}
\rightarrow
\mathrm{mammalian\ region\ loop}
}
$$

이 결론은 지능을 단순한 규모 증가가 아니라, 보존되는 제어 동역학 위에 새 연산자가 추가되는 과정으로 읽게 한다.

## 중심 질문

인간 뇌는 갑자기 생긴 장치가 아니라, 원시 생명, 원시 신경계, 감각-운동 회로, 학습 가능한 내부 loop, 척추동물 활동 상태공간, 포유류 영역 loop가 층층이 쌓인 진화체다. 따라서 뇌 방정식은 다음 질문을 받아야 한다.

$$
\boxed{
\text{생명에서 지능까지 올라갈 때, 방정식의 어떤 항이 새로 필요해지는가?}
}
$$

단순히 neuron 수, synapse 수, brain volume만 비교하지 않는다. 각 단계에서 실제로 닫히는 항을 본다.

| 축 | 질문 | 닫힘의 의미 |
|---|---|---|
| self-maintenance | 열린 반응계가 선택 가능한 자기유지 동역학으로 닫히는가 | 생명 최소항 |
| routing | 자극 domain이 같은 행동/output domain으로 안정적으로 흐르는가 | 원시 신경계 항 |
| differentiation | cell type, action, memory 역할이 분화되는가 | 곤충 단계 항 |
| activity closure | 실제 neural activity state가 다음 상태를 예측하는가 | 척추동물 activity 항 |
| behavior bridge | activity가 trial/frame 행동을 예측하는가 | activity-to-behavior 항 |
| region loop | 포유류 영역이 decision, action, internal state를 나누는가 | mouse/human 전 단계 항 |
| workspace | 장기 memory, replay, abstract task state가 전역적으로 묶이는가 | 고등 지능 후보항 |

현재까지 얻은 핵심 결론은 다음이다.

$$
\boxed{
\text{지능은 신경계 양의 단순 증가가 아니라, 보존되는 weighted routing 위에 역할 분화와 닫힌 activity state가 쌓인 결과다.}
}
$$

## 주장과 기여

이 장은 관측 자료를 곧바로 하나의 거대 식에 맞추려는 문서가 아니다. 먼저 종별로 가장 작은 닫힌 gate를 만들고, 통과한 항만 상위 방정식으로 승격한다. 그래서 주장은 네 층으로 나뉜다.

| 층 | 주장 | 현재 지위 |
|---|---|---|
| 생명 최소식 | boundary/copy/autocatalysis가 선택 가능한 열린 자기유지 동역학을 만든다 | 아직 gate 없음 |
| 원시 신경계 | weighted chemical graph가 최초 neural routing operator다 | C. elegans에서 supported |
| 곤충 단계 | C. elegans 다음 항은 memory 단독이 아니라 celltype/action/memory 공동 분화다 | adult FlyWire에서 closed |
| 척추동물 단계 | 실제 neural activity state는 recurrent/assembly 구조와 discrete behavior bridge를 가진다 | zebrafish에서 closed with caveat |
| 포유류 단계 | region loop가 decision/action/internal state를 분리해야 한다 | mouse/IBL candidate panel passed with registration caveat reduced |

이 문서의 기여는 "어떤 생물이 더 지능적이다"라는 서술이 아니라, 각 진화 단계에서 방정식에 새로 들어가는 항을 검증 가능한 손실함수로 분리한 점이다. 따라서 아래의 식들은 생물학적 장식이 아니라 실험 gate를 생성하는 압축 표현이다.

## 공통 상태 묶음

종 또는 발달 단계 \(s\)의 상태를 다음 네 묶음으로 둔다.

$$
Z_n^{(s)}
=
\left(
P_n^{(s)},q_n^{(s)},m_n^{(s)},W_n^{(s)}
\right)
$$

| 기호 | 의미 |
|---|---|
| \(P_n^{(s)}\) | 빠른 biochemical, neural, calcium activity state |
| \(q_n^{(s)}\) | 몸 상태, 에너지, 각성, 항상성 변수 |
| \(m_n^{(s)}\) | 느린 내부 상태, short memory, synaptic trace, eligibility trace |
| \(W_n^{(s)}\) | weighted chemical, structural, effective graph |

원시 생명 단계에는 \(P\) 대신 chemical population \(X\)가 주 상태다. C. elegans 구조 gate에서는 \(W\)가 고정 connectome으로 관측된다. Zebrafish부터는 \(P_t\)가 실제 calcium activity matrix로 들어온다. Mouse/IBL부터는 \(P_t\)를 spike-rate/region-binned activity로 접는다.

## 공통 고도화식

빠른 상태식은 다음 controlled dynamical family로 둔다.

$$
P_{n+1}^{(s)}
=
\Pi_{\mathcal S_s}
\left[
\rho_sP_n^{(s)}
+(1-\rho_s)P_*^{(s)}
+\gamma_s\mathcal L(W_{\mathrm{chem/effective}}^{(s)})P_n^{(s)}
+\sum_{d\in\mathcal D_s}a_{d,n}^{(s)}U_d^{(s)}(z_n,q_n)
+H_s(q_n^{(s)}-q_*^{(s)})
+A_sb_n^{(s)}
+M_sm_n^{(s)}
+\eta_n^{(s)}
\right]
$$

행동 관측식은 다음으로 둔다.

$$
b_{n+1}^{(s)}
=
\Omega_s(P_n^{(s)},q_n^{(s)},E_n)
+\xi_{b,n}^{(s)}
$$

항상성식은 다음으로 둔다.

$$
q_{n+1}^{(s)}
=
q_n^{(s)}
+B_s(E_n)
-C_s(b_n^{(s)},P_n^{(s)})
-\chi_s(q_n^{(s)}-q_*^{(s)})
$$

느린 memory/synaptic trace는 다음으로 둔다.

$$
m_{n+1}^{(s)}
=
\lambda_sm_n^{(s)}
+\Psi_s(P_n^{(s)},b_n^{(s)},r_n)
$$

학습 가능한 graph 또는 synaptic weight 변화는 다음으로 둔다.

$$
W_{n+1}^{(s)}
=
\Pi_{\mathcal W_s}
\left[
W_n^{(s)}
+\epsilon_s\Phi_s(P_n^{(s)},b_n^{(s)},r_n)
-\mu_sW_n^{(s)}
\right]
$$

여기서 가장 중요한 제약은 다음이다.

$$
\boxed{
\mathcal L(W)\neq \mathcal L(A_{\mathrm{binary}})
\quad\text{and}\quad
\mathcal L(W)=\mathcal L(W_{\mathrm{weighted}})
}
$$

C. elegans gate에서 반복적으로 확인된 것은 binary adjacency가 아니라 weighted chemical graph가 원시 신경계의 routing operator라는 점이다. 따라서 포유류나 인간으로 올라가도 graph term을 검증할 때 단순 edge existence보다 weight/effective connectivity를 우선한다.

## 식의 유도

공통 고도화식은 임의로 쓴 큰 식이 아니다. 출발점은 모든 단계에서 관측 가능한 상태가 다음 세 힘 사이에서 움직인다는 가정이다.

1. 자기 상태를 유지하려는 residual term
2. 구조 또는 graph를 따라 퍼지는 coupling term
3. 외부 자극, 몸 상태, 행동 결과, 느린 memory가 밀어 넣는 control term

이를 가장 작은 affine-controlled map으로 쓰면 다음이다.

$$
\tilde P_{n+1}
=
\rho P_n
+(1-\rho)P_*
+G(P_n;W)
+U_n
+H_n
+A_nb_n
+M_nm_n
+\eta_n
$$

생물학적 상태는 임의의 유클리드 벡터가 아니라 확률, firing proportion, calcium activity score, population fraction처럼 제한된 상태공간에 놓인다. 그래서 마지막에 projection을 둔다.

$$
P_{n+1}
=
\Pi_{\mathcal S}(\tilde P_{n+1})
$$

여기서 \(\Pi_{\mathcal S}\)의 역할은 세 가지다.

| 역할 | 의미 |
|---|---|
| boundedness | activity, probability, fraction이 물리적으로 가능한 범위를 벗어나지 않게 한다 |
| conservation | coarse state를 simplex나 정규화된 score space에 남긴다 |
| comparability | 종마다 다른 관측량을 같은 gate 문법으로 비교하게 한다 |

### Graph term

graph coupling은 처음에는 일반 \(G(P;W)\)로 둘 수 있다. 하지만 C. elegans에서 실제로 통과한 것은 edge existence가 아니라 weighted chemical synapse였으므로, 최소 operator는 Laplacian 또는 row-normalized transition의 형태로 좁힌다.

$$
G(P;W)
=
\gamma\mathcal L(W)P
$$

Laplacian형으로 쓰면

$$
[\mathcal L(W)P]_i
=
\sum_j w_{ij}(P_j-P_i)
$$

transition형으로 쓰면

$$
T_{ij}(W)
=
\frac{w_{ij}}{\sum_k w_{ik}+\varepsilon},
\qquad
P_{n+1}\supset \gamma T(W)P_n
$$

두 표현의 공통 의미는 "상태가 연결 강도에 따라 이동한다"는 것이다. C. elegans 결과가 중요한 이유는 여기서 \(W\)를 binary adjacency로 두면 gate가 깨지고, chemical weighted \(W\)로 둘 때 gate가 닫히기 때문이다.

### Stimulus-output term

자극 domain \(d\)가 들어오면 상태는 해당 domain의 input subspace에서 output subspace로 흐른다. 이를

$$
U_n
=
\sum_{d\in\mathcal D}a_{d,n}U_d
$$

로 둔다. \(a_{d,n}\)은 그 순간 domain \(d\)가 켜져 있는 정도이고, \(U_d\)는 그 domain이 밀어 넣는 방향이다. connectome proxy에서는 이 항을 직접 시간열로 보지 못하므로, row-normalized flow로 다음 비율을 검증한다.

$$
R_d
=
\frac{\mathrm{Flow}(L1_d\to L3_d)}
{\mathrm{mean}_{d'\ne d}\mathrm{Flow}(L1_d\to L3_{d'})}
$$

\(R_d>1\)이면 같은 domain output으로 흐르는 경로가 더 강하다는 뜻이고, \(R_d>1.5\) 및 permutation \(p<0.05\)면 domain-preserving channel로 승격한다. 이 단계는 실제 행동 trial은 아니지만, 원시 신경계가 이미 "자극을 행동 출력 계열로 보존해서 보내는 구조"를 가진다는 것을 보여준다.

### Homeostatic term

원시 생물부터 포유류까지 nervous system은 몸 밖에 떠 있는 계산기가 아니다. 에너지, 각성, 대사, 내부 상태가 activity를 밀고 당긴다. 그래서

$$
H_n
=
H_s(q_n-q_*^{(s)})
$$

를 둔다. 이 항은 C. elegans connectome gate에서는 직접 측정하지 않았지만, 상위 뇌 문서의 항상성 축과 연결되는 자리다. 수학적으로는 기준 몸 상태 \(q_*\)에서 벗어난 만큼 neural state를 특정 방향으로 이동시키는 forcing이다.

### Behavior and memory terms

행동이 단순 output label이면

$$
b_{n+1}
=
\Omega(P_n,q_n,E_n)
$$

로 충분하다. 그러나 Drosophila부터는 output이 단순 reflex가 아니라 action selection과 memory loop에 의해 바뀐다. 그래서 행동 항 \(A_sb_n\)과 느린 memory 항 \(M_sm_n\)을 분리한다.

$$
m_{n+1}
=
\lambda_sm_n
+\Psi_s(P_n,b_n,r_n)
$$

\(\lambda_s\)는 흔적이 얼마나 오래 남는지, \(\Psi_s\)는 neural state와 행동 결과, reward 또는 reinforcement signal이 memory trace를 어떻게 갱신하는지 나타낸다. Adult FlyWire에서 닫힌 것은 이 memory 항 하나가 아니라, memory 구조가 action/central-complex/descending 구조와 함께 분화한다는 점이다.

### Activity closure와 behavior bridge

Zebrafish부터는 \(W\)만 보지 않고 실제 activity \(P_t\)를 본다. 이때 최소 동역학 gate는 다음이다.

$$
\hat P_{t+1}
=
f_\theta(P_t),
\qquad
\mathcal L_{\mathrm{dyn}}
=
\frac{\mathrm{MSE}(P_{t+1},\hat P_{t+1})}
{\mathrm{MSE}(P_{t+1},\bar P)}
$$

이 값이 1보다 작으면 평균 상태로 예측하는 것보다 현재 상태가 다음 상태를 더 잘 설명한다. 즉 activity state가 단순 noise가 아니라 닫힌 상태공간 위에 있다는 뜻이다.

행동 bridge는 한 단계 더 강하다.

$$
\hat y_t
=
g_\theta(P_{t-\ell},q_t),
\qquad
y_t\in\{\mathrm{bout},\mathrm{direction},\mathrm{speed},\mathrm{turn},\mathrm{heading}\}
$$

discrete bout/direction은 현재 zebrafish 자료에서 통과했다. 그러나 continuous speed/turn/heading은 \(P_t\)와 \(y_t\)의 timestamp alignment가 필요하다. alignment가 없으면 \(g_\theta\)를 아무리 잘 맞춰도 물리적 검증이 아니라 임의 대응이 된다. 그래서 zebrafish continuous는 실패가 아니라 data-boundary로 판정한다.

## 검증 문법

모든 단계는 같은 판정 문법으로 비교한다.

Graph structure:

$$
\mathcal L_{\mathrm{graph}}
=
\frac{
\mathcal L(\mathrm{structured\ block})
}{
\mathcal L(\mathrm{flat})
}
$$

성공 조건:

$$
\mathcal L_{\mathrm{graph}}<1
\quad\text{and}\quad
p_{\mathrm{perm}}<0.05
$$

Domain routing:

$$
\mathcal L_{\mathrm{route}}
=
\frac{
\mathrm{Flow}(d\rightarrow d)
}{
\mathrm{mean}_{d'\ne d}\mathrm{Flow}(d\rightarrow d')
}
$$

성공 조건:

$$
\mathcal L_{\mathrm{route}}>1.5
\quad\text{and}\quad
p_{\mathrm{perm}}<0.05
$$

Activity closure:

$$
\mathcal L_{\mathrm{dyn}}
=
\frac{
\mathrm{MSE}(P_{t+1}\mid P_t)
}{
\mathrm{MSE}(P_{t+1}\mid \bar P)
}
$$

성공 조건:

$$
\mathcal L_{\mathrm{dyn}}<1
$$

Continuous behavior:

$$
\mathcal L_{\mathrm{beh}}
=
\frac{
\mathrm{MSE}(y_t\mid P_{t-\ell},q_t)
}{
\mathrm{MSE}(y_t\mid \bar y)
}
$$

성공 조건:

$$
\mathcal L_{\mathrm{beh}}<1,\qquad R^2>0,\qquad p_{\mathrm{shift}}<0.05
$$

Loop enrichment:

$$
\mathcal E_{\mathrm{loop}}
=
\frac{
\mathrm{LoopFraction}_{\mathrm{observed}}
}{
\mathbb E[\mathrm{LoopFraction}_{\mathrm{random}}]
}
$$

성공 조건:

$$
\mathcal E_{\mathrm{loop}}>2
\quad\text{and}\quad
p_{\mathrm{random}\ge\mathrm{observed}}<0.05
$$

자료 경계 판정:

$$
\mathrm{data\ boundary}
\Longleftrightarrow
\left(
\text{필요한 time alignment 또는 behavior bridge가 없음}
\right)
$$

이 경우는 "방정식 실패"로 올리지 않는다. 특히 zebrafish continuous movement는 여기에 해당한다.

## 통계 판정과 모델 비교

각 gate는 "숫자가 좋아 보인다"만으로 통과하지 않는다. 최소한 다음 세 가지를 분리한다.

| 비교 | 목적 | 예 |
|---|---|---|
| flat baseline | 구조를 넣는 것이 평균/단일 모델보다 나은가 | block/flat, recurrent/baseline |
| random label baseline | 같은 label 수의 무작위 구조보다 나은가 | permutation p |
| competing model | 우리가 제안한 항 말고 더 간단한 설명이 있는가 | action split, module family, all-one |

block model에서는 관측된 weighted adjacency \(W\)를 label pair별 평균 block matrix \(\hat W\)로 재구성한다. 손실은 다음으로 둔다.

$$
\mathcal L_{\mathrm{block}}
=
\sum_{i,j}
\left(
W_{ij}-\hat W_{\ell_i,\ell_j}
\right)^2
$$

flat model은 모든 edge를 하나의 평균으로 예측한다.

$$
\mathcal L_{\mathrm{flat}}
=
\sum_{i,j}
\left(
W_{ij}-\bar W
\right)^2
$$

따라서 \(\mathcal L_{\mathrm{block}}/\mathcal L_{\mathrm{flat}}<1\)이면 label 구조가 아무 구조 없는 평균보다 낫다. 하지만 label 수가 늘면 손실은 기계적으로 줄어들 수 있으므로 permutation과 BIC-like penalty를 같이 본다.

Permutation p는 같은 label count를 유지한 채 label assignment를 무작위로 섞어 계산한다.

$$
p_{\mathrm{perm}}
=
\frac{
1+\#\{r:\mathcal L_{\mathrm{rand},r}\le \mathcal L_{\mathrm{obs}}\}
}{
1+N_{\mathrm{perm}}
}
$$

여기서 \(p_{\mathrm{perm}}<0.05\)이면 관측 label이 같은 크기의 무작위 label보다 유의하게 낫다.

BIC-like 점수는 포화모델을 경계하기 위한 보조 기준이다.

$$
\mathrm{BIC}^*
=
N\log
\left(
\frac{\mathcal L}{N}+\varepsilon
\right)
+k\log N
$$

\(k\)는 block parameter 수다. 이 값은 엄밀한 likelihood BIC가 아니라 "label을 너무 많이 쪼갠 모델이 과도하게 이기는지"를 보는 안전장치다. 그래서 C. elegans에서는 layer가 살아남고, Drosophila larva에서는 memory 단독 가설이 충분히 강하지 않다는 판정이 가능했다.

Activity gate에서는 같은 원칙을 시간열에 적용한다.

$$
\mathcal L_{\mathrm{recurrent}}
=
\frac1T
\sum_t
\|P_{t+1}-\hat P_{t+1}(P_t)\|^2
$$

baseline은 다음 시점을 전체 평균 \(\bar P\)로 예측하는 모델이다.

$$
\mathcal L_{\mathrm{baseline}}
=
\frac1T
\sum_t
\|P_{t+1}-\bar P\|^2
$$

따라서 recurrent/baseline이 1보다 작으면 activity가 단순 평균 주변 noise가 아니라 자기 상태를 가진 동역학이라는 뜻이다.

## 진화 사다리

현재 사다리는 다음처럼 둔다.

```text
생명: autocatalysis + boundary + copying
원시 신경계: weighted chemical routing
단순 동물: stimulus-output domain channel
곤충: cell type/action/memory co-differentiation
척추동물: assembly + recurrent activity state
포유류: region loop + decision/action bridge + internal state
고등 지능: memory/replay + workspace + abstract task state
```

현재까지의 판정표:

| 단계 | 상태 | 닫힌 항 | 남은 항 |
|---|---|---|---|
| 생명 최소 동역학 | not tested | autocatalysis, boundary, copying 후보식 | LUCA, ribozyme, protocell gate |
| C. elegans connectome | closed enough | weighted chemical L1/L2/L3 routing | 실제 trial behavior |
| C. elegans stimulus-output | closed enough | stimulus domain to output domain routing | 더 촘촘한 행동 trial |
| Drosophila larva | candidate | memory/action/celltype 후보 | strict permutation 미통과 |
| Drosophila adult FlyWire | closed | celltype/action/memory block, memory/action loop | 행동 trial dynamics |
| Zebrafish activity | closed with caveat | assembly, recurrent state, perturbation behavior, discrete behavior association | timestamp-certified continuous movement |
| Zebrafish continuous movement | data-boundary | speed weak candidate only | e2 timestamp 또는 e2-resampled speed/turn trace |
| Mouse Neuropixels/IBL | mixed | region/probe bridge, channel rescue, timing-only rejected, block-regularized unit residual survives; strict temporal GLM coupling fails | larger registered panel, directed/state-space coupling |

## 단계 0: 생명 최소 동역학

신경계 이전의 생명은 "선택 가능한 열린 자기유지 동역학"으로 둔다. 상태는 molecule population \(X_n\)다.

$$
X_{n+1}
=
\Pi_{\mathcal X}
\left[
X_n
+R_{\mathrm{autocat}}(X_n,E_n)
-D_{\mathrm{decay}}(X_n)
-L_{\mathrm{leak}}(X_n)
+B_{\mathrm{boundary}}(X_n)
+C_{\mathrm{copy}}(X_n)
\right]
$$

항의 의미:

| 항 | 의미 |
|---|---|
| \(R_{\mathrm{autocat}}\) | 자기촉매 반응망 |
| \(D_{\mathrm{decay}}\) | 분해와 열적 손실 |
| \(L_{\mathrm{leak}}\) | boundary 밖으로 빠지는 leakage |
| \(B_{\mathrm{boundary}}\) | 막, compartment, vesicle에 의한 유지 |
| \(C_{\mathrm{copy}}\) | template copying 또는 복제 bias |

이 단계의 의의는 boundary와 copy가 열린 화학 반응계를 선택 가능한 자기유지 동역학으로 바꾼다는 점이다. 아직 직접 gate는 만들지 않았다. 다음에 이 층을 열려면 LUCA 비교유전체학, RNA world/ribozyme 실험, autocatalytic set 모델, protocell/lipid vesicle 실험을 따로 잡아야 한다.
