# Life to Intelligence Evolution Gates

이 폴더는 `docs/6_뇌/09_생명에서지능까지.md`의 실행 근거를 모은다.

역할 분리:

| 종류 | 역할 |
|---|---|
| `*_gate.py` | 통과/실패 판정을 내리는 재현 스크립트 |
| `*_audit.py` | 데이터에 필요한 bridge가 있는지 확인하는 감사 스크립트 |
| `*_probe.py` | 최종 gate 전 정찰 스크립트 |
| `*_report.md` | 스크립트가 생성한 숫자 리포트 |

정본 해석은 이 폴더가 아니라 `docs/6_뇌/09_생명에서지능까지.md`를 기준으로 한다.
초기 작업 노트는 9장으로 흡수한 뒤 제거했다.
